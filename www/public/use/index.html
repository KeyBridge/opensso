Builds

Get the latest builds.

opensso.war
(checksum opensso.war.sha)		 This WAR is deployable on all J2EE compliant Web Container. After deploying it, we can configure the server via a JSP page.
opensso.jar
(checksum opensso.jar.sha)		 This contains the binaries (all bytecodes that are generated in the project) and resources (such as i18n files (english locale) and service XML files).
opensso-src.jar
(checksum opensso-src.jar.sha)		 This contains all items in opensso.jar, plus Javadocs and all the source codes.
amclientsdk.jar
(checksum amclientsdk.jar.sha)		 This contains the necessary binaries and resources to allow Java client to do Single Sign On; send policy evaluation request; and receive policy evaluation results/advices.
openssodemo.war
(checksum openssodemo.war.sha)		 This is opensso.war, plus some JSPs; Java binaries; and additional directives and declarations in web.xml. This allows us to demostrate Single Sign On and Access Controls without having to install a Web Agent.
