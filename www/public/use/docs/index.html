Product Documentation

This table contains links to PDF versions of the documentation for the beta release of Sun Java System Access Manager 7.1. They have been donated by Sun Microsystems. The documents will open in a new window.

Title

Description

Administration Guide

Describes various administrative tasks such as Realms Management, Policy Management, Authentication and Directory Management.

Administration Reference Provides look-up information for administrators including sections on error codes, configuration files and attributes, and command-line tools.

C API Reference

Provides overview information and a look-up listing of the C APIs (including functions, syntax, and parameters).

Java API Reference

Describes the Java API Library. (HTML only.)

Federation and SAML Administration Guide

Provides information about the features based on the Liberty Alliance Project and Security Assertions Markup Language (SAML) specifications including instructions for enabling a Liberty-based environment, and summaries of the application programming interface (API) for extending the framework.

Performance Tuning Guide

Provides recommendations and procedures for tuning the application for optimum performance and reliability.

Policy Agent User's Guide

Describes Web and J2EE policy agents including installation and configuration information. (Individual agents are described in separate docs.)
