Documentation

The documentation on this page has been donated by Sun Microsystems. It includes beta quality tomes from the Sun Java System Access Manager 7.1 2006Q4 release.

Book Title

Description

Adminsitration Guide

Describes how to configure, monitor, manage, and maintain services, identities, and policies using either the graphical user interface or the command-line interface.

Administration Reference Provides reference information for administrators including sections on error codes, configuration files and attributes, and command-line tools.

C API Reference

Describes the C API Library.

Federation and SAML Administration Guide

Provides information about the federation and SAML components including information on the specifications, integrated web services, and API.

Java API Reference

Describes the Java API Library.

Performance Tuning Guide

Provides recommendations and procedures for optimum performance.

Postinstallation Guide

Provides configuration information and procedures for system administrators.

Policy Agent User's Guide

Describes Web and J2EE policy agents including installation and configuration information. (Individual agents are described in separate docs.)

Beta Release Notes

Describes last-minute issues relevant for the Access Manager administrator, including new features, bugs fixed in the release, installation notes, known issues and limitations (with workarounds, if available), how to report problems, and various Sun resources.
