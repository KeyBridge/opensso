Topics and Procedures

This page contains links to documents written by OpenSSO community members. These might include drafts for review, procedures not yet tested, blog entries, and one pagers culled from information seen on the aliases. If you have an article, or would like to write one, for inclusion on this page, contact dev@opensso.dev.java.net.
  • July 27, '07 : Adding Attributes to the OpenSSO Liberty Personal Profile Service is a procedure written by DocTeger that explains how to extend an identity profile with new attributes.

  • July 26, '07 : Performance Benchmark for SAML2 Protocols Using Federation Manager and Access Manager contains performance numbers that can be used as a reference point for determining how to optimize performance in your SAMLv2/Federation environment.

  • July 26, '07 : Exclusive DSEE6 Multi-Mastered Deployment Considerations contains information concerning multi-master deployments with Directory Server Enterprise Edition. The document is about a year old but still contains some valuable information.

  • July 26, '07 : Building OpenSSO from Netbeans on Mac OS X are the steps that Sebastien Stormacq took to build OpenSSO from within Netbeans.

  • July 23, '07 : Reconfiguring the SAMLv2 Plug-in After Patching is a blog entry by DocTeger that describes how to reconfigure the SAMLv2 plug-in after you have installed one of the recently released patches.

  • June 25, '07 : Developing Secure Applications With Sun Java System Access Manager, Part 1: Basic Authorization describes how to use Sun Microsystems' productized OpenSSO (Access Manager) to fullfil authorization requirements for a fictitious health-care insurance company.

  • June 22, '07 : The OpenSSO WS-Federation How To, written by 'SuperPat' Patterson, describes the sequence of actions required to deploy WS-Federation in OpenSSO.

  • June 22, '07 : com.sun.identity.configuration Package for Service Configurations contains information on the title package.

  • June 22, '07 : Deploying OpenSSO on WebLogic 8.1 contains a modification you must do in order to deploy OpenSSO on WebLogic 8.1.

  • June 19, '07 : Single Logout: A Demo, written by Pat Patterson and Marina Sum, explains the concept of a SAML circle of trust as well as describing a single logout process that terminates user sessions with all the members of the circle of trust.

  • June 12, '07 : Installation of OpenSSO : Hong-Mei May Zhang blogged about how easy it is to install the OpenSSO nightly build on June 5, 2007.

  • June 6, '07 : Securing Site Access With CardSpace and OpenSSO: An Overview : Martin Gee wrote this article for the Sun Developer Network in April of 2007.

  • June 6, '07 : Authentication Login URLs and Redirection URL Precedence : DocTeger has streamlined this section to reflect the attribute names used in the OpenSSO console as opposed to the back-end names used in the source.

  • May 25, '07 : Building Custom Identity Repository Service Plug-ins : DocTeger documented the process you use to build a custom plug-in for the Identity Repository Service.

  • May 24, '07 : Guide to Authentication : The alpha version of this guide is a one-stop shop for all your authentication information needs. DocTeger is compiling the book (with new content) from Sun Microsystems' documentation for Access Manager 7.1. This PDF will open in a new window.

  • May 22, '07 : OpenSSO Command Line Interface Installation : Masoud Kalali wrote these instructions on how to install and use the OpenSSO command line interface administration tool.

  • May 18, '07 : Installing Glassfish for OpenSSO : DocTeger documented the process he used to install Glassfish into which he then installed openfm.war.

  • April 25 '07 : OpenFM Installation on a Macintosh : Andreas Åkre Solberg has written this guide to installing Open Federation and configuring SAML 2.0 entities.

  • Sept. 7 '06 : OpenSSO Nightly Build Installation Guide in Japanese : Tatsuo Kudo has written this guide to installing the nightly build in Japanese.

  • Sept. 7 '06 : OpenSSO Setup in Portuguese : Rodrigo Urubatan has written this blog regarding OpenSSO setup in Portuguese.

  • Sept. 6 '06 : Develop and Deploy Secure Identity Management Solutions : Secure web services in a step-by-step demo with Aravindan Ranganathan, the architect of Sun Microsystems' Access Manager.
