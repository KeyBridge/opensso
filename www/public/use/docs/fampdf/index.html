Sun Federated Access Manager 8.0 Early Access Documentation

This table contains links to PDF files (and one ZIP) of some of the documentation currently being written for the productized version of OpenSSO, Federated Access Manager. Be aware that not all procedures and technical information included has been reviewed or tested. These books have been donated by Sun Microsystems.
PDF files will open in a new window. Download and decompress the ZIP.

Title

Description

Technical Overview

Describes the architecture and how the many features work together to protect resources in your enterprise.

Installation and Configuration Guide

Describes Federated Access Manager installation and configuration tasks.

Developer's Guide

Provides information and procedures for use by engineers and developers.

Administration Guide

Describes various administrative tasks such as Realms Management, Policy Management, Authentication and Directory Management.

Administration Reference

Provides information for administrators including sections on error codes, configuration files and attributes, and command-line tools.

C API Reference

Provides overview information and a look-up listing of the C APIs (including functions, syntax, and parameters).

Java API Reference

Describes the Java API Library. (HTML files compressed in a ZIP file).

3.0 Policy Agent Guide for Application Server 9.1

Provides installation and configuration information about the version 3.0 policy agent for Sun Java System Application Server 9.1 and Glassfish.

3.0 Policy Agent Guide for Web Server 7.0

Provides installation and configuration information about the version 3.0 policy agent for Sun Java System Web Server 7.0.

3.0 Policy Agent Guide for WebLogic Server/Portal 10

Provides installation and configuration information about the version 3.0 policy agent for BEA WebLogic Server/Portal 10.
