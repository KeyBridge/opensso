Sun Federated Access Manager Early Release Documentation

This page is not yet complete. This table contains links to PDF files of early release documentation for Sun Microsystems' productized version of OpenSSO, Federated Access Manager. Be aware that not all procedures and technical information have been reviewed and tested. These books have been donated by Sun Microsystems.
NOTE: You can find the documentation in HTML form on docs.sun.com. The PDFs will open in a new window.
