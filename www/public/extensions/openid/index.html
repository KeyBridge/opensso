OpenID Extension for OpenSSO

Updated: 2007-04-06
by Paul C. Bryan

Introduction

This is an initial release of an OpenID 1.1 identity provider, which uses OpenSSO for authentication and single-signon. It is alpha quality, not yet ready to be included into OpenSSO. When it's ready, the hope is it will be a candidate for inclusion into the core OpenSSO release.
