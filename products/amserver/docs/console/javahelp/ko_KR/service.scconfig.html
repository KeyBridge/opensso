구성 속성

구성 페이지에는 Access Manager의 기본 서비스를 구성하는 모든 속성이 포함되어 있습니다. Access Manager 서비스를 구성하는 속성은 다음 유형 중 하나로 분류됩니다(둘 이상의 유형을 갖는 서비스도 있음).

전역 – Access Manager 구성 전체에 적용됩니다. 전역 속성의 목표는 Identity Server 응용 프로그램을 사용자 정의하는 것이므로 이러한 값은 사용자, 역할 또는 영역에 적용할 수 없습니다.

영역 – 영역 속성은 영역에만 할당됩니다. 또한 영역 속성에는 객체 클래스가 연결되지 않습니다. 인증은 하위 트리 또는 사용자 수준이 아니라 영역 수준으로 수행되기 때문에 인증 서비스에 나열된 속성은 영역 속성으로 정의됩니다.

동적 – 구성된 Access Manager 역할이나 영역에 할당됩니다. 역할이 사용자에게 할당되거나 영역에서 사용자가 만들어질 때 동적 속성이 사용자의 특징이 됩니다.

사용자 – 각 사용자에게 직접 할당됩니다. 이 속성은 역할이나 영역으로부터 상속되지 않으며 일반적으로 각 사용자마다 다릅니다.
