PPLDAP 속성 맵 목록

리버티 개인 프로필 서비스에 의해 정의된 각 Identity 속성에는 LDAP 속성과 일대일로 일치하는 항목이 있습니다. 이 속성 값은 이러한 매핑을 지정하는 목록입니다. 예를 들어, JobTitle=sunIdentityServerPPEmploymentIdentityJobTitle은 Liberty JobTitle 속성을 Access Manager sunIdentityServerPPEmploymentIdentityJobTitle 속성에 매핑합니다. 둘 중 한쪽에 새 속성을 추가하면 이 속성에 새 속성 매핑이 구성됩니다. 속성 매핑 구성을 참조하십시오.
