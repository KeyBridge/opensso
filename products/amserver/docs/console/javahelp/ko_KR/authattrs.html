인증

Access Manager는 일련의 기본 인증 모듈 유형으로 설치됩니다. 인증 모듈 인스턴스는 사용자 아이디 및 비밀번호와 같은 사용자 정보를 수집하고, 데이터베이스의 항목과 비교하여 정보를 확인하며 사용자의 액세스를 허용하거나 거부하는 플러그인입니다. 같은 유형의 여러 인스턴스가 별도로 만들어지고 구성됩니다.
