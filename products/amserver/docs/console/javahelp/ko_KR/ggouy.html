에이전트가 그룹에서 등록 정보를 상속받도록 하려면
시작하기 전에

먼저 에이전트가 속성을 상속할 그룹을 만들어야 합니다.

  1. [구성] 탭의 [에이전트]를 누릅니다.
  2. 필요한 경우 적절한 에이전트 유형에 해당하는 탭을 선택합니다.
    • 웹 에이전트:

      [웹] 탭은 웹 에이전트용이며, 기본적으로 선택됩니다. 다른 선택은 필요하지 않습니다.

    • J2EE 에이전트:

      [J2EE] 탭은 J2EE 에이전트용입니다. 해당하는 경우 이 탭을 누릅니다.

    • 2.2 에이전트:

      [2.2 에이전트] 탭은 정책 에이전트 2.2 소프트웨어 집합에서 에이전트용으로 사용됩니다. 해당하는 경우 이 탭을 누릅니다.

  3. [에이전트] 섹션에서 구성할 에이전트의 이름을 누릅니다.
  4. [전역] 탭이 선택된 상태에서 그룹 속성에 대해 에이전트가 속성을 상속할 그룹의 이름을 선택합니다.

    페이지 맨 위의 [상속 설정] 버튼이 활성화됩니다.

  5. [상속 설정]을 누릅니다.
  6. 에이전트가 그룹에서 상속할 속성(등록 정보)을 선택합니다.
  7. [저장]을 누릅니다.
