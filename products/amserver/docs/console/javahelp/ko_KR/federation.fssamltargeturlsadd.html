대상 URL 구성

다음 하위 속성이 POST에 의해 명제를 받을 각 대상 URL에 대해 정의(또는 수정)될 수 있습니다.

  1. [새로 만들기]를 눌러 새 대상 URL을 추가합니다.

    [새 대상 URL에 새 POST 추가] 페이지가 표시됩니다. 또한 정의된 대상 URL의 [편집] 또는 [복사] 버튼을 선택하여 이 페이지로 이동할 수 있습니다.

  2. 속성의 값을 입력합니다.

    프로토콜

    http 또는 https를 선택합니다.

    서버 이름

    TARGET URL이 있는 서버의 이름입니다(예: www.sun.com).

    포트

    이 속성에는 58080과 같은 포트 번호가 있습니다.

    경로

    이 속성에는 /amserver/console과 같은 URI가 있습니다.

  3. [확인]을 눌러 대상 URL 구성을 완료합니다.

  4. SAML 프로필 페이지에서 [저장]을 눌러 SAML 구성을 완료합니다.
