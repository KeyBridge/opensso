국제화 설정

국제화 설정 서비스에는 여러 로켈 및 문자 집합에 대해 Access Manager를 구성할 수 있는 전역 속성이 포함되어 있습니다. 속성은 다음과 같습니다.

  • 각 로켈이 지원하는 문자 집합

  • 문자 집합 별칭

  • 자동 생성된 공통 이름 형식

각 로켈이 지원하는 문자 집합

이 속성은 각 로켈에서 지원하는 문자 집합을 나열하며 로켈과 문자 집합 간의 매핑을 나타냅니다. 형식은 다음과 같습니다.

지원되는 새 문자 집합을 추가하려면 [추가]를 누르고 다음 매개 변수를 정의합니다.

로켈

추가하려는 새 로켈입니다. 자세한 내용은 지원되는 언어 로켈을 참조하십시오.

지원되는 문자 집합

특정 로켈에서 지원되는 문자 집합을 입력합니다. 각 문자 집합을 세미콜론으로 구분합니다. 예를 들면 charset=charset1;charset2;charset3;...;charsetn과 같습니다.

기존의 지원되는 문자 집합을 편집하려면 지원되는 문자 집합 테이블에서 해당 이름을 누릅니다. 작업을 마치면 [확인]을 누릅니다.

문자 집합 별칭

이 속성은 응답을 보내는 데 사용되는 코드 집합 이름(IANA 이름에 매핑됨)을 나열합니다. 이러한 코드 집합 이름은 Java 코드 집합 이름과 일치할 필요가 없습니다. 현재는 Java 문자 집합을 IANA 문자 집합으로 또는 그 반대로 매핑하기 위한 해시 테이블이 있습니다.

새 문자 집합 별칭을 추가하려면 [추가] 버튼을 누르고 다음 매개 변수를 정의합니다.

MIME 이름

IANA 매핑 이름입니다. 예를 들면 Shift_JIS와 같습니다.

Java 이름

IANA 문자 집합에 매핑할 Java 문자 집합입니다.

기존의 문자 집합 별칭을 편집하려면 테이블에서 해당 이름을 누릅니다. 작업을 마치면 [확인]을 누릅니다.

자동 생성된 공통 이름 형식

이 디스플레이 옵션을 사용하면 다른 로켈 및 문자 집합에 대한 이름 형식을 적용하여 이름을 자동으로 생성하는 방법을 정의할 수 있습니다. 기본 구문은 다음과 같습니다(정의에 포함된 쉼표 및/또는 공백이 이름 형식에 표시됨).

en_us = {givenname} {initials} {sn}

예를 들어, 사용자(User One)의 새 이름 형식을 중국어 문자 집합의 uid (11111)를 사용하여 표시하려면 다음과 같이 정의합니다.

zh = {sn}{givenname}({uid})

다음과 같이 표시됩니다.

OneUser 11111
