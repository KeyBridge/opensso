서비스 설명 구성

서비스 설명 속성은 개별 프로토콜 종점에서 실행 중인 웹 서비스를 정의합니다. 이 속성은 부트스트랩용 자원 오퍼링을 구성할 때 정의됩니다. 서비스 인스턴스에 대한 정보는 다양한 컨텍스트에서 통신이 가능해야 합니다. 예를 들어, 정의된 검색 서비스는 자원 오퍼링(각각 서버 인스턴스 설명을 포함)을 열거하는 Identity 서비스입니다.

  1. [설명 추가]를 눌러 새 서비스 설명 속성을 표시하거나 [편집]을 눌러 기존 설명을 수정합니다.

  2. 다음 정보를 기반으로 속성에 대한 값을 제공합니다.

    보안 체계 아이디

    이 속성에는 검색 서비스가 지원하는 인증 방법이 추가됩니다. 이 보안 기법은 웹 서비스 사용자가 웹 서비스 공급자에게 인증되는 방법 또는 메시지 수준 보안을 제공하는 방법을 나타냅니다. 기본적으로 서비스 인스턴스가 지원하는 사용 가능한 모든 방법이 선택됩니다. 인증 방법을 선택하지 않은 경우 웹 서비스 사용자가 해당 방법을 사용하여 요청을 보내면 요청이 거부됩니다. 지원되는 인증 기법을 참조하십시오.

    종점 URL

    이 속성 값은 SOAP-over-HTTP 종점의 URL입니다. URI 체계는 https://soap.profile-provider.com/soap과 같이 HTTP 또는 HTTPS여야 합니다.

    SOAP 작업

    이 옵션 속성의 값은 서비스의 구체적인 WSDL 기반 설명에 있는 wsdlsoap:operation 요소의 wsdlsoap:soapAction 속성과 같습니다.

  3. [확인]을 눌러 서비스 구성을 완료합니다.

  4. 검색 서비스 페이지에서 [저장]을 눌러 서비스 구성을 완료합니다.
