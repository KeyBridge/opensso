웹 서비스

웹 서비스 모듈은 Liberty Alliance Project ID-WSF(Identity Web Services Framework) 사양의 Access Manager 구현입니다. ID-WSF는 Liberty ID-FF(Identity Federation Framework)를 토대로 구축되며 연합 네트워크 환경에서 Identity 데이터를 검색, 업데이트 또는 기타 작업을 수행하는 웹 서비스를 구축하기 위한 사양을 제공합니다. 이 사양은 달력 서비스, 전자 지갑 서비스 또는 경고 서비스와 같은 웹 서비스를 구축하는 데 필요한 기술 구성 요소를 간략하게 설명합니다.

주 – Identity 데이터에는 이름, 성, 집 주소, 직장 주소 및 비상 연락처를 정의하는 속성과 기타 속성이 포함될 수 있습니다.

웹 서비스는 인터넷을 통한 분산 컴퓨팅의 기본입니다. 웹 서비스 사용자(WSC)는 웹 서비스를 찾고 요청을 보내 웹 서비스가 제공하는 작업을 호출합니다. 웹 서비스 공급자(WSP)는 요청에 따라 웹 서비스를 구현하는 응용 프로그램으로 WSC와 동일한 Java™ 가상 시스템에 있을 수도 있고 수천 마일 떨어진 곳에 있을 수도 있습니다. WSC가 WSP에서 Identity 속성을 검색해야 하는 경우 먼저 검색 서비스에 연결하여 특정 속성이 저장된 위치를 찾아야 합니다. 이 정보가 반환되면 WSC는 WSP(예: 개인 프로필 서비스)에 연결하여 필요한 속성을 검색합니다.

Access Manager 콘솔의 웹 서비스 모듈은 다음을 구성하기 위한 인터페이스를 제공합니다.

  • 리버티 개인 프로필 서비스

  • 검색 서비스

  • SOAP 바인드 서비스

  • 인증 웹 서비스
