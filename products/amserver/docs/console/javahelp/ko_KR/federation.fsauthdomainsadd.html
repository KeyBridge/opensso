새 인증 도메인 작성

다음 절차에 따라 새 인증 도메인을 작성합니다. 시작 위치는 연합 인터페이스 아래의 인증 도메인입니다.

  1. [새로 만들기]를 눌러 인증 도메인 속성을 표시합니다.

    새 인증 도메인 프로필 페이지가 표시됩니다.

  2. 인증 도메인의 이름을 입력합니다.

  3. [설명] 필드에 인증 도메인에 대한 설명을 입력합니다.

  4. 작성기 서비스 URL의 값을 입력합니다.

    작성기 서비스 URL은 공통 도메인 쿠키를 작성하는 서블릿의 위치를 지정합니다. http://common-domain-host :port/common/writer 형식을 사용합니다.

  5. 판독기 서비스 URL의 값을 입력합니다.

    판독기 서비스 URL은 공통 도메인 쿠키를 읽는 서블릿의 위치를 지정합니다. http://common-domain-host :port/common/transfer 형식을 사용합니다.

  6. [활성] 또는 [비활성]을 선택합니다.

    기본 상태는 [활성]입니다. [비활성]을 선택하면 인증 도메인 내의 통신을 사용할 수 없게 됩니다.

  7. [확인]을 눌러 구성을 완료합니다.

    새 인증 도메인이 인증 도메인 화면에 표시됩니다.
