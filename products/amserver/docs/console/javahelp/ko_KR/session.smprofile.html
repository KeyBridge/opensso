세션 창

현재 Access Manager에 로그인한 사용자와 각 로그인의 세션 시간을 표시합니다. 표시 필드는 다음과 같습니다.

사용자 아이디

현재 로그인한 사용자의 사용자 아이디를 표시합니다.

남은 시간(분)

사용자가 재인증을 수행하기 전에 해당 세션에 대해 남은 시간(분)을 표시합니다.

최대 세션 시간(분)

세션이 만료되고 액세스 권한을 다시 얻기 위해 재인증을 수행해야 하기 전까지 사용자가 로그인할 수 있는 최대 시간(분)을 표시합니다.

유휴 시간(분)

사용자가 유휴 상태인 시간(분)을 표시합니다.

최대 유휴 시간(분)

사용자가 재인증을 수행해야 하기 전까기 유휴 상태로 있을 수 있는 최대 시간(분)을 표시합니다. 시간 제한은 관리자가 세션 서비스에서 정의합니다. 세션 속성 정의는 세션을 참조하십시오.
