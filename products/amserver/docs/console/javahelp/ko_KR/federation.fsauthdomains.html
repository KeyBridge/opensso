인증 도메인

인증 도메인은 기본 사용자가 안전하고 완벽하게 통합된 환경에서 비즈니스를 함께 수행할 수 있는 서비스 공급자(및 하나 이상의 Identity 공급자)의 연합입니다. 인증 도메인을 만들고 채우려면 먼저 인증 도메인의 구성원이 될 각 공급자에 대해 메타데이터(특정 Identity 서비스 아키텍처를 정의하는 구성 정보)를 저장하는 엔티티를 만듭니다. 그런 다음, 인증 도메인을 구성 및 저장합니다. 마지막으로 인증 도메인에 추가할 엔티티(구성된 공급자)의 등록 정보를 편집합니다.

주 – 엔티티에 대한 자세한 내용은 엔티티: 공급자 및 계열사를 참조하십시오.

구성된 모든 인증 도메인은 인증 도메인에 나열됩니다. 인증 도메인과 연관된 작업은 다음과 같습니다.

  • 새 인증 도메인 작성

  • 인증 도메인 프로필 수정

  • 인증 도메인에 공급자 추가

  • 인증 도메인 프로필 삭제

주 – 인증 도메인의 구성원은 Liberty Trust Model Guidelines에 기초한 circle of trust 운영 협약을 미리 설정해야 합니다.
