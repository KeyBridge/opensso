정책에 조건 추가
  1. 이미 정책을 만든 경우 조건을 추가하려는 정책의 이름을 누릅니다. 정책을 아직 만들지 않은 경우 새 일반 정책 만들기를 참조하십시오.
  2. 조건 목록에서 [새로 만들기]를 누릅니다.
  3. 조건 유형을 선택하고 [다음]을 누릅니다.
  4. 조건 유형의 필드를 정의합니다. 조건 유형에 대한 설명을 보려면 다음 중 하나를 누르십시오.

    • 활성 세션 시간

    • 인증 체인

    • 인증 수준(보다 크거나 같음)

    • 인증 수준(보다 낮거나 같음)

    • 인증 모듈 인스턴스

    • 현재 세션 등록 정보

    • IP 주소/DNS 이름

    • LDAP 필터 조건

    • 영역 인증

    • 시간(요일, 날짜, 시간 및 표준 시간대)

  5. [마침]을 누릅니다.
