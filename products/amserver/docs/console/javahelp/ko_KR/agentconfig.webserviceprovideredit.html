웹 서비스 공급자

웹 서비스 공급자 에이전트 프로필은 웹 서비스 공급자에서 보낸 웹 서비스 요청을 보호하기 위해 사용하는 구성을 설명합니다. 웹 서비스의 이름은 각 웹 서비스 공급자에 대해 고유해야 합니다.

상태

웹 서비스 공급자 에이전트가 아웃바운드 서비스 요청 보호에 참여할지 여부를 정의합니다. 기본적으로 활성으로 설정되며, 이는 아웃바운드 서비스 요청 보호에 참여함을 의미합니다.

보안

다음 속성은 웹 서비스 공급자 보안 속성을 정의합니다.

보안 메커니즘

웹 서비스 공급자 요청을 보호하는 데 사용할 보안 자격 증명 유형을 정의합니다. 다음 보안 유형 중에서 선택할 수 있습니다.

  • 익명 — 익명 보안 메커니즘에 보안 자격 증명이 포함되지 않습니다.

  • LibertyBearerToken

  • LibertySAMLToken

  • LibertyX509Token — Libery 정의 X509 자격 증명을 사용하여 웹 공급자 요청을 보호합니다.

  • SAML-HolderOfKey — SAML 1.1 명제 유형 Holder-Of-Key를 사용하여 요청을 보호합니다.

  • SAML-SenderVouches — SAML 1.1 명제 유형 Sender Vouches를 사용하여 요청을 보호합니다.

  • SAML2–HolderOfKey — SAML 2.0 명제 토큰 유형 Holder-Of-Key를 사용하여 요청을 보호합니다.

  • SAML2–SenderVouches — SAML 2.0 명제 토큰 유형 Sender Vouches를 사용하여 요청을 보호합니다.

  • UserNameToken — 사용자 이름 토큰을 사용하여 웹 공급자 클라이언트 요청을 보호합니다.

  • UserNameToken-Plain — 일반 텍스트 비밀번호가 있는 사용자 이름 토큰을 사용하여 웹 공급자 요청을 보호합니다.

  • X509Token — X509 인증서를 사용하여 웹 공급자 요청을 보호합니다.

인증 체인

정의 –

메시지의 보안 헤더 보존

이 속성을 사용하면 향후 처리를 위해 웹 서비스 공급자가 SOAP 보안 헤더를 보존합니다.

리버티 서비스 유형 URN

URN(Universal Resource Name)은 웹 서비스 공급자가 서비스 조회에 사용할 리버티 서비스 유형을 설명합니다.

사용자 토큰의 자격 증명

이 속성은 웹 서비스 공급자가 사용자 이름 보안 토큰을 생성하는 데 사용할 사용자 이름/비밀번호 공유 비밀을 나타냅니다.

개인 키 유형

정의 –

SAML 구성

다음 속성은 웹 서비스 공급자의 SAML(Security Assertion Markup Language)을 구성합니다.

SAML 속성 매핑

이 속성은 보안 토큰 서비스에서 명제를 만드는 동안 생성해야 하는 SAML 속성을 나타냅니다. 동일한 매핑이 웹 서비스 공급자가 SSOToken에 해당 속성을 적용하는 데 사용됩니다.

SAML NameID Mapper 플러그인

SAML 계정 매핑에 사용하는 NameID 매퍼 플러그인 클래스를 정의합니다.

SAML 속성 이름 공간

SAML 속성 생성에 사용하는 이름 공간을 정의합니다.

구성원 포함

이 속성은 기본의 구성원을 SAML 속성으로 포함해야 함을 정의합니다.

서명 및 암호화

다음 속성은 웹 공급자 보안에 대한 서명 및 암호화 구성을 정의합니다.

응답 서명됨

이 속성을 사용하면 웹 서비스 공급자가 지정된 토큰 유형을 사용하여 응답을 서명합니다.

응답 암호화됨

이 속성을 사용하면 웹 서비스 응답이 암호화됩니다.

요청 서명 확인됨

이 속성을 사용하면 웹 서비스 요청 서명을 확인합니다.

요청 헤더 암호화됨

이 속성을 사용하면우 웹 서비스 클라이언트 보안 헤더가 암호화됩니다.

요청 해독됨

이 속성을 사용하면 웹 서비스 공급자 요청이 해독됩니다.

키 저장소

다음 속성은 인증서 저장소 및 검색에 사용할 키 저장소를 구성합니다.

웹 서비스 클라이언트의 공개 키 별칭

이 속성은 웹 서비스 응답을 암호화하는 데 사용하는 공개 인증서 키 별칭을 정의합니다.

개인 키 별칭

이 속성은 웹 서비스 요청을 서명/해독하는 데 사용하는 개인 인증서 키 별칭을 정의합니다.

키 저장소 사용

이 구성은 기본 키 저장소를 사용할지 또는 사용자 정의 키 저장소를 사용할지 정의합니다. 사용자 정의 키 저장소에 대해 다음 값을 정의해야 합니다.

  • 키 저장소 위치

  • 키 저장소 비밀번호

  • 키 비밀번호

웹 서비스 엔드포인트
