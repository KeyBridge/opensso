검색 에이전트

검색 에이전트 프로필에는 웹 서비스의 클라이언트/프로필에서 웹 서비스 조회, 등록 및 보안 자격 증명 획득을 위해 리버티 검색 서비스와 통신하는 데 사용할 신뢰 기관 구성이 포함되어 있습니다.

그룹

그룹 메커니즘을 사용하면 유사한 에이전트 유형 모음을 정의할 수 있습니다. 그룹은 특정 에이전트를 에이전트 유형 모음에 포함하기 전에 정의해야 합니다.

검색 에이전트의 비밀번호를 정의합니다.

비밀번호를 확인합니다.

에이전트 구성 저장소 위치

이 속성은 검색 에이전트에 대한 구성 저장소의 에이전트 위치를 정의합니다.

개인 키 별칭

이 속성은 신뢰 기관 클라이언트가 검색 서비스에 대한 SOAP 메시지 요청에 서명하는 데 사용할 인증서 별칭을 정의합니다.

검색 서비스 엔드포인트

이 속성은 신뢰 기관 클라이언트가 서비스 등록 및 조회를 위해 통신을 연결하는 검색 서비스 엔드포인트를 정의합니다.

인증 웹 서비스 엔드포인트

이 속성은 웹 서비스 클라이언트가 검색 서비스 자원 오퍼링(부트스트랩 자원 오퍼링이라고도 함)을 수신하기 위해 최종 사용자의 SSO 토큰을 사용하여 인증하는 데 사용하는 인증 서비스 엔드포인트를 정의합니다.
