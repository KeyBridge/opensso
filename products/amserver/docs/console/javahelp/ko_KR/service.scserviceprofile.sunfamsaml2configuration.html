SAMLv2 서비스 구성

캐시 정리 간격

이 속성은 캐시 정리 간격(초)을 지정합니다.

이름 아이디 정보의 속성 이름

사용자의 항목에 이름 식별자 정보를 저장하는 데 사용되는 속성 이름을 지정합니다. 값을 지정하지 않은 경우 기본 속성(sun-fm-saml2-nameid-info)이 사용됩니다.

이름 아이디 정보 키의 속성 이름

사용자의 항목에 이름 식별자 키를 저장하는 데 사용되는 속성 이름을 지정합니다. 값을 지정하지 않은 경우 기본 속성(sun-fm-saml2-nameid-infokey)이 사용됩니다.

IDP 검색 서비스의 쿠키 도메인

SAMLv2 IDP 검색 서비스에서 사용할 쿠키 도메인을 지정합니다.

IDP 검색 서비스의 쿠키 유형

SAMLv2 IDP 검색 서비스에서 사용할 쿠키 유형(영구적 또는 세션)을 지정합니다. 기본값은 세션입니다.

IDP 검색 서비스의 URL 체계

SAMLv2 IDP 검색 서비스에서 사용할 URL 체계를 지정합니다.

SML 암호화 SPI 구현 클래스

SAMLv2 암호화 공급자 인터페이스의 구현 클래스 이름을 지정합니다. 이 클래스는 SAMLv2 프로필에서 XML 암호화 및 암호 해독을 수행하는 데 사용합니다.

KeyInfo 요소 내에 암호화 키 포함

이 속성은 com.sun.identity.saml2.xmlenc.FMEncProvider 클래스에서 사용됩니다. 이 속성을 사용하면 XML 암호화 작업 수행 시 EncryptedData 요소의 KeyInfo에 EncryptedKey가 포함됩니다. 이 속성을 사용하지 않으면 EncryptedKey가 EncryptedData 요소와 병렬로 처리됩니다. 기본값은 사용하는 것입니다.

XML 서명 구현 클래스

SAMLv2 서명 공급자 인터페이스의 구현 클래스 이름을 지정합니다. 이 클래스는 SAMLv2 프로필에서 XML 서명 및 검증을 수행하는 데 사용합니다.

XML 서명 인증서 유효성 검사

이 속성을 사용하면 SAML Identity 공급자 또는 서비스 공급자가 서명에 사용된 인증서의 유효성을 검사합니다. 인증서의 유효성을 검사하여 승인되면 공급자가 서명의 유효성을 검사합니다. 그렇지 않은 경우 유효성 검사를 중지하고 오류를 반환합니다.

CA 인증서 유효성 검사

이 속성을 사용하면 SAML Identity 공급자 또는 서비스 공급자가 서명에 사용된 인증서와 함께 CA 인증서의 유효성을 검사합니다. 인증서의 유효성을 검사하는 경우 공급자는 서명의 유효성을 검사합니다. 그렇지 않은 경우 유효성 검사를 중지하고 오류를 반환합니다.
