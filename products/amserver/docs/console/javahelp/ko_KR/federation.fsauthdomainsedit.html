인증 도메인 프로필 수정

다음 절차에 따라 기존 인증 도메인의 구성된 일반 속성을 편집하거나 인증 도메인에 공급자를 추가합니다. 시작 위치는 연합 인터페이스 아래의 인증 도메인입니다.

  1. 구성된 인증 도메인의 이름을 눌러 해당 프로필을 수정하거나 공급자를 추가합니다.

    인증 도메인 편집 페이지가 표시됩니다.

  2. 새 값을 입력하거나 인증 도메인의 일반 속성에 대한 기존 값을 편집합니다.
    이름

    이 속성의 정적 값은 인증 도메인을 작성할 때 제공된 이름입니다.

    설명

    이 속성 값은 인증 도메인의 설명입니다. 적용되는 경우 이미 입력한 설명을 수정할 수 있습니다.

    작성기 서비스 URL

    이 속성은 공통 도메인 쿠키를 작성하는 서비스의 위치를 지정합니다. URL의 형식은 http://common-domain-host:port/common/writer입니다.

    판독기 서비스 URL

    이 속성은 공통 도메인 쿠키를 판독하는 서비스의 위치를 지정합니디. URL의 형식은 http://common-domain-host:port/common/transfer입니다.

    상태

    기본 상태는 [활성]입니다. [비활성]을 선택하면 인증 도메인 내의 통신을 사용할 수 없게 됩니다.

  3. [추가]를 눌러 인증 도메인에 공급자를 추가합니다.

    신뢰할 수 있는 파트너 선택 페이지가 표시됩니다.

  4. 하나 이상의 사용 가능한 공급자를 선택하고 [추가] 화살표를 눌러 선택합니다.

    제공된 목록에는 공급자로 작성하고 채운 엔티티의 이름이 포함되어 있습니다. 자세한 내용은 인증 도메인에 공급자 추가를 참조하십시오.

  5. [확인]을 눌러 인증 도메인에 공급자를 저장합니다.

    공급자가 저장되면 인증 도메인 프로필 화면으로 돌아갑니다.

  6. [저장]을 눌러 작업을 완료합니다.
