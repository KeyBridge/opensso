Data Stores

A data store is a database where you can store user attributes and user configuration data.

Access Manager provides an identity repository plug-in that connects to an LDAPv3 identity repository framework. This new model enables you to view and retrieve Access Manager user information without having to make changes in your existing user database. The Access Manager framework integrates data from the identity repository plug-in with data from other Access Manager plug-ins to form a virtual identity for each user. Access Manager can then use the universal identity in authentication and authorization processes among more than one identity repository. The virtual user identity is destroyed when the user's session ends.

See To Create a New LDAPv3 Data Store for information on creating a new data store and configuring the repository plug-in.

If you have installed Access Manager in Realms Mode, you can configure the Access Manger SDK to manage Directory Server data. See To Create a New Access Manager Repository Plugin.
