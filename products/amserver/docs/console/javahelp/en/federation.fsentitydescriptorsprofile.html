To Modify the General Attributes of a Provider Entity or an Affiliate Entity

After creating an entity, you can edit its profile. This might include modifying attributes, adding identity providers and service providers, or configuring affiliations. Editing the General attributes of an entity profile might entail modifying the already-defined Common attributes, adding a contact person, or providing optional organizational information. The following procedure is for editing the General attributes. The starting point is the Entities screen under Federation.

  1. Select the name of a configured entity to modify its profile.

    The General attribute page of the entity's profile is displayed, by default. If you are not on the General attribute page, you can select General from the View menu.

  2. Edit the values of any of the Entity Common Attributes.
    Entity Name

    The static value of this attribute is the name provided when you created the entity.

    Type

    The static value of this attribute is the type of entity, Provider or Affiliate.

    Description

    The value of this attribute is the description provided when you created the entity. You may modify the description originally entered or add one if none exists.

  3. Provide values for the Entity Contact Person Profile attributes.

    The Entity Contact Person Profile attributes provide information on who to contact regarding the entity. These attributes are optional.

    First Name

    Type the given name of the entity’s contact person.

    Last Name

    Type the surname of the entity’s contact person.

    Type

    Choose the type of contact from the drop-down menu:

    • Administrative

    • Billing

    • Technical

    • Other

    Company

    Type the name of the company that employs this person.

    Liberty Principal ID

    Type a uniform resource identifier (URI) that points to an instance of the contact person’s personal information profile.

    Emails

    Type one or more email addresses for the contact person and click Add.

    Telephone Numbers

    Type one or more telephone numbers for the contact person and click Add.

  4. Provide values for the Organization Profiles attributes.

    The Organizations Profile attributes provide basic information that may be required when interacting with a principal. These attributes are optional.

    Note – If the Names attribute contains a value, it is required to add values to the Display Names and URL attributes.

    Names

    Enter the legal name for the entity’s organization and click Add. The value is defined in the format locale| organization-name. For example, en| example.com.

    Display Names

    Enter a name for the entity’s organization that is suitable for display to a principal and click Add. The value is defined in the format locale|organization-display-name . For example, en|example.com.

    URL

    Enter the URL of the organization and click Add. The value is defined in the format locale|organization-URL . For example, en|http://www.example.com .

  5. Click Save to complete the configuration, or continue defining values for Identity Provider, Service Provider or Affiliate attributes.

    If continuing your configuration, see:

    • To Add an Identity Provider to a Provider Entity to define values for Identity Provider attributes.
    • To Add a Service Provider to a Provider Entity to define values for Service Provider attributes.
    • To Add Members to an Affiliate Entity to populate an affiliate entity with members.
