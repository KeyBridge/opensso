To Create a New Normal Policy

Steps

  1. Choose the realm for which you would like to create a policy.

  2. Click New Policy from the Policies list.

  3. Add a name and a description for the policy.

  4. If you wish the policy to be active, select Yes in the Active attribute.

  5. It is not necessary to define all of the fields for normal policies at this time. You may create the policy, then add rules, subjects, conditions, and response providers later. If you wish to define the fields now, see the following topics:

    • To Add a Rule to a Policy

    • To Add a Subject to a Policy

    • To Add a Condition to a Policy

    • To Add a Response Provider to a Policy

  6. Click Create.
