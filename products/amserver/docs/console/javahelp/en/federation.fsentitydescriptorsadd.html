To Create a Provider Entity or an Affiliate Entity

Establishing an entity is a two-step process. First, you create the entity. (This is when you define whether it is a provider entity or an affiliate entity.) After creating the entity, you populate it with providers. The following procedure creates a new entity. The starting point is Entities under the Federation interface.

Note – An entity can be created but it will not be available for assignment to a circle of trust until it has been populated with providers.

  1. Click New to display the entity attributes.

    The New Entity page is displayed.

  2. Type a name for the entity.

    The Entity Name attribute defines a uniform resource identifier (URI) for the entity. This value must be unique among entity names.

  3. Type a description of the entity.
  4. Choose the type of entity.

    • If creating a provider entity, select Provider.

    • If creating an affiliate entity, select Affiliate, and define values for the required Affiliate Name and Affiliate Owner attributes.

      • Affiliate Name defines a URI for the affiliate entity. This value must be unique among affiliate names.

      • Affiliate Owner specifies a provider as the parent administrator of the affiliation. The value of this attribute is a URI that would match the value defined in the Entity Name attribute of the entity to which the affiliate owner/provider belongs.

      Note – Defining a provider as the Affiliate Owner does not automatically include it as a member of the affiliation. If an owner is also a member, the URI must be defined here as the owner and within the affiliate entity itself as a member. To add members to an affiliation, see To Add Members to an Affiliate Entity.

  5. Click OK.

    The new entity is displayed on the Entities screen.
