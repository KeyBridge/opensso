Session Window

Displays the users that are currently logged in to Access Manager and the session time for each login. The display fields are:

User ID

Displays the user ID of the user who is currently logged in.

Time Remaining (minutes)

Displays the amount of time (in minutes) remaining that the user has for that session before having to reauthenticate.

Max Session Time (minutes)

Displays the maximum time (in minutes) that the user can be logged in before the session expires and must reauthenticate to regain access.

Time Idle (minutes)

Displays the time (in minutes) that the user has been idle.

Max Idle Time (minutes)

Displays the maximum time (in minutes) that a user can remain idle before having to reauthenticate. The time limits are defined by the administrator in the Session service. See Session for session attribute definitions.
