讓代理程式從群組繼承特性
開始之前

您必須先建立群組，才能讓代理程式從其中繼承屬性。

  1. 按一下 [配置] 標籤下的 [代理程式]。
  2. 如有必要，請選取適當代理程式類型的標籤。
    • Web 代理程式：

      [Web] 標籤用於 Web 代理程式，且為預設選項。不需要再進行選取。

    • J2EE 代理程式：

      [J2EE] 標籤用於 J2EE 代理程式。如果適用，請按一下此標籤。

    • 2.2 代理程式：

      [2.2 代理程式] 標籤用於「策略代理程式 2.2」軟體集中的代理程式。如果適用，請按一下此標籤。

  3. 在 [代理程式] 區段，按一下要配置的代理程式名稱。
  4. 在選取 [全域] 標籤的情況下，針對標示為「群組」的屬性選取要讓代理程式繼承其屬性的群組名稱。

    在頁面頂端，[繼承性設定] 按鈕會變成作用中。

  5. 按一下 [繼承性設定]。
  6. 選取要讓代理程式從群組繼承的屬性 (特性)。
  7. 按一下 [儲存]。
