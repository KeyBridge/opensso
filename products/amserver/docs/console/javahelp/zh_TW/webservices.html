Web 服務

Web 服務模組是 Liberty Alliance Project 識別 Web 服務架構 (Identity Web Services Framework, ID-WSF) 規格的 Access Manager 實作。ID-WSF 建立在 Liberty 識別聯合架構 (Identity Federation Framework, ID-FF) 之上，方法為提供規格以建立對聯合網路環境中的識別資料進行擷取、更新或執行動作之服務。這些規格概述了建立 Web 服務所需的技術元件，如行事曆服務、錢包服務或警示服務。

備註 – 識別資料包括定義名字、姓氏、家庭住址、公司地址及緊急連絡人的屬性，但不限於此。

Web 服務是跨網際網路的分散式運算的基礎。Web 服務使用者 (WSC) 會提出請求，尋找 Web 服務的位置並呼叫它提供的作業。Web 服務提供者 (WSP) 是實作基於請求的 Web 服務的應用程式；它可以位於與 WSC 相同的 Java™ 虛擬機器，也可以在數千英哩以外。當 WSC 需要從 WSP 擷取識別屬性時，首先必須連絡探索服務，以尋找儲存特殊屬性的位置。當傳回此資訊時，WSC 隨後會連絡 WSP (例如，個人設定檔服務) 以擷取必需的屬性。

Access Manager 主控台中的 Web 服務模組提供了用於配置下列各項的介面：

  • Liberty 個人設定檔服務

  • 探索服務

  • SOAP 連結服務

  • 認證 Web 服務
