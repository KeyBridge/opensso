範圍

存取控制範圍是一組可與使用者或使用者群組產生關聯的認證特性與授權策略。範圍資料儲存於一個專有權資訊樹狀結構中，其為 Access Manager 於您指定的資料存放區中所建立。Access Manager 架構於 Access Manager 資訊樹狀結構中聚集每一個範圍中的策略與特性。依預設，Access Manager 7.1 會將 Access Manager 資訊樹狀結構做為特殊分支插入到 Sun Java Enterprise System Directory Server 中，但使用者資料除外。當使用任何 LDAPv3 資料庫時，您可以使用存取控制範圍。

於 [範圍] 標籤中，您可為存取控制配置下列特性：

  • 一般特性

  • 認證

  • 服務
