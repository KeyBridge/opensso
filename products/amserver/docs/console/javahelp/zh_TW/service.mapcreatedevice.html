新增用戶端

  1. 在 [用戶端類型] 清單中按一下 [新建]。

  2. 利用下列欄位選取裝置類型：

    樣式

    顯示裝置的基本樣式。例如 HTML。

    裝置使用者代理程式

    接受裝置的名稱。

  3. 按 [下一步]。

  4. 為新裝置輸入下列資訊：

    用戶端類型名稱

    接受裝置的名稱。名稱在所有裝置中必須是唯一的。

    HTTP 使用者字串

    定義 HTTP 請求標頭中的使用者代理程式。例如 Mozilla/4.0。

  5. 按一下 [完成]。

  6. 若要複製裝置及其特性，請按一下 [複製] 連結。裝置名稱必須唯一。依預設，Access Manager 會將此裝置重新命名為 copy_of_devicename。
