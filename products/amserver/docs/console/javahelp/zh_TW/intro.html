簡介

Sun Java System Access Manager 可防止對 Web 服務應用程式與 Web 內容未經授權的存取。Access Manager 整合認證與授權服務、策略代理程式及識別聯合以提供保護網路資源的完整解決方案。

許多企業基於每個應用程式對資訊授予權限。例如，員工可能必須設定一個使用者名稱與密碼來存取公司的健康津貼管理網頁。相同的員工必須使用不同的使用者名稱與密碼來存取會計部門的線上表格。於相同的企業中，客戶可設定使用者名稱與密碼來存取公司網頁上的公共部門。對於每個網頁或服務，管理員必須將企業使用者的輸入轉換為服務可以辨識的資料格式。每個新增至企業的服務必須分別提供與維護。

Access Manager 可以降低管理成本，並消除與每個應用程式解決方案相關聯的備用使用者資訊。Access Manager 可使管理員指定特定的規則或策略，用於控制每個使用者可存取的資訊或服務。策略代理程式可部署於應用程式或 Web 伺服器之上，用於處理 HTTP 請求並強制執行使用中的策略。使用者資訊與相關聯的存取策略一起構成使用者的企業識別。Access Manager 可使使用者僅以一個識別存取企業中的許多資源。
