配置支援的容器

容器是一種定義識別資料片段保存器的屬性。以下是新增屬性至 Liberty 個人設定檔服務 的程序。起點是 Web 服務模組內的 Liberty 個人設定檔服務 配置畫面。

  1. 按一下 [新建] 以顯示新的支援容器屬性，或按一下已配置的容器名稱以修改其設定檔。

  2. 根據下列資訊提供容器屬性值：

    容器名稱

    輸入容器的名稱，例如 CreditCard。

    外掛程式

    輸入要處理整個容器的類別名稱。這可以用來置換預設實作 com.sun.identity.liberty.ws.idpp.plugin.IDPPContainer。

  3. 按一下 [確定] 以完成容器配置。

  4. 按一下Liberty 個人設定檔服務 頁面上的 [儲存] 以完成服務配置。
