目錄配置

[目錄配置] 屬性定義內嵌式目錄存放區的基本配置資訊：

目錄配置

[目錄配置] 屬性包括：

最小連線池

指定用於連線至 Directory Server 的連線池最小大小，如 LDAP 伺服器屬性中指定的最小大小。預設值為 1。

最大連線池

此屬性指定用於連線至 Directory Server 的連線池最大大小，如 LDAP 伺服器屬性中指定的最大大小。預設值為 10。

連結 DN

指定 LDAP 伺服器中的連結 DN。

連結密碼

定義用於連結至 LDAP 伺服器的密碼。依預設，將使用安裝期間輸入的 amldapuser 密碼做為連結使用者。

舊有配置

下列屬性會針對 Federated Access Manager 的舊有模式實例定義基本目錄伺服器配置。

最小連線池

指定用於連線至 Directory Server 的連線池最小大小，如 LDAP 伺服器屬性中指定的最小大小。預設值為 1。

最大連線池

此屬性指定用於連線至 Directory Server 的連線池最大大小，如 LDAP 伺服器屬性中指定的最大大小。預設值為 10。

伺服器

此屬性會列出負載平衡器協定、主機名稱和連接埠。例如：http://lb.example.com:80。
