請求處理程式清單

SOAP 連結服務提供 RequestHandler 介面，以處理請求訊息並傳回一個回應。每一個使用 SOAP 連結服務並基於 Liberty 的識別服務都必須在伺服器端實作此介面。請求處理程式清單屬性會儲存 Web 服務 (用來執行請求處理程式) 實作類別的相關資訊。請求處理程式清單會顯示包含以管道分隔的鍵/值對的項目，如下所示：key=disco|class=com.example.identity.liberty.ws.disco.DiscoveryService。下表將說明各參數。

參數

標頭

key

必需的 key 參數是 SOAP 端點的 URI 路徑的最後部份。Access Manager 中的 SOAP 端點是 SOAP 收件者 Servlet。SOAP 接收器的 URI 為：protocol://host:port /deploy-uri/Liberty/key。如果您將 disco 定義為 key，則對應探索服務的 SOAP 端點的 URI 路徑將是：protocol://host:port /amserver/Liberty/disco。當連接到 SOAP 接收器時，不同的服務用戶端會使用不同的鍵值。

class

必需的 class 參數指定特殊識別服務的請求處理程式實作類別的名稱。例如：class=com.example.identity.liberty.ws.disco.DiscoveryService。

若要新增請求處理程式，請參閱配置請求處理程式。
