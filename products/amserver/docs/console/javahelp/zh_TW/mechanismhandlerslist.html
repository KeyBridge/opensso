機制處理程式清單

[機制處理程式清單] 屬性會儲存 認證 Web 服務 所支援之 SASL 機制的相關資訊。它會顯示包含以管道分隔的鍵/值對的項目，如下所示：key=PLAIN|class=com.sun.identity.liberty.ws.authnsvc.mechanism.PlainMechanismHandler。下表將說明各參數。

參數

值

key

必需的 key 參數定義 認證 Web 服務 支援的 SASL 機制。

class

必需的 class 參數指定 SASL 機制的實作類別名稱。認證 Web 服務 層提供一個需要實作的處理程式介面，以便讓每一個 SASL 機制處理請求的訊息並傳回回應。

若要新增請求處理程式，請按一下 [新增]，並參閱配置新的機制處理程式。
