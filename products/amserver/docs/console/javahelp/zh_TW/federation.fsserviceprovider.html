新增服務提供者至提供者實體

將服務提供者新增至提供者實體之程序的第一步是定義要新增的服務提供者的類型。

  • 代管提供者代管在與 Access Manager 相同的伺服器。

  • 遠端提供者不代管在與 Access Manager 相同的伺服器。

下列程序說明此初始步驟。起點為聯合介面下的實體。

  1. 選取配置的提供者實體名稱以修改其設定檔。

    會顯示實體的設定檔頁面。

  2. 從 [檢視] 功能表選取 [服務提供者]。

  3. 選取您要新增的提供者類型：

    • 如果提供者代管在與 Access Manager 相同的伺服器上，請選擇 [新建代管供應程式]。

    • 如果提供者不代管在與 Access Manager 相同的伺服器上，請選擇 [新建遠端供應程式]。

    會顯示適當的 [服務供應程式] 設定檔頁面。

  4. 根據您在前一個步驟中的選擇，選擇下列其中一項：

    • 新增一個新的代管服務提供者至提供者實體

    • 新增一個新的遠端服務提供者至提供者實體
