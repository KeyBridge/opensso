SAML

安全指定標記語言 (SAML) 是一種開放式標準協定，它使用 XML 架構在授權以及信任夥伴網站之間交換安全資訊的。安全資訊涉及本身與主體的認證狀態、存取授權以及屬性資訊。(由電子郵件地址識別的個人，其主體可能會和印表機一樣。)SAML 授權機構 (亦稱為指定方) 是已與 SAML API 整合的平台或應用程式，因此它可以中繼安全資訊。例如，假設已認證主體可藉由傳遞必要屬性進入其系統。信任夥伴網站會接收安全資訊並信任其確實性。

可為您的實作配置以下的 SAML 屬性：

  • 目標指定符號

  • 網站識別碼

  • 可信任的合作夥伴

  • 目標 URL

  • 指定逾時

  • notBefore 時間的假設偏斜因素。

  • 工件逾時

  • SAML 工件名稱

  • 簽名 SAML 指定

  • 簽名 SAML 請求

  • 簽名 SAML 回應

以下的作業與配置 SAML 相關聯：

  • 配置網站識別碼

  • 可信任的合作夥伴：選取夥伴類型及設定檔

  • 可信任的合作夥伴：配置可信任的合作夥伴屬性

  • 配置目標 URL
