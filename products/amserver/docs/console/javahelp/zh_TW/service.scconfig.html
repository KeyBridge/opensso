配置屬性

[配置] 頁面包含所有用於配置 Access Manager 預設服務的屬性。構成 Access Manager 服務的屬性可歸類為以下類型之一 (某些服務可能具有多個類型)：

[全域] – 適用於整個 Access Manager 配置。(由於全域屬性的目標是自訂識別伺服器應用程式，因此它們無法套用至使用者、角色或範圍。)

[範圍] – 範圍屬性僅指定給範圍。沒有物件類別與範圍屬性相關聯。認證服務中列出的屬性會定義為範圍屬性，因為認證是在範圍層級執行，而不是在子樹或使用者層級執行。

[動態] – 指定給 Access Manager 配置的角色或範圍。當角色指定給使用者，或在範圍中建立使用者時，動態屬性隨後會變成使用者的特徵。

[使用者] – 直接指定給每一個使用者。它們不是繼承自角色或範圍，因此通常對於每一個使用者均不同。

您可以修改的配置特性如下：

  • 認證

  • 主控台特性

  • 全域特性

  • 系統特性
