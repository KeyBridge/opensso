認證

Access Manager 會隨著一組預設認證模組類型一起安裝。認證模組實例是一種收集使用者資訊 (例如使用者 ID 和密碼) 的外掛程式，它可以對照資料庫內的項目檢查資訊，並允許或拒絕使用者進行存取。可個別建立與配置相同類型的多重實例。

本節提供了配置預設認證模組類型的屬性說明。
