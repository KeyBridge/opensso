Web 服務提供者

Web 服務提供者代理程式設定檔說明用來保護 Web 服務提供者之 Web 服務請求的配置。Web 服務的名稱對於每個 Web 服務提供者來說必須是唯一的。

一般

下列 [一般] 屬性定義基本的 Web 服務提供者特性：

群組

群組機制可讓您定義一個相似類型的代理程式的集合。您必須先定義群組，才能將特定代理程式加入集合中。

密碼

定義 Web 服務提供者屬性的密碼。

密碼確認

確認密碼。

狀態

定義 Web 服務提供者代理程式是否將參與保護離埠服務請求。依預設，代理程式會設定為 [使用中]，表示將會參與。

通用識別碼

列出基本 LDAP 特性，對 Web 服務提供者代理程式進行唯一定義。

安全性

下列屬性定義 Web 服務提供者安全屬性：

安全機制

定義安全憑證的類型，用來保護 Web 服務提供者請求。您可以選擇下列安全類型：

  • 匿名 — 匿名安全機制不包含安全憑證。

  • LibertyBearerToken

  • LibertySAMLToken

  • LibertyX509Token — 使用 Liberty 定義的 X509 憑證保護 Web 提供者請求。

  • SAML-HolderOfKey — 使用 SAML 1.1 指定類型 Holder-Of-Key 來保護請求。

  • SAML-SenderVouches — 使用 SAML 1.1 指定類型 Sender Vouches 保護請求。

  • SAML2–HolderOfKey — 使用 SAML 2.0 指定記號類型 Holder-Of-Key 保護請求。

  • SAML2–SenderVouches — 使用 SAML 2.0 指定標記類型 Sender Vouches 保護請求。

  • UserNameToken — 使用使用者名稱記號保護 Web 提供者用戶端請求。

  • UserNameToken-Plain — 使用使用者名稱記號搭配明文密碼保護 Web 提供者請求。

  • X509Token — 使用 X509 憑證保護 Web 提供者請求。

認證鏈接

定義 –

保留訊息中的安全性標頭

啟用時，此屬性會定義對 Web 服務提供者保留 SOAP 安全性標頭以供進一步處理。

Liberty 服務類型 URN

URN (Universal Resource Name，通用資源名稱) 說明 Web 服務提供者用來進行服務查找的 Liberty 服務類型。

使用者記號的憑證

此屬性代表 Web 服務提供者用來產生「使用者名稱」安全記號的使用者名稱/密碼的共用密碼。

私有金鑰類型

定義 –

SAML 配置

下列屬性會配置 Web 服務提供者的安全宣示標記語言 (Security Assertion Markup Language, SAML)：

SAML 屬性對映

此配置代表需要在由 [安全性記號服務] 建立指定時產生的 SAML 屬性。Web 服務提供者會使用相同的對映將其填入 SSOToken。

SAML NameID 對映程式外掛程式

定義 NameID 對映程式外掛程式類別，用於 SAML 帳號對映。

SAML 屬性名稱空間

定義用來產生 SAML 屬性的名稱空間。

包含成員身份

如果啟用，此屬性會定義主體的成員身份必須當作 SAML 屬性包含。

簽署與加密

下列屬性會為 Web 提供者的安全定義簽署與加密配置：

回應是否已簽署

啟用時，Web 服務提供者會使用提供的記號類型簽署回應。

回應是否已加密

啟用時，Web 服務回應將會加密。

請求簽名是否已驗證

啟用時，會驗證 Web 服務請求簽名。

請求標頭是否已加密

啟用時，Web 服務用戶端安全性標頭將會加密。

請求是否已解密

啟用時，Web 服務提供者請求將會被解密。

金鑰庫

下列屬性配置用於憑證儲存和擷取的金鑰庫：

Web 服務用戶端的公開金鑰別名

此屬性定義用來加密 Web 服務回應的公開憑證金鑰別名。

私有金鑰別名

此屬性定義用來簽署/解密 Web 服務請求的私有憑證金鑰別名。

金鑰庫用法

此配置定義要使用預設的金鑰庫還是自訂金鑰庫。必須為自訂金鑰庫定義以下值：

  • 金鑰庫位置

  • 金鑰庫密碼

  • 金鑰密碼

端點

下列屬性可定義 Web 服務端點：

Web 服務端點

此屬性定義 Web 服務端點，Web 服務用戶端會對此端點發出請求。
