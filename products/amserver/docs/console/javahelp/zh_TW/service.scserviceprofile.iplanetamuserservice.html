使用者

預設使用者喜好設定是透過使用者服務來定義的。這些喜好設定包括時區、語言環境及 DN 啟動檢視。使用者服務屬性是動態屬性。

  • 使用者喜好的語言

  • 使用者喜好的時區

  • 繼承的語言環境

  • 管理員啟動檢視

  • 預設使用者狀態

使用者喜好的語言

此欄位指定於 Access Manager 主控台中顯示的文字語言之使用者選項。預設值為 en。此值會將本土化鍵集對映至使用者階段作業，從而螢幕文字會以適於使用者使用的語言顯示。

使用者喜好的時區

此欄位指定使用者存取 Access Manager 主控台所在的時區。此欄位沒有預設值。

繼承的語言環境

此欄位指定使用者的語言環境。預設值為 en_US。請參閱支援的語言環境，以取得語言環境的清單。

管理員啟動檢視

如果此使用者是 Access Manager 管理員，則此欄位指定該使用者登入時，做為 Access Manager 主控台中顯示之起點的節點。此欄位沒有預設值。可以使用該使用者至少具有讀取權限的有效 DN。

預設使用者狀態

此選項指示任何新建使用者的預設狀態。此狀態會由「使用者項目」狀態取代。只有使用中的使用者才可以透過 Access Manager 進行認證。預設值為使用中。可以從下拉式功能表中選取以下任一選項：

使用中

使用者可以透過 Access Manager 進行認證。

非使用中

使用者無法透過 Access Manager 進行認證，但使用者設定檔仍儲存在該目錄。

個別使用者狀態的設定方法如下：註冊使用者服務，選擇此值並將其套用於某種角色，然後將此角色加入到使用者設定檔。
