建立新的實例名稱

命名服務會在初始化時讀取此屬性。此清單包含單一 Access Manager 配置中的 Access Manager 階段作業伺服器。

  1. 在實例名稱清單中按一下 [新建]。

  2. 在伺服器欄位中輸入主機名稱及連接埠。

  3. 輸入網站名稱。

    此值唯一識別伺服器。參與負載平衡或容錯遷移的每個伺服器都需要具有唯一的識別碼。這也會用於藉由將伺服器 URL 對映至伺服器 ID，來縮短 Cookie 長度。語法為：

    intance_ID(|site_ID)

  4. 按一下 [確定]。

    若要編輯實例名稱，請按一下實例名稱清單中的項目，然後變更相應的值。

  5. 在 [平台服務] 主頁面中按一下 [儲存]。
