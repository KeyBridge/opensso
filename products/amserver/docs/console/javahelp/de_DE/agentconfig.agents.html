Agenten

Die zentralisierte Agentenkonfiguration ermöglicht einem Agentenadministrator die Verwaltung mehrerer Agentenkonfigurationen von einem zentralen Ort aus. Die Agentenkonfigurationen sind im Daten-Repository des Federated Access Manager gespeichert und werden von einem Administrator über die Federated Access Manager-Konsole verwaltet. Alle Änderungen an der Agentenkonfiguration werden an die betroffenen Agenten übermittelt, und die Agenten reagieren entsprechend in Abhängigkeit von der Art der aktualisierten Eigenschaften. Sie können folgende spezifische Agentenprofile erstellen:

  • Web-Richtlinien-Agent

  • J2EE-Richtlinien-Agent

  • Webdienstanbieter

  • Webdienstclient-Attribute

  • Sicherheitstoken-Dienst

  • Discovery Agent

  • 2.2 Policy Agent
