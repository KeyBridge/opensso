ResourceID-Mapper

Der Wert dieses Attributs gibt die Implementierung von com.sun.identity.liberty.ws.interfaces an.ResourceIDMapper . Es kann zwar eine neue Implementierung entwickelt werden, Access Manager stellt jedoch das standardmäßige com.sun.identity.liberty.ws.idpp.plugin bereit.IDPPResourceIDMapper ordnet jeder Benutzer-ID einen Discovery Resource Identifier zu.
