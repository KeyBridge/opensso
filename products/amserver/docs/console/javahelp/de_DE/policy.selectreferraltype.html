So fügen Sie einer Richtlinie Bezüge hinzu

  1. Wenn die Richtlinie, der Sie einen Response-Anbieter hinzufügen möchten, bereits erstellt ist, klicken Sie auf den Namen der Richtlinie. Wenn die Richtlinie noch nicht erstellt wurde, ziehen Sie So erstellen Sie eine neue Bezugsrichtlinie zurate.

  2. Klicken Sie unterhalb der Liste mit den Bezügen auf “Neu".

  3. Definieren Sie die Ressource in den Feldern unter “Regeln". Die Felder sind:

    Bezug: Zeigt den aktuellen Bezugstyp an.

    Name: Geben Sie den Namen des Bezugs ein.

    Inhalt: Legt einen Filter für die Bereichsnamen fest, die im Feld “Wert" angezeigt werden. Standardmäßig werden alle Bereichsnamen angezeigt.

    Wert: Wählen Sie den Bereichsnamen des Bezugs.

  4. Klicken Sie auf “Fertig stellen"

    Zum Entfernen eines Bezugs aus einer Richtlinie wählen Sie den Bezug und klicken Sie auf “Löschen".

    Sie können eine beliebige Bezugsdefinition bearbeiten, indem Sie auf den Link “Bearbeiten" neben dem Bezugsnamen klicken.
