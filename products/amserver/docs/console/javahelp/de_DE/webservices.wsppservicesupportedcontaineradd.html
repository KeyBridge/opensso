So konfigurieren Sie einen unterstützten Container

Ein Container ist ein Attribut, das den Eigentümer für bestimmte Identitätsdaten festlegt. Im Folgenden wird die Vorgehensweise zum Hinzufügen neuer Attribute in Persönlicher Liberty-Profildienst beschrieben. Der Persönlicher Liberty-Profildienst-Konfigurationsbildschirm im Modul “Webdienste" bildet den Ausgangspunkt.

  1. Klicken Sie auf “Neu", um die Attribute für den neuen unterstützten Container anzuzeigen, bzw. auf den Namen eines konfigurierten Containers, um ihn zu ändern.

  2. Legen Sie die Werte der Container-Attribute entsprechend der folgenden Informationen fest:

    Containername

    Geben Sie einen Namen für den Container ein, beispielsweise Kredikarte.

    Plugin

    Geben Sie den Namen der Klasse ein, mit der sämtliche Containervorgänge abgewickelt werden sollen. Dies könnte zur Außerkraftsetzung der Standardimplementierung com.sun.identity.liberty.ws.idpp.plugin.IDPPContainer verwendet werden.

  3. Klicken Sie auf “OK", um die Container-Konfiguration abzuschließen.

  4. Klicken Sie auf der Persönlicher Liberty-Profildienst-Seite auf “Speichern", um die Konfiguration des Dienstes abzuschließen.
