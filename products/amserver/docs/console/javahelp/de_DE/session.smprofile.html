Sitzungsfenster

Zeigt die Benutzer an, die derzeit bei Access Manager angemeldet sind, und die Sitzungszeit für jeden angemeldeten Benutzer. Folgende Felder werden angezeigt:

Benuzter-ID

Zeigt die Benutzer-ID des derzeit angemeldeten Benutzers an.

Restliche Zeit (Minuten)

Zeigt die Zeit (in Minuten) an, die dem Benutzer verbleibt, bevor er eine erneute Authentifizierung durchführen muss.

Maximale Sitzungsdauer (Minuten)

Zeigt die maximale Zeit (in Minuten) an, innerhalb der sich der Benutzer anmelden kann, bevor die Sitzung abläuft und eine erneute Authentifizierung durchgeführt werden muss, um erneuten Zugriff zu erhalten.

Leerlaufzeit (Minuten)

Zeigt die Zeit (in Minuten) an, in der der Benutzer inaktiv war.

Maximale Leerlaufzeit (Minuten)

Zeigt die maximale Dauer (in Minuten) an, die der Benutzer inaktiv bleiben kann, ohne eine erneute Authentifizierung durchführen zu müssen. Das Zeitlimit wird vom Administrator im Sitzungsdienst festgelegt. Weitere Informationen zu Sitzungsattributdefinitionen finden Sie unter Sitzung.
