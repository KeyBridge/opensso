Bereiche

Bei einem Zugriffssteuerungsbereich handelt es sich um eine Gruppe von Authentifizierungseigenschaften und Autorisierungsrichtlinien, die mit einem Benutzer oder einer Gruppe von Benutzern verknüpft werden können. Bereichsdaten werden in einem proprietären Informationsbaum gespeichert, der von Access Manager in dem von Ihnen angegebenen Data Store erstellt wird. Das Access Manager-Framework fasst die Richtlinien und Eigenschaften der einzelnen Bereiche des Access Manager-Informationsbaums zusammen. Standardmäßig fügt Access Manager 7.1 den Access Manager-Informationsbaum zusätzlich zu den Benutzerdaten automatisch als speziellen Zweig in Sun Java Enterprise System Directory Server ein. Zugriffssteuerungsbereiche können in Verbindung mit jeder beliebigen LDAPv3-Datenbank verwendet werden.

Auf der Registerkarte “Bereiche" können Sie die folgenden Zugriffssteuerungseigenschaften festlegen:

  • Allgemeine Eigenschaften

  • Authentifizierung

  • Dienste
