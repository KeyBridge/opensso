Klassen für ResourceID-Mapper-Plugin

Der Wert dieses Attributs ist eine Liste von Klassen, mit denen IDs für ein für die Organisation oder Rolle konfiguriertes Ressourcenangebot erstellt werden. com.sun.identity.liberty.ws.interfaces.ResourceIDMapper ist eine Schnittstelle, mit der Benutzer-IDs den entsprechenden Ressourcen-IDs zugeordnet werden. Der Discovery-Dienst stellt für diese Schnittstelle zwei Implementierungen bereit:

  • com.sun.identity.liberty.ws.disco.plugins.Default64ResourceIDMapper (Format: providerID + "/" + the Base64 encoded userIDs)

  • com.sun.identity.liberty.ws.disco.plugins.DefaultHexResourceIDMapper (Format: providerID + "/" + the hex string of userID)

Mithilfe der Implementierungsklasse können Sie weitere Implementierungen entwickeln und als Wert dieses Attributs hinzufügen. Klicken Sie dazu auf "Neu" und verwenden Sie das folgende Format:

providerid=providerID| class_name_and_path
