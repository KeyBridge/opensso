Gefilterte Rolle

Eine gefilterte Rolle ist eine dynamische Rolle, die mithilfe eines LDAP-Filters erstellt wurde. Alle Benutzer werden durch den Filter geleitet und zum Zeitpunkt der Rollenerstellung der Rolle zugewiesen. Der Filter sucht nach jedem Attributwertepaar (z. B. ca=user* ) in einem Eintrag und weist automatisch die Benutzer zu, die das Attribut für die Rolle enthalten.

Weitere Informationen finden Sie unter So erstellen Sie eine gefilterte Rolle
