So legen Sie fest, dass ein Agent Eigenschaften von einer Gruppe übernehmen soll
Bevor Sie beginnen

Die Gruppe, aus der der Agent Attribute übernehmen soll, muss zunächst erstellt worden sein.

  1. Klicken Sie auf der Registerkarte für die Konfiguration auf "Agenten".
  2. Wählen Sie gegebenenfalls die Registerkarte für den entsprechenden Agententyp aus.
    • Webagenten:

      Die Registerkarte "Web" bezieht sich auf Webagenten und ist standardmäßig ausgewählt. Es ist keine weitere Auswahl erforderlich.

    • J2EE-Agenten:

      Die Registerkarte "J2EE" bezieht sich auf J2EE-Agenten. Klicken Sie bei Bedarf auf diese Registerkarte.

    • 2.2-Agenten

      Die Registerkarte "2.2 Agenten" bezieht sich auf Agenten im Policy Agent 2.2-Software-Set. Klicken Sie bei Bedarf auf diese Registerkarte.

  3. Klicken Sie im Abschnitt "Agent" auf den Namen des zu konfigurierenden Agenten.
  4. Wählen Sie bei ausgewählter Registerkarte "Global" für das Attribut "Gruppe" den Namen der Gruppe aus, aus der der Agent Attribute übernehmen soll.

    Oben auf der Seite wird die Schaltfläche "Vererbungseinstellungen" aktiv.

  5. Klicken Sie auf "Vererbungseinstellungen".
  6. Wählen Sie die Attribute (Eigenschaften) aus, die der Agent aus der Gruppe übernehmen soll.
  7. Klicken Sie auf Speichern.
