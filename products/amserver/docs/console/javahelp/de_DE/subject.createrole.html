So erstellen oder ändern Sie eine Rolle

  1. Klicken Sie auf die Registerkarte “Rolle".

  2. Klicken Sie in der Liste der Rollen auf “Neu".

  3. Geben Sie einen Namen für die Rolle ein.

  4. Klicken Sie auf “OK".

    Nachdem Sie die Rolle erstellt haben, klicken Sie auf deren Namen, um die in den nachfolgenden Themen beschriebenen Aufgaben auszuführen.

    • So fügen Sie einer Identität Dienste hinzu

    • So fügen Sie Benutzer einer Rolle oder Gruppe hinzu
