Verzeichnisverwaltung

Die Registerkarte "Verzeichnisverwaltung" wird nur angezeigt, wenn Federated Access Manager im Legacy-Modus installiert wird. Diese Verzeichnisverwaltungsfunktion bietet eine Identitätsverwaltungslösung für Sun Java System Directory Server-fähige Federated Access Manager-Implementierungen.

Weitere Informationen zu den Legacymodus-Installationsoptionen finden Sie im Sun Java Enterprise System Installation Guide for UNIX.
