So erstellen Sie eine Anbieter-Entity oder eine Partner-Entity

Die Einrichtung von Entities besteht aus zwei Schritten. Zunächst erstellen Sie die Entity. (In diesem Schritt legen Sie auch fest, ob es sich um eine Anbieter-Entity oder eine Partner-Entity handelt.) Nach der Erstellung der Entity füllen Sie sie mit Anbietern. Mit dem folgenden Verfahren wird eine neue Entity erstellt. Der Ausgangspunkt ist “Entities" in der Verbindungsschnittstelle.

Hinweis – Eine Entity kann erst dann einer Authentifizierungsdomäne zugewiesen werden, wenn sie Anbieter enthält.

  1. Klicken Sie auf “Neu", um die Attribute für Entities anzuzeigen.

    Die Seite “Neue Entity" wird angezeigt.

  2. Geben Sie einen Namen für die Entity ein.

    Das Attribut “Entity-Name" definiert einen Uniform Resource Identifier (URI) für die Entity. Dieser Wert muss unter den Entity-Namen eindeutig sein.

  3. Geben Sie eine Beschreibung der Entity ein.

  4. Wählen Sie den Entity-Typ.

    • Wenn Sie eine Anbieter-Entity erstellen möchten, wählen Sie Anbieter aus.

    • Wenn Sie eine Partner-Entity erstellen möchten, wählen Sie Partner aus und definieren Sie die Werte für die erforderlichen Attribute “Partner-Name" und “Partner-Eigentümer".

      • “Partner-Name" definiert einen URI für die Partner-Entity. Dieser Wert muss unter den Partner-Namen eindeutig sein.

      • “Partner-Eigentümer" gibt einen Anbieter als übergeordneten Administrator der Partnerschaft an. Der Wert dieses Attributs ist ein URI, der mit dem Wert übereinstimmt, der im Attribut “Entity-Name" der Entity definiert wurde, zu der der Partner-Eigentümer/Anbieter gehört.

      Hinweis – Wenn ein Anbieter als “Partner-Eigentümer" definiert wird, wird er damit nicht automatisch als Mitglied der Partnerschaft aufgenommen. Wenn ein Eigentümer gleichzeitig Mitglied ist, muss der URI hier als Eigentümer und innerhalb der Partner-Entity selbst als Mitglied definiert werden. Informationen zum Hinzufügen von Mitgliedern zu einer Partnerschaft finden Sie unter So fügen Sie Mitglieder zu einer Partner-Entity hinzu.

  5. Klicken Sie auf “OK".

    Die neue Entity wird im Bildschirm “Entities" angezeigt.
