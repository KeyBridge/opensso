Authentifizierung

Access Manager wird mit einer Reihe von Modultypen zur Standardauthentifzierung installiert. Eine Authentifizierungsmodulinstanz ist ein Plugin, das Benutzerinformationen wie die Benutzer-ID und das Passwort abfragt, die Informationen mit den Einträgen in einer Datenbank vergleicht und dem Benutzer je nach Überprüfungsergebnis Zugriff gewährt oder verweigert. Es können mehrere Instanzen des gleichen Typs separat erstellt und konfiguriert werden.

In diesem Abschnitt werden die Attribute für die Konfiguration der Standard-Authentifizierungsmodultypen beschrieben.
