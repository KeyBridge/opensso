Ressourcenangebote für Bootstrapping

Dieses Attribut legt ein Ressourcenangebot für das Bootstrapping eines Dienstes fest. Nach einem Single Sign-On (SSO) werden dieses Ressourcenangebot und die zugehörigen Anmeldeinformationen an den Client der SSO-Assertion gesendet. Für Bootstrapping ist nur ein Ressourcenangebot zulässig. Standardmäßig enthält dieses Angebot Informationen über den Discovery-Dienst. Mit diesem Attribut sind folgende Aufgaben verbunden:

  • So konfigurieren Sie ein Ressourcenangebot für Bootstrapping

  • So konfigurieren Sie eine Dienstbeschreibung
