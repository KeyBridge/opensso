So löschen Sie das Profil einer Authentifizierungsdomäne

Führen Sie die folgenden Schritte aus, um eine vorhandene Authentifizierungsdomäne zu löschen. Der Ausgangspunkt ist der Eintrag für Authentifizierungsdomänen der Verbindungsschnittstelle.

  1. Aktivieren Sie das Kontrollkästchen neben dem Namen der Authentifizierungsdomäne, die Sie löschen möchten.

  2. Klicken Sie auf “Löschen".

    Die zugehörigen Anbieter werden durch das Löschen einer Authentifizierungsdomäne nicht gelöscht.
