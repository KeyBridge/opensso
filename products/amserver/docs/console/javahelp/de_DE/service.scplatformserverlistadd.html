So erstellen Sie einen neuen Objektnamen

Der Namensdienst liest dieses Attribut zum Zeitpunkt der Initialisierung. Diese Liste enthält die innerhalb einer Access Manager-Konfiguration verfügbaren Access Manager-Sitzungs-Server.

  1. Klicken Sie in der Liste “Objektname" auf “Neu".

  2. Geben Sie den Hostnamen und den Port im Feld “Server" ein.

  3. Geben Sie den Site-Namen ein.

    Dieser Wert kennzeichnet den Server eindeutig. Jeder Server, der am Lastenausgleich oder am Failover teilnimmt, muss einen eindeutigen Bezeichner haben. Dieser wird auch zur Kürzung der Cookie-Länge verwendet, indem der Server-URL der Server-ID zugeordnet wird. Die Syntax lautet:

    intance_ID(|site_ID)

  4. Klicken Sie auf “OK".

    Zur Bearbeitung eines Objektnamens klicken Sie auf einen Eintrag in der Objektnamen-Liste und ändern Sie die Werte wie gewünscht.

  5. Klicken Sie auf der Hauptseite des Plattformdiensts auf “Speichern".
