Benutzer

Ein Benutzer steht für die Identität einer Einzelperson. Benutzer können in Gruppen erstellt und gelöscht werden und in Rollen oder Gruppen hinzugefügt oder entfernt werden. Außerdem können Sie einem Benutzer Dienste zuweisen.

Weitere Informationen finden Sie unter So erstellen oder ändern Sie einen Benutzer
