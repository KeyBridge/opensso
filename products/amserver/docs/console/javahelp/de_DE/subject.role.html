Rolle

Bei den Mitgliedern einer Rolle handelt es sich um LDAP-Einträge, die die Rolle besitzen. Die Kriterien der Rolle sind in einem LDAP-Eintrag festgelegt, dessen Attribute durch das DN-Attribut des Eintrags gekennzeichnet sind. Nachdem eine Rolle erstellt ist, können Sie ihr manuell Dienste und Benutzer hinzufügen.

Weitere Informationen finden Sie unter So erstellen oder ändern Sie eine Rolle
