Verzeichniskonfiguration

Die Attribute für die Verzeichniskonfiguration definieren grundlegende Konfigurationsinformationen für den eingebetteten Verzeichnisspeicher:

Verzeichniskonfiguration

Es gibt folgende Attribute für die Verzeichniskonfiguration:

Minimaler Verbindungspool

Gibt die minimale Größe der Verbindungspools an, die für die Verbindung zum Directory Server verwendet werden, der im LDAP Server-Attribut festgelegt ist. Der Standardwert ist 1.

Maximaler Verbindungspool

Gibt die maximale Größe der Verbindungspools an, die für die Verbindung zum Directory Server verwendet werden, der im LDAP Server-Attribut festgelegt ist. Der Standardwert ist 10.

Verbindungs-DN

Dieses Attribut gibt den BIND-DN des LDAP-Servers an.

Verbindungs-Passwort

Dieses Attribut legt das Passwort fest, dass für die Verbindung zum LDAP-Server verwendet wird. Standardmäßig wird das Passwort amldapuser, das während der Installation eingegeben wurde, als BIND-Passwort verwendet.

Legacy-Konfiguration

Die folgenden Attribute definieren grundlegende Verzeichnisserver-Konfigurationen für Instanzen von Federated Access Manager im Legacy-Mode.

Minimaler Verbindungspool

Gibt die minimale Größe der Verbindungspools an, die für die Verbindung zum Directory Server verwendet werden, der im LDAP Server-Attribut festgelegt ist. Der Standardwert ist 1.

Maximaler Verbindungspool

Gibt die maximale Größe der Verbindungspools an, die für die Verbindung zum Directory Server verwendet werden, der im LDAP Server-Attribut festgelegt ist. Der Standardwert ist 10.

Server

Dieses Attribut führt Protokoll, Hostname und Anschluss für das Lastenausgleichsprotokoll auf. Beispiel: http://lb.example.com:80.
