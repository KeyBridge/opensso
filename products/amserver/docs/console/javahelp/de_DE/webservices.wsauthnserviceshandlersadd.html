So konfigurieren Sie einen neuen Mechanismus-Handler

  1. Klicken Sie auf “Hinzufügen", um die neuen Mechanismus-Handler-Attribute anzuzeigen, oder klicken Sie auf einen konfigurierten Schlüsselwert, um vorhandene Attribute zu bearbeiten.

  2. Legen Sie die Attributwerte entsprechend der folgenden Informationen fest:

    key

    Mit diesem erforderlichen Parameter wird der vom Authentifizierungs-Webdienst unterstützte SASL-Mechanismus definiert.

    Klasse

    Dieser erforderliche Parameter gibt den Namen der Implementierungsklasse für den SASL-Mechanismus an. Die Authentifizierungs-Webdienst-Schicht bietet eine Handler-Schnittstelle, die implementiert werden muss, damit jeder SASL-Mechanismus die angeforderte Meldung verarbeiten und eine Antwort zurückgeben kann.

  3. Klicken Sie auf “OK" oder “Speichern", um die Konfiguration des Anforderungs-Handlers abzuschließen.

  4. Klicken Sie auf der Authentifizierungs-Webdienst-Seite auf “Speichern", um die Konfiguration des Dienstes abzuschließen.
