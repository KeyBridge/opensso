So fügen Sie einer Richtlinie einen Response-Anbieter hinzu.

  1. Wenn Sie die Richtlinie bereits erstellt haben, klicken Sie auf den Namen der Richtlinie, für die Sie den Response-Anbieter hinzufügen möchten. Wenn die Richtlinie noch nicht erstellt wurde, ziehen Sie So erstellen Sie eine neue normale Richtlinie zurate.

  2. Klicken Sie unter der Liste “Response-Anbieter" auf “Neu".

  3. Geben Sie einen Namen für den Response-Anbieter ein.

  4. Legen Sie die folgenden Werte fest:

    StaticAttribute

    Es handelt sich hierbei um statische Attribute im Attributwertformat; die Definition erfolgt in einer Instanz von IDResponseProvider (Speicherung in der Richtlinie).

    DynamicAttribute

    Die hier ausgewählten Response-Attribute müssen zunächst im Richtlinienkonfigurationsdienst des entsprechenden Bereichs definiert werden. Bei den definierten Attributnamen sollte es es sich um eine Untermenge der im konfigurierten Data Store (IDRepository) enthaltenen Namen handeln. Details zur Definition der Attribute finden Sie in den Attributdefinitionen zur Richtlininenkonfiguration. Wenn Sie bestimmte bzw. mehrere Attribute auswählen möchten, halten Sie die Strg-Taste gedrückt und klicken Sie mit der linken Maustaste.

  5. Klicken Sie auf “Fertig stellen"

  6. Zum Entfernen eines Response-Anbieters aus einer Richtlinie wählen Sie den Betreff aus und klicken Sie auf “Löschen". Zum Bearbeiten einer Response-Anbieterdefinition klicken Sie auf den Namen des Response-Anbieters.
