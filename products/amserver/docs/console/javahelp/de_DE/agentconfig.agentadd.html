So erstellen Sie einen neuen Agenten

Das Erstellen eines neuen Agenten wird auch als Erstellen eines neuen Agentenprofils bezeichnet. Sie können diese Aufgabe an der Federated Access Manager -Administrationskonsole ausführen. Diese Aufgabe bezieht sich auf Webagenten, J2EE-Agenten und 2.2-Agenten. Allerdings gelten einige der Einzelschritte nicht für die 2.2-Agenten.

  1. Klicken Sie auf der Registerkarte für die Konfiguration auf "Agenten".
  2. Wählen Sie gegebenenfalls die Registerkarte für den entsprechenden Agententyp aus.
    • Webagenten:

      Die Registerkarte "Web" bezieht sich auf Webagenten und ist standardmäßig ausgewählt. Es ist keine weitere Auswahl erforderlich.

    • J2EE-Agenten:

      Die Registerkarte "J2EE" bezieht sich auf J2EE-Agenten. Klicken Sie bei Bedarf auf diese Registerkarte.

    • 2.2-Agenten

      Die Registerkarte "2.2-Agenten" bezieht sich auf Agenten im Policy Agent 2.2-Software-Set. Klicken Sie bei Bedarf auf diese Registerkarte.

  3. Klicken Sie im Abschnitt "Agent" auf "Neu".
  4. Geben Sie im Feld "Name" den Namen des neuen Agentenprofils ein.
  5. Geben Sie das Passwort ein, und bestätigen Sie es.

    Achtung – Dieses Passwort muss mit dem Passwort in der Datei für das Agentenprofilpasswort übereinstimmen, die Sie angeben, wenn Sie den Agenten durch Ausführung des Programms agentadmin installieren.

  6. Wählen Sie die entsprechende Konfiguration aus: "Lokal" oder "Zentralisiert" (Gilt nicht für 2.2-Agenten).

    Außerdem können bei Auswahl von "Lokal" die zu diesem Agenten gehörenden Attribute nur über eine Datei im Agenten bearbeitet werden. Eine Bearbeitung über die Konsole ist nicht möglich.

  7. Geben Sie im Feld für die Server-URL die Server-URL des Federated Access Manager ein. (Gilt nicht für 2.2-Agenten.)

    Beispiel:

    http://fam.example.com:8080/fam
  8. Geben Sie im Feld für die Agenten-URL die URL für die Agentenanwendung, agentapp ein. (Gilt nicht für 2.2-Agenten.)

    Beispiel:

    http://fam.example.com:8090/agentapp
  9. Klicken Sie auf "Erstellen".

    Die Konsole erstellt das Agentenprofil und zeigt die Agentenseite (WebAgent, J2EEAgent bzw. 2.2_Agent) mit einer Verknüpfung zum neuen Agentenprofil an.

    Wenn Sie weitere Konfigurationen am Agentenprofil vornehmen möchten, klicken Sie auf diese Verknüpfung, um die Seite zur Bearbeitung des Agenten anzuzeigen.
