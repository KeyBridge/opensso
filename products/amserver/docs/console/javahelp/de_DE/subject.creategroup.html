So erstellen oder ändern Sie eine Gruppe

  1. Klicken Sie auf die Registerkarte “Gruppe".

  2. Klicken Sie in der Liste der Gruppen auf “Neu".

  3. Geben Sie einen Namen für die Gruppe ein.

  4. Klicken Sie auf “OK".

    Nachdem Sie die Gruppe erstellt haben, können Sie ihre Benutzer hinzufügen. Klicken Sie dazu auf den Namen der Gruppe und öffnen Sie die Registerkarte “Benutzer". Weitere Informationen finden Sie unter So fügen Sie Benutzer einer Rolle oder Gruppe hinzu.
