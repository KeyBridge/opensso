Data Stores

Als Data Store wird eine Datenbank bezeichnet, in der Benutzerattribute und Benutzerkonfigurationsdaten gespeichert werden können.

Access Manager bietet ein Identitäts-Repository-Plugin, das eine Verbindung zu einem LDAPv3-Identitäts-Repository-Framework herstellt. Auf diese Weise können Sie Access Manager-Benutzerinformationen anzeigen und abrufen, ohne Änderungen an Ihrer vorhandenen Benutzerdatenbank vornehmen zu müssen. Das Access Manager-Framework fasst die Daten des Identitäts-Repository-Plugins mit den Daten anderer Access Manager-Plugins zusammen und bildet so eine virtuelle Identität jedes einzelnen Benutzers. Diese universelle Identität kann Access Manager bei der Authentifizierung und Autorisierung für mehrere Identitäts-Repositories gemeinsam verwenden. Die virtuelle Benutzeridentität wird vernichtet, sobald die Benutzersitzung endet.

Access Manager unterstützt folgende Data Stores:

  • Access Manager-Repository-Plugin

  • Active Directory

  • Repository für Flat Files

  • Generisches LDAPv3

  • Sun Directory Server mit Access Manager-Schema
