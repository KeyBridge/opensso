So erstellen Sie eine neue Authentifizierungsmodulinstanz

  1. Klicken Sie auf den Namen des Bereichs, für den Sie eine neue Authentifizierungsmodulinstanz hinzufügen möchten.

  2. Klicken Sie auf die Registerkarte “Authentifizierung".

  3. Klicken Sie in der Liste der Modulinstanzen auf “Neu".

  4. Geben Sie einen Namen für die Authentifizierungsmodulinstanz ein. Der Name muss eindeutig sein.

  5. Wählen Sie den Authentifizierungsmodultyp für den Bereich aus.

  6. Klicken Sie auf “OK".

  7. Klicken Sie auf den Namen der neu erstellten Modulinstanz und bearbeiten Sie die Eigenschaften dieses Moduls. Eigenschaftsdefinitionen sämtlicher Modultypen finden Sie unter Konfigurationsattribute des Authentifizierungsmoduls.

  8. Wiederholen Sie diese Schritte, wenn Sie weitere Modulinstanzen hinzufügen möchten.
