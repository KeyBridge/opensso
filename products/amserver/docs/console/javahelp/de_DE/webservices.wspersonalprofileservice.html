Persönlicher Liberty-Profildienst

Die Liberty ID-SIS Personal Profile-Dienstspezifikation beschreibt einen Datendienst, der allgemeine Profilinformationen zu einer Identität bereitsstellt. Der Dienst dient als letzter gemeinsamer Nenner für das Speichern von benutzerbasierten Informationen zu einem Principal. Access Manager hat diese Spezifikation implementiert und die Persönlicher Liberty-Profildienst ausgearbeitet. Die Persönlicher Liberty-Profildienst kann hinsichtlich Identitätsdaten abgefragt werden bzw. diese Daten können aktualisiert werden.

Hinweis – Um auf diesen Dienst zuzugreifen, muss der Anbieter der Persönlicher Liberty-Profildienst, der als Host agiert, im Namen der einzelnen Identitäts-Principals bei Discovery-Dienst registriert sein. Weitere Informationen finden Sie unter Discovery-Dienst.
