Plattform

Mithilfe des Plattformdiensts können zusätzliche Server zur Access Manager-Konfiguration hinzugefügt und andere Optionen auf der obersten Ebene der Access Manager-Anwendung übernommen werden. Bei den Plattformdienstattributen handelt es sich um globale Attribute. Es existieren folgende Attribute:

  • Site-Name

  • Objektname

  • Plattform-Ländereinstellung

  • Cookie-Domänen

  • URL des Anmeldedienstes

  • URL des Abmeldedienstes

  • Verfügbare Ländereinstellungen

  • Client-Zeichensätze

Site-Name

Der Namensdienst liest diese Attribute während der Initialisierung ein. Diese Liste identifiziert den FQDN mit der Anschlussnummer des Lastenausgleichs oder dem SRA für den Lastenausgleich auf den Access Manager-Back-end-Servern eindeutig. Wenn sich der in einer Dienst-URL-Anforderung angegebene Host nicht in dieser Liste befindet, lehnt der Namensdienst diese Anforderung ab. Nur das Namensdienstprotokoll sollte für dieses Attribut angegeben sein. Weitere Informationen finden Sie unter So erstellen Sie einen neuen Site-Namen.

Objektname

Der Namensdienst liest diese Attribute während der Initialisierung ein. Diese Liste enthält die innerhalb einer Access Manager-Konfiguration verfügbaren Access Manager-Sitzungs-Server. Sind beispielsweise zwei Access Manager installiert und sollen diese als eine Instanz zusammenarbeiten, müssen beide in dieser Liste eingetragen sein. Wenn sich der in einer Dienst-URL-Anforderung angegebene Host nicht in dieser Liste befindet, lehnt der Namensdienst diese Anforderung ab. Nur das Namensdienstprotokoll sollte für dieses Attribut angegeben sein. Weitere Informationen finden Sie unter So erstellen Sie einen neuen Objektnamen.

Plattform-Ländereinstellung

Der Wert dieses Attributs ist der Standard-Sprachenuntertyp, mit dem Access Manager installiert wurde. Die Authentifizierungs-, Protokoll- und Verwaltungsdienste werden in der mit diesem Attribut eingestellten Sprache verwaltet. Der Standardwert ist en_US. Eine Liste der unterstützten Sprachuntertypen finden Sie unter Unterstützte Sprach- bzw. Ländereinstellungen.

Cookie-Domänen

Dieses Attribut gibt die Liste der Domänen an, die im Cookie-Header zurückgegeben werden, wenn während der Authentifizierung ein Cookie für den Browser des Benutzers eingerichtet wird. Falls dieses Attribut leer ist, wird keine Cookie-Domäne eingerichtet. Mit anderen Worten: Das Access Manager-Sitzungscookie wird lediglich an Access Manager weitergeleitet und an keinen anderen Server der Domäne.

Wenn für andere Server in der Domäne SSO erforderlich ist, muss in diesem Attribut die Cookie-Domäne angegeben werden. Wenn auf einem Access Manager zwei Schnittstellen in unterschiedlichen Domänen vorhanden sind, müssen in diesem Attribut beide Cookie-Domänen angegeben werden. Wird ein Lastenausgleichssystem eingesetzt, muss die Domäne dieses Systems als Cookie-Domäne angegeben werden und nicht die Domäne der Server hinter diesem System. Der Standardwert für dieses Feld ist die Domäne des installierten Access Manager.

URL des Anmeldedienstes

In diesem Feld wird der URL der Anmeldeseite angegeben. Der Standardwert für dieses Attribut lautet /Service_DEPLOY_URI/UI/Login.

URL des Abmeldedienstes

In diesem Feld wird der URL der Abmeldeseite angegeben. Der Standardwert für dieses Attribut lautet /Service_DEPLOY_URI/UI/Logout.

Verfügbare Ländereinstellungen

Dieses Attribut speichert alle für die Plattform verfügbaren Ländereinstellungen. Verwenden Sie nach Möglichkeit eine Anwendung, die dem Benutzer die Auswahl der Ländereinstellungen ermöglicht. Eine solche Anwendung ruft dieses Attribut aus dem Plattformprofil ab und zeigt dem Benutzer die Liste der verfügbaren Ländereinstellungen an. Die vom Benutzer ausgewählte Ländereinstellung wird von der Anwendung im Benutzereintrag preferredLocale gespeichert.

Client-Zeichensätze

Dieses Attribut gibt die Zeichensätze für unterschiedliche Clients auf Plattformebene an. Es enthält eine Liste der Client-Typen mit den zugehörigen Zeichensätzen. Weitere Informationen finden Sie unter So erstellen Sie einen neuen Zeichensatz.
