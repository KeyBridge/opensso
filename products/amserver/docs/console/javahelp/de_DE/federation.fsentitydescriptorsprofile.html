So ändern Sie die allgemeinen Attribute einer Anbieter-Entity oder einer Partner-Entity

Nachdem Sie eine Entity erstellt haben, können Sie deren Profil bearbeiten. Dazu kann die Änderung von Attributen, das Hinzufügen von Identity-Anbietern und Dienstanbietern bzw. das Konfigurieren von Partnerschaften gehören. Das Bearbeiten der allgemeinen Attribute eines Entity-Profils kann eine Änderung der bereits festgelegten allgemeinen Attribute, das Hinzufügen einer Kontaktperson oder die Eingabe optionaler Organisationsinformationen umfassen. Folgendes Verfahren dient zur Bearbeitung der allgemeinen Attribute. Der Ausgangspunkt ist der Bildschirm “Entities" unter “Verbindung".

  1. Wählen Sie den Namen einer konfigurierten Entity, um deren Profil zu ändern.

    Standardmäßig wird die Seite für die allgemeinen Attribute des Profils der Entity angezeigt. Wenn Sie sich nicht auf der Seite für die allgemeinen Attribute befinden, können Sie im Menü “Ansicht" die Option “Allgemein" auswählen.

  2. Bearbeiten Sie die Werte einer der allgemeinen Entity-Attribute.

    Entity-Name

    Der statische Wert dieses Attributs ist der Name, den Sie bei der Erstellung der Entity eingegeben haben.

    Typ

    Der statische Wert dieses Attributs ist der Entity-Typ, “Anbieter" bzw. “Partner".

    Beschreibung

    Der Wert dieses Attributs ist die Beschreibung, die Sie bei der Erstellung der Entity eingegeben haben. Sie können die ursprünglich eingegebene Beschreibung ändern bzw. eine hinzufügen, wenn noch keine vorhanden ist.

  3. Geben Sie die Werte für die Attribute von “Profil der Entity-Kontaktperson" an.

    Die Attribute von “Profil der Entity-Kontaktperson" bieten Informationen dazu, an wen man sich in Bezug auf die Entity wenden kann. Diese Attribute sind optional.

    Vorname

    Geben Sie den Vornamen der Entity-Kontaktperson ein.

    Nachname

    Geben Sie den Nachnamen der Entity-Kontaktperson ein.

    Typ

    Wählen Sie den Kontakttyp im Dropdown-Menü aus:

    • Administrativ

    • Rechnungswesen

    • Technisch

    • Weitere

    Firma

    Geben Sie den Namen der Firma ein, bei der diese Person beschäftigt ist.

    Liberty-Haupt-ID

    Geben Sie einen URI (Uniform Resource Identifier) ein, der auf eine Instanz des (URI) Profils mit den personenbezogenen Informationen zu der Kontaktperson verweist.

    E-Mails

    Geben Sie mindestens eine E-Mail-Adresse für die Kontaktperson ein und klicken Sie dann auf “Hinzufügen".

    Telefonnummern

    Geben Sie mindestens eine Telefonnumer für die Kontaktperson ein und klicken Sie dann auf “Hinzufügen".

  4. Geben Sie die Werte für die Organisationsprofilattribute an.

    Die Attribute des Organisationsprofils beinhalten grundlegende Informationen, die bei der Interaktion mit einem Principal verlangt werden können. Diese Attribute sind optional.

    Hinweis – Wenn das Namensattribut einen Wert enthält, müssen Werte zu den Anzeigenamen- und URL-Attributen hinzugefügt werden.

    Namen

    Geben Sie den zulässigen Namen für die Organisation der Entity ein und klicken Sie dann auf “Hinzufügen". Der Wert wird im Format locale| organization-name definiert. Beispiel: en| example.com.

    Anzeigename

    Geben Sie einen Namen für die Organisation der Entity ein, der sich für die Anzeige für einen Principal eignet, und klicken Sie dann auf “Hinzufügen". Der Wert wird im Format locale|organization-display-name definiert. Beispiel: en|example.com.

    URL

    Geben Sie den URL der Organisation ein und klicken Sie dann auf “Hinzufügen". Der Wert wird im Format locale|organization-URL definiert. Beispiel: en|http://www.example.com.

  5. Klicken Sie auf “Speichern", um die Konfiguration abzuschließen, oder legen Sie weitere Werte für Identity-Anbieter, Dienstanbieter oder Partner fest.

    Wenn Sie Ihre Konfiguration fortsetzen möchten, beachten Sie folgende Informationen:

    • So fügen Sie einer Anbieter-Entity einen Identity-Anbieter hinzu zur Definition von Werten für Identity-Anbieter-Attribute.

    • So fügen Sie einer Anbieter-Entity einen Dienstanbieter hinzu zur Definition von Werten für Dienstanbieter-Attribute.

    • So fügen Sie Mitglieder zu einer Partner-Entity hinzu zum Füllen einer Partner-Entity mit Mitgliedern.
