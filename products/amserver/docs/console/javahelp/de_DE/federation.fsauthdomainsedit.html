So ändern Sie das Profil einer Authentifizierungsdomäne

Führen Sie die folgenden Schritte aus, um die allgemeinen Attribute einer vorhandenen Authentifizierungsdomäne zu ändern oder der Domäne Anbieter hinzuzufügen. Der Ausgangspunkt ist der Eintrag für Authentifizierungsdomänen der Verbindungsschnittstelle.

  1. Klicken Sie auf den Namen einer konfigurierten Authentifizierungsdomäne, um ihr Profil zu ändern oder um ihr Anbieter hinzuzufügen.

    Die Seite "Authentifizierungs-Domäne bearbeiten" wird angezeigt.

  2. Geben Sie neue Werte ein oder bearbeiten Sie bestehende Werte für die Attribute der Authentifizierungsdomäne unter "Allgemein":
    Name

    Der statische Wert dieses Attributs ist der Name, den Sie bei der Erstellung der Authentifizierungsdomäne eingegeben haben.

    Beschreibung

    Beim Wert dieses Attributs handelt es sich um eine Beschreibung der Authentifizierungsmethode. Die bereits eingegebene Beschreibung kann im Bedarfsfall geändert werden.

    URL des Schreibprogramm-Dienstes

    Dieses Attribut legt den Pfad des Dienstes fest, der das allgemeine Domänencookie erstellt. Der URL hat folgendes Format: http://common-domain-host:port/common/writer .

    URL des Leseprogramm-Dienstes

    Dieses Attribut legt den Pfad des Dienstes fest, der das allgemeine Domänencookie liest. Der URL hat folgendes Format: http://common-domain-host:port/common/transfer .

    Status

    Der Standardstatus ist "Aktiv". Durch Auswahl von "Inaktiv" deaktivieren Sie die Kommunikation mit der Authentifizierungsdomäne.

  3. Klicken Sie auf "Hinzufügen", um der Authentifizierungsdomäne Anbieter hinzuzufügen.

    Die Seite zur Auswahl vertrauenswürdiger Partner wird angezeigt.

  4. Wählen Sie einen oder mehrere Anbieter aus und klicken Sie auf den Pfeil zum Hinzufügen, um sie zu markieren.

    Die Liste enthält die Namen von Entities, die erstellt und mit Anbietern aufgefüllt wurden. Weitere Informationen finden Sie unter So fügen Sie einer Authentifizierungsdomäne Anbieter hinzu.

  5. Klicken Sie auf "OK", um die Anbieter in der Authentifizierungsdomäne zu speichern.

    Sie kehren zum vorherigen Profilfenster der Authentifizierungsdomäne zurück.

  6. Klicken Sie auf "Speichern", um den Vorgang abzuschließen.
