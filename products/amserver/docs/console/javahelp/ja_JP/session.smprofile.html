「セッション」ウィンドウ

現在 Access Manager にログイン中のユーザー、および各ログインのセッション時間が表示されます。表示フィールドは次のとおりです。

ユーザ ID

現在ログイン中のユーザー ID が表示されます。

残り時間 (分)

ユーザーの再認証までの、セッションの残り時間 (分単位) が表示されます。

最大セッション時間 (分)

セッションが期限切れになって再度認証を受ける必要がでてくるまでの、ユーザーがログインしていられる最大時間 (分単位) が表示されます。

アイドル時間 (分)

ユーザーがアイドル状態であった時間 (分単位) が表示されます。

最大アイドル時間 (分)

ユーザーの再認証までの、セッションの残りの最大アイドル時間 (分単位) が表示されます。時間の制限値は、管理者がセッションサービスに定義します。セッションの属性定義については、「セッション」を参照してください。
