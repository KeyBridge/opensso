レルム

アクセス制御レルムとは、特定の認証プロパティーと承認ポリシーの集合体のことです。アクセス制御レルムには、ユーザーやユーザーグループを関連付けることができます。レルムのデータは、ユーザーが指定したデータストア内に Access Manager が作成した専用の情報ツリー内に格納されます。Access Manager フレームワークは、各レルム内に格納されているポリシーとプロパティーを集約し、それらを Access Manager 情報ツリー内に格納します。Access Manager 7.1 はデフォルトで、Access Manager 情報ツリーを Sun Java Enterprise System Directory Server 内に自動的に挿入しますが、その際、ユーザーデータとは別の特殊なブランチとして挿入します。アクセス制御レルムは、任意の LDAPv3 データベースと併用できます。

「レルム」タブで設定可能なアクセス制御用プロパティーは、次のとおりです。

  • 「一般プロパティー」

  • 「認証」

  • 「サービス」
