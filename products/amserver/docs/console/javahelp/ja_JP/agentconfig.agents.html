エージェント

エージェント設定の集中管理は、複数のエージェント設定を 1 か所から管理する手段をエージェント管理者に提供します。エージェント設定は Federated Access Manager のデータリポジトリに格納され、管理者によって Federated Access Manager コンソールで管理されます。エージェント設定の変更はすべて、影響を受けるエージェントに伝達されます。エージェントは、更新されたプロパティーの性質に基づいて適切に対処します。作成できる具体的なエージェントプロファイルは次のとおりです。

  • 「Web ポリシーエージェント」

  • 「J2EE ポリシーエージェント」

  • 「Web サービスプロバイダ」

  • 「Web サービスクライアント属性」

  • 「セキュリティートークンサービス」

  • 「ディスカバリエージェント」

  • 「2.2 ポリシーエージェント」
