設定属性

「設定」ページには、Federated Access Manager のデフォルトサービスを設定するための属性がすべて含まれています。Federated Access Manager サービスを構成している属性は、次のいずれかのタイプに分類されます (複数のタイプを含むサービスもある)。

グローバル – Federated Access Manager 設定全体に適用されます。グローバル属性の目的は Identity Server アプリケーションのカスタマイズなので、ユーザー、ロール、およびレルムに適用することはできません。

レルム – レルム属性はレルムにのみ割り当てられます。レルム属性には、オブジェクトクラスは関連付けられません。認証はサブツリーレベルやユーザーレベルではなくレルムレベルで行われるため、認証サービスに一覧表示された属性は、レルム属性として定義されます。

動的 – Federated Access Manager で設定済みのロールまたはレルムに割り当てられます。ロールがユーザーに割り当てられるか、ユーザーがレルム内に作成されると、動的属性がそのユーザーの特性になります。

ユーザー – 各ユーザーに直接割り当てられます。ユーザー属性はロールとレルムからは継承されないため、通常はユーザーごとに異なります。

変更できる設定プロパティーは次のとおりです。

  • 「認証」

  • 「コンソールプロパティー」

  • 「グローバル属性」

  • 「システムプロパティー」
