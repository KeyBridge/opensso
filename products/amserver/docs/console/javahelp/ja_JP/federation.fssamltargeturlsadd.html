ターゲット URL を設定する

次の副属性は、POST により表明を受け取る各ターゲット URL に対して定義したり変更したりできます。

  1. 「新規」をクリックして、新しいターゲット URL を追加します。

    「ターゲット URL に新規 POST を追加」ページが表示されます。定義済みのターゲット URL で「編集」または「複製」ボタンをクリックしても、このページを表示できます。

  2. 各属性に値を入力します。

    プロトコル

    http または https のどちらかを選択します。

    サーバー名

    ターゲット URL があるサーバーの名前。例、www.sun.com。

    ポート

    この属性には、 58080 などのポート番号が入ります。

    パス

    この属性には、/amserver/console などの URI が入ります。

  3. 「了解」をクリックしてターゲット URL の設定を完了します。

  4. 「SAML プロファイル」ページで「保存」をクリックして、SAML の設定を完了します。
