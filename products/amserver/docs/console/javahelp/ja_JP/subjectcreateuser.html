ユーザーを作成または変更する

  1. 「ユーザー」タブをクリックします。

  2. 「新規」をクリックします。

  3. 次のフィールドにデータを入力します。

    ユーザー ID: このフィールドには、ユーザーが Access Manager にログインするときに使用する名前を入力します。このプロパティーは非 DN 値であってもかまいません。

    名: このフィールドには、ユーザーの下の名前を入力します。

    姓: このフィールドには、ユーザーの姓を入力します。

    フルネーム: このフィールドには、ユーザーのフルネームを入力します。

    パスワード: このフィールドには、「ユーザー ID」フィールドで指定した名前のパスワードを入力します。

    パスワード (確認): パスワードを確認します。

    ユーザー状態: このオプションは、ユーザーが Access Manager を通じた認証を許可されているかどうかを示します。

  4. 「了解」をクリックします。

  5. ユーザーの作成が完了したら、そのユーザーの情報を編集できます。それには、ユーザーの名前をクリックします。ユーザーの属性の詳細については、「ユーザー属性」を参照してください。実行可能なその他の変更作業を次に示します。

    • 「サービスをアイデンティティーに追加する」

    • 「ユーザーをロールとグループに追加する」
