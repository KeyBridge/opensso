サービス

Access Manager において、サービスとは、Access Manager コンソールでまとめて管理される属性のグループのことです。属性は、従業員の名前、職種、電子メールアドレスなどといった、単なる一連の関連情報であってもかまいません。しかし、属性は通常、電子メールアプリケーションや従業員名簿サービスなど、特定のソフトウェアモジュールの設定パラメータとして使用されます。

「サービス」タブでは、Access Manager の各種デフォルトサービスをレルムに追加したり、それらのサービスを設定したりできます。追加可能なサービスは次のとおりです。

  • 「管理」

  • ディスカバリサービス

  • 国際化設定

  • 「パスワードリセット」

  • セッション

  • 「ユーザー」
