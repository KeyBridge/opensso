新しいレルムを作成する

  1. 「アクセス制御」タブの下にある「レルム」リストから「新規」を選択します。

  2. 次の一般属性を定義します。

    名前

    レルムの名前を入力します。

    親

    作成するレルムの格納場所を定義します。新しいレルムに対する親レルムを選択します。

  3. 次のレルム属性を定義します。

    レルムの状態

    「アクティブ」、「非アクティブ」のいずれかの状態を選択します。デフォルトは「アクティブ」です。この値は、レルムのライフサイクルの任意のタイミングで変更可能です。変更するにはラジオボタンを選択します。「非アクティブ」を選択すると、ログイン時にユーザーアクセスが無効になります。

    レルムまたは DNS のエイリアス

    このレルムの DNS 名に対するエイリアス名を追加できます。この属性に設定できるのは、「実際の」ドメインエイリアスだけです (ランダム文字列は許可されない)。

  4. 「了解」をクリックして保存するか、「取消し」をクリックして前のページに戻ります。
