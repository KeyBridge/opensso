エージェント

Access Manager のポリシーエージェントは、Web サーバーや Web プロキシサーバー上のコンテンツを不正な侵入から保護します。それらのエージェントは、管理者が設定したポリシーに基づいて、サービスや Web リソースへのアクセスを制御します。

「エージェント」オブジェクトはポリシーエージェントプロファイルを定義し、Access Manager リソースを保護している特定のエージェントについての認証情報やその他のプロファイル情報を Access Manager で格納できるようにします。管理者は、Access Manager コンソールを使ってエージェントプロファイルを参照、作成、変更、および削除できます。

「エージェントを作成または変更する」を参照してください。
