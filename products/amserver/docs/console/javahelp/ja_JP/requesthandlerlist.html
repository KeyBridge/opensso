要求ハンドラリスト

SOAP バインドサービスは、要求メッセージを処理して応答を返信するための RequestHandler インタフェースを提供します。このインタフェースは、SOAP バインドサービスを使用している Liberty ベースの各アイデンティティーサービスによって、サーバー側に実装される必要があります。「要求ハンドラリスト」属性は、要求ハンドラを実装している Web サービスの実装クラスに関する情報を格納します。要求ハンドラリストには、キーと値のペアを含むエントリがパイプ文字で区切られて表示されます。たとえば、key=disco|class=com.example.identity.liberty.ws.disco.DiscoveryService のように表示されます。次の表は各パラメータの説明です。

パラメータ

ヘッダー

key

必須のキーパラメータは、SOAP エンドポイントに対する URI パスの最後の部分です。Access Manager の SOAP エンドポイントは、SOAP 受信者サーブレットです。SOAP 受信者の URI は protocol:// host:port/deploy-uri/Liberty/key です。キーとして disco を定義する場合、対応するディスカバリサービスの SOAP エンドポイントへの URI パスは protocol://host:port/amserver/Liberty/disco のようになります。SOAP 受信者への接続時には、さまざまなサービスクライアントがそれぞれ別のキーを使用します。

class

必須のクラスパラメータは、特定のアイデンティティーサービスの要求ハンドラ実装クラスの名前を指定します。たとえば、次のように指定します。class=com.example.identity.liberty.ws.disco.DiscoveryService

新しい要求ハンドラを追加するには、「要求ハンドラを設定する」を参照してください。
