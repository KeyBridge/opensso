新しいクライアントを追加する

  1. 「クライアントタイプ」リストで「新規」をクリックします。

  2. 次のフィールドを使用して、デバイスタイプを選択します。

    スタイル

    デバイスのベーススタイルを表示します。たとえば、HTML と指定します。

    デバイスユーザーエージェント

    デバイス名を指定します。

  3. 「次へ」をクリックします。

  4. 新しいデバイスについて次の情報を入力します。

    クライアントタイプ名

    デバイス名を指定します。クライアントタイプ名は、すべてのデバイスを通じて一意である必要があります。

    HTTP ユーザーエージェント文字列

    HTTP 要求ヘッダーのユーザーエージェントを定義します。たとえば、Mozilla/4.0 を定義します。

  5. 「完了」をクリックします。

  6. デバイスとそのプロパティーを複製するには、「複製」リンクをクリックします。デバイス名は、一意である必要があります。デフォルトでは、Access Manager によって、デバイス名が copy_of_devicename に変更されます。
