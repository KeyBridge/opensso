リソース ID マッパーを設定する

com.sun.identity.liberty.ws.interfaces.ResourceIDMapper は、ユーザー ID を、それと関連付けられたリソース ID にマップするために使用されるインタフェースです。実装クラスを使用してさまざまな実装プラグインを開発し、属性に追加することができます。新しいリソース ID マッパーをディスカバリサービスに追加する手順を次に示します。Web サービスモジュール内のディスカバリサービス設定画面から作業を始めます。

  1. 「新規」をクリックして新規のマッパー属性を表示するか、設定済みマッパーの名前をクリックしてそのマッパーを変更します。

  2. 次の情報に基づいて、マッパー属性の値を入力します。

    プロバイダ ID

    この属性は、ディスカバリサービスを指す URI を値として受け入れます。値は http://host: port/amserver/Liberty/disco の形式で記述します。

    ID マッパー

    実装クラスの名前を入力します。

  3. 「了解」をクリックしてマッパーの設定を完了します。

  4. ディスカバリサービスページで「保存」をクリックして、サービスの設定を完了します。
