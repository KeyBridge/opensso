新規メカニズムハンドラを設定する

  1. 「追加」をクリックして「新規メカニズムハンドラ」属性を表示するか、設定済みのキー値をクリックしてそれを変更します。

  2. 次の情報に基づいて属性の値を入力します。

    キー

    この必須パラメータは、認証 Web サービスがサポートする SASL メカニズムを定義します。

    クラス

    この必須パラメータは、SASL メカニズムの実装クラス名を指定します。認証 Web サービスレイヤーは、ハンドラインタフェースを提供します。これは、各 SASL メカニズムが要求されたメッセージを処理して応答を返すために実装される必要があります。

  3. 「了解」または「保存」をクリックして、要求ハンドラの設定を完了します。

  4. 認証 Web サービスページで「保存」をクリックして、サービスの設定を完了します。
