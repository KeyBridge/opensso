新しいインスタンス名を作成する

ネーミングサービスは初期化時にこの属性を読み込みます。このリストには、1 つの Access Manager 設定での Access Manager セッションサーバーが含まれています。

  1. 「インスタンス名」リストの「新規」をクリックします。

  2. 「サーバー」フィールドにホスト名とポートを入力します。

  3. サイト名を入力します。

    この値はサーバーを一意に識別します。負荷分散またはフェイルオーバーに関連する各サーバーには、一意の ID が必要です。この ID は、サーバー URL をサーバー ID にマッピングして Cookie の文字数を短くするためにも使用されます。構文は次のとおりです。

    intance_ID(|site_ID)

  4. 「了解」をクリックします。

    インスタンス名を編集するには、「インスタンス名」リストのエントリをクリックして、値を適宜変更します。

  5. プラットフォームサービスのメインページで「保存」をクリックします。
