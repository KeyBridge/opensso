エージェントがグループからプロパティーを継承できるようにする
始める前に

エージェントの属性の継承元となるグループは、あらかじめ作成しておく必要があります。

  1. 「設定」タブの「エージェント」をクリックします。
  2. 必要に応じて、適切なエージェントタイプのタブを選択します。
    • Web エージェント:

      「Web」タブは Web エージェント用で、デフォルトで選択されています。これ以上の選択は必要ありません。

    • J2EE エージェント:

      「J2EE」タブは J2EE エージェント用です。該当する場合は、このタブをクリックします。

    • 2.2 エージェント

      「2.2 エージェント」タブは、ポリシーエージェント 2.2 ソフトウェアセットのエージェント用です。該当する場合は、このタブをクリックします。

  3. 「エージェント」セクションで、設定するエージェントの名前をクリックします。
  4. 「グローバル」タブを選択し、「グループ」という名前の属性に対して、エージェントが属性を継承するグループの名前を選択します。

    ページの上部にある「継承設定値」ボタンがアクティブになります。

  5. 「継承設定値」をクリックします。
  6. エージェントがグループから継承する属性 (プロパティー) を選択します。
  7. 「保存」をクリックします。
