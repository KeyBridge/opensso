Pour permettre à un agent d'hériter des propriétés d'un groupe
Avant de commencer

Le groupe dont l'agent doit hériter des propriétés doit d'abord être créé.

  1. Cliquez sur Agents dans l'onglet Configuration.
  2. Si nécessaire, sélectionnez l'onglet correspondant au type d'agent approprié.
    • Agents Web :

      L'onglet Web est destiné aux agents Web et est sélectionné par défaut. Aucune autre sélection n'est nécessaire.

    • Agents J2EE :

      L'onglet J2EE est destiné aux agents J2EE. Le cas échéant, cliquez sur cet onglet.

    • Agents 2.2

      L'onglet Agents 2.2 est destiné aux agents de la suite logicielle Policy Agent 2.2. Le cas échéant, cliquez sur cet onglet.

  3. Dans la section Agent, cliquez sur le nom de l'agent que vous souhaitez configurer.
  4. L'onglet Global étant sélectionné, pour l'attribut Groupe, sélectionnez le nom du groupe dont les propriétés doivent être héritées par l'agent.

    En haut de la page, le bouton Paramètres d'héritage devient actif.

  5. Cliquez sur Paramètres d'héritage.
  6. Sélectionnez les attributs (propriétés) que l'agent doit hériter du groupe.
  7. Cliquez sur Enregistrer.
