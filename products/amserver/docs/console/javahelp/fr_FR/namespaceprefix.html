Préfixe d' espace de noms

La valeur de cet attribut spécifie le préfixe d'espace de noms utilisé pour les messages XML de Service de profil personnel Liberty. Un espace de noms différencie les éléments portant le même nom mais provenant de schémas XML différents. Ce préfixe est ajouté devant l'élément et permet de distinguer les métadonnées des différents espaces de noms de schéma XML.
