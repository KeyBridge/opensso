Configuration de l'authentification

Lorsqu'une instance de module d'authentification est définie, elle peut être configurée pour l'enchaînement de modules d'authentification, fournissant ainsi des URL de redirection, et une spécification de classe Java de post-traitement basée sur un processus d'authentification ayant réussi ou échoué. Pour pouvoir configurer une instance de module d'authentification, il est nécessaire de modifier l'attribut d'authentification Noyau Configuration de l'authentification des organisations et d'y inclure le nom de l'instance du module d'authentification spécifique.
