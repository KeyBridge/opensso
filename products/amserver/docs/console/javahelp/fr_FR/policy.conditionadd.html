Ajout d'une condition à une stratégie

  1. Si vous avez déjà créé la stratégie, cliquez sur le nom de la stratégie à laquelle vous souhaitez ajouter la condition. Si vous n'avez pas encore créé la stratégie, reportez-vous à Création d'une stratégie normale

  2. Sous la liste Conditions, cliquez sur Nouveau.

  3. Sélectionnez le type de condition et cliquez sur Suivant.

  4. Définissez les champs du type de condition. Pour une description des types de condition, cliquez sur l'un des liens suivants :

    • Durée de session active

    • Chaîne d'authentification

    • Niveau d'authentification (supérieur ou égal à)

    • Niveau d'authentification (inférieur ou égal à)

    • Instance du module d'authentification

    • Propriétés de session active

    • Adresse IP/Nom DNS

    • Condition de filtre LDAP

    • Authentification de domaine

    • Temps (jour, date, heure et fuseau horaire)

  5. Cliquez sur Terminer.
