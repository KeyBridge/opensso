Modification du profil d'un contact dans une entité fournisseur

Les informations relatives à un contact (individu) pour un fournisseur de services ou d'identités particulier peuvent être ajoutées au profil du fournisseur dans une entité fournisseur. Le point de départ est l'écran Entités sous Fédération.

  1. Cliquez sur le nom d'une entité configurée pour modifier son profil.

  2. Sélectionnez Fournisseur d'identités ou Fournisseur de services dans le menu Affichage.

  3. Cliquez sur le nom d'un contact configuré pour modifier son profil.

  4. Fournissez les valeurs nouvelles ou modifiées du profil du contact.

    Prénom

    Tapez le nom de baptême du contact de l'entité.

    Nom

    Tapez le nom de famille du contact de l'entité.

    Type

    Choisissez le type du contact dans le menu déroulant :

    • Administration

    • Facturation

    • Technique

    • Autre

    Société

    Tapez le nom de la société qui emploie cette personne.

    ID principal Liberty

    Tapez un URI qui pointe sur une instance en ligne du profil d'informations personnelles du contact.

    Adresses e-mail

    Tapez une ou plusieurs adresses e-mail pour le contact, puis cliquez sur Ajouter.

    Numéros de téléphone

    Tapez un ou plusieurs numéros de téléphone pour le contact, puis cliquez sur Ajouter.

  5. Cliquez sur Enregistrer pour terminer la configuration du contact.

  6. Cliquez sur Enregistrer dans la page Profil de l'entité pour terminer la configuration de l'entité.
