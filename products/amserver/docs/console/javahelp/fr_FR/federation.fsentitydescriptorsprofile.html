Modification des attributs généraux d'une entité fournisseur ou d'une entité filiale

Après avoir créé une entité, vous pouvez modifier son profil. En particulier, vous pouvez modifier des attributs, ajouter des fournisseurs d'identités et de services ou configurer des affiliations. Les opérations de modification des attributs généraux d'un profil d'entité peuvent inclure la modification des attributs communs déjà définis, l'ajout d'un contact ou la saisie d'informations facultatives concernant l'organisation. La procédure suivante permet de modifier les attributs généraux. Le point de départ est l'écran Entités sous Fédération.

  1. Sélectionnez le nom d'une entité configurée pour modifier son profil.

    Par défaut, la page des attributs généraux du profil de l'entité s'affiche. Si vous n'accédez pas à cette page, vous pouvez sélectionner Général dans le menu Affichage.

  2. Modifiez les valeurs des attributs communs de l'entité.

    Nom de l'entité

    La valeur statique de cet attribut est le nom fourni lors de la création de l'entité.

    Type

    La valeur statique de cet attribut est le type de l'entité, Fournisseur ou Filiale.

    Description

    La valeur de cet attribut est la description fournie lors de la création de l'entité. Vous pouvez modifier la description initiale ou en ajouter une s'il n'y en a pas.

  3. Fournissez les valeurs des attributs de profil du contact de l'entité.

    Ces attributs donnent des informations sur la personne à contacter pour l'entité. Ils sont facultatifs.

    Prénom

    Tapez le nom de baptême du contact de l'entité.

    Nom

    Tapez le nom de famille du contact de l'entité.

    Type

    Choisissez le type du contact dans le menu déroulant :

    • Administration

    • Facturation

    • Technique

    • Autre

    Société

    Tapez le nom de la société qui emploie cette personne.

    ID principal Liberty

    Tapez un URI (Uniform Resource Identifier) qui pointe sur une instance du profil des informations personnelles du contact.

    Adresses e-mail

    Tapez une ou plusieurs adresses e-mail pour le contact, puis cliquez sur Ajouter.

    Numéros de téléphone

    Tapez un ou plusieurs numéros de téléphone pour le contact, puis cliquez sur Ajouter.

  4. Fournissez les valeurs des attributs des profils de l'organisation.

    Ces attributs donnent des informations de base qui peuvent être nécessaires lors d'une interaction avec une entité principale. Ils sont facultatifs.

    Remarque – Si l'attribut Noms contient une valeur, il faut donner des valeurs aux attributs Noms d'affichage et URL.

    Noms

    Entrez le nom officiel de l'organisation de l'entité, puis cliquez sur Ajouter. La valeur est définie au format locale| organization-name. Par exemple, en| example.com.

    Noms d'affichage

    Entrez le nom de l'organisation de l'entité qui s'affichera en premier, puis cliquez sur Ajouter. La valeur est définie au format locale|organization-display-name . Par exemple, fr|exemple.com.

    URL

    Entrez l'URL de l'organisation, puis cliquez sur Ajouter. La valeur est définie au format locale| organization-URL. Par exemple, en| http://www.example.com .

  5. Cliquez sur Enregistrer pour terminer la configuration, ou continuez à définir des valeurs pour le fournisseur d'identités, le fournisseur de services ou les attributs de filiale.

    Si vous poursuivez la configuration, consultez :

    • Ajout d'un fournisseur d'identités à une entité fournisseur pour définir les valeurs des attributs du fournisseur d'identités.

    • Ajout d'un fournisseur de services à une entité fournisseur pour définir les valeurs des attributs du fournisseur de services.

    • Ajout de membres à une entité filiale pour renseigner une entité filiale avec des membres.
