Magasins de données

Un magasin de données est une base de données dans laquelle vous pouvez stocker des attributs utilisateur et des données de configuration utilisateur.

Access Manager fournit des plug-ins du référentiel d'identités qui se connectent à une structure de référentiel d'identités LDAPv3. Ceci vous permet de consulter et de récupérer les informations utilisateur Access Manager sans apporter de modification à votre base de données utilisateur existante. La structure Access Manager intègre les données du plug-in du référentiel d'identités avec les données d'autres plug-ins Access Manager afin de constituer une identité virtuelle pour chaque utilisateur. Access Manager peut alors utiliser cette identité universelle dans les processus d'authentification et d'autorisation au sein de plusieurs référentiels d'identités. L'identité d'utilisateur virtuelle est détruite lorsque la session de l'utilisateur prend fin.

Access Manager prend en charge les magasins de données suivants :

  • Plug-in du référentiel d'Access Manager

  • Active Directory

  • Référentiel fichiers plats

  • LDAPv3 générique

  • Sun Directory Server avec Access Manager Schema

Pour plus d'informations sur la création d'un magasin de données et sur la configuration de plug-ins du référentiel, reportez-vous à Création d'un magasin de données.
