Création d'un cercle d'approbation

Procédez comme suit pour créer un cercle d'approbation. Le point de départ est Nouveau cercle d'approbation sous l'interface Fédération.

  1. Cliquez sur Nouveau pour afficher le cercle d'attributs d'approbation.

    La page Nouveau cercle de profil d'approbation s'affiche.

  2. Entrez le nom du cercle d'approbation.
  3. Entrez la description du cercle d'approbation dans le champ Description.
  4. Tapez la valeur de l'URL du service écriture IDFF.

    L'URL du service écriture IDFF indique l'emplacement du servlet qui écrit le cookie du domaine commun. Utilisez le format http:// hôte-domaine-commun :port/common/writer .

  5. Tapez la valeur de l'URL du service lecture IDFF.

    L'URL du service lecture IDFF indique l'emplacement du servlet qui lit le cookie du domaine commun. Utilisez le format http:// hôte-domaine-commun :port/common/transfer ..

  6. Tapez la valeur de l'URL du service écriture SAML2.

    Ceci indique l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun.

  7. Tapez la valeur de l'URL du service lecture SAML2.

    Ceci indique l'emplacement du service lecture SAML2 qui lit le cookie dans le domaine commun.

  8. Choisissez Actif ou Inactif.

    Le statut par défaut est Actif. Le choix de Inactif désactive les communications au sein du domaine d'authentification.

  9. Sélectionnez le domaine dans lequel le cercle d'approbation sera créé.
  10. Choisissez un ou plusieurs fournisseurs disponibles et cliquez sur la flèche Ajouter pour les sélectionner.

    La liste fournie contient les noms des entités créées et renseignées avec des fournisseurs. Pour plus d'informations, voir Ajouts de fournisseurs à un cercle d'approbation

  11. Cliquez sur OK pour terminer la configuration.

    Le nouveau cercle d'approbation s'affiche dans la liste Cercle d'approbation.
