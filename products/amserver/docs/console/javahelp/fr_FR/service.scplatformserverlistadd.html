Création d'un nom d'isntance

Ce service d'attribution de nom lit cet attribut à l'initialisation. Cette liste contient les serveurs de session Access Manager d'une configuration Access Manager unique.

  1. Cliquez sur Nouveau dans la liste Nom d'instance.

  2. Entrez le nom d'hôte et le port dans le champ Serveur.

  3. Entrez le Nom du site.

    Cette valeur identifie le serveur de manière unique. Chaque serveur qui participe à l'équilibrage de la charge ou au basculement doit disposer d'un ID unique. Ceci permet également de raccourcir la longueur des cookies en mappant l'URL du serveur à l'ID du serveur. La syntaxe est la suivante :

    ID_intance(|ID_site)

  4. Cliquez sur OK.

    Pour modifier le nom d'une instance, cliquez sur une entrée dans la liste Nom d'instance et modifiez les valeurs souhaitées.

  5. Cliquez sur Enregistrer dans la page principale du service Plate-forme.
