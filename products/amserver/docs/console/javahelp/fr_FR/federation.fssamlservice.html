SAML

SAML (Security Assertion Markup Language) est un protocole de norme ouverte utilisant la structure XML pour échanger des informations de sécurité entre une autorité et un site partenaire de confiance. Les informations de sécurité ont trait à l'état d'authentification de l'objet, à l'autorisation d'accès et aux informations d'attribut. (Une personne identifiée par une adresse e-mail est un objet, comme peut l'être une imprimante.) Une autorité SAML (appelée parfois partie certifiante) est une plate-forme ou une application intégrée à l'API SAML, l'autorisant à relayer des informations de sécurité. Par exemple, la certification qu'un objet a été authentifié dans son système en fournissant les attributs nécessaires. Les sites partenaires de confiance reçoivent des informations de sécurité dont l'authenticité est avérée.

Les attributs SAML suivants peuvent être configurés pour votre implémentation :

  • Indicateur cible

  • Identificateurs de site

  • Partenaires de confiance

  • URL cibles

  • Délai d'assertion

  • Facteur d'inclinaison de l'assertion pour la durée notBefore

  • Délai d'artéfact

  • Nom de l'artéfact SAML

  • Signer l'assertion SAML

  • Signer la demande SAML

  • Signer la réponse SAML

Les tâches suivantes sont associées à la configuration SAML :

  • Configuration d'un identificateur de site

  • Partenaires de confiance : sélection du type et du profil de partenaire

  • Partenaires de confiance : configuration des attributs du partenaire de confiance

  • Configuration d'un URL cible
