Gestion d'annuaire

L'onglet Gestion d'annuaire ne s'affiche que lorsque Access Manager est installé en mode hérité. Cette fonctionnalité de gestion d'annuaire fournit une solution de gestion des identités pour les déploiements Access Manager compatibles avec Sun Java System Directory Server.

Pour plus d'informations sur les options d'installation du mode hérité, reportez-vous au manuel Sun Java Enterprise System Installation Guide for UNIX
