Ajout d'un objet à une stratégie

  1. Si vous avez déjà créé la stratégie, cliquez sur le nom de la stratégie à laquelle vous souhaitez ajouter l'objet. Si vous n'avez pas encore créé la stratégie, reportez-vous à Création d'une stratégie normale.

  2. Sous la liste Objet, cliquez sur Nouveau.

  3. Sélectionnez l'un des types d'objet par défaut. Pour une description des types d'objet, reportez-vous à Objets.

  4. Cliquez sur Suivant.

  5. Entrez un nom pour l'objet.

  6. Sélectionnez ou désélectionnez le champ Exclusif.

    Si ce champ n'est pas sélectionné (valeur par défaut), la stratégie s'applique à l'identité membre de l'objet. Si ce champ est sélectionné, la stratégie s'applique à l'identité qui n'est pas membre de l'objet.

    Si la stratégie comporte plusieurs objets, elle s'applique à l'identité lorsqu'au moins un des objets implique que la stratégie s'applique à l'identité donnée.

  7. Effectuez une recherche afin d'afficher les identités à ajouter à l'objet. Cette étape n'est pas applicable dans le cas des objets Utilisateurs authentifiés ou Clients de services Web.

    Le modèle de recherche par défaut (*) affiche toutes les entrées qualifiées.

  8. Sélectionnez les différentes identités que vous voulez ajouter à l'objet, ou cliquez sur Tout ajouter pour ajouter toutes les identités en une seule opération. Cliquez sur Ajouter pour déplacer les identités vers la liste Sélectionné. Cette étape n'est pas applicable dans le cas d'un objet Utilisateurs authentifiés.

  9. Cliquez sur Terminer.

  10. Pour supprimer un objet d'une stratégie, sélectionnez l'objet et cliquez sur Supprimer. Vous pouvez modifier n'importe quelle définition d'objet en cliquant sur son nom.
