Nom de l'artéfact SAML

Cet attribut assigne un nom de variable à un artéfact SAML. Cet artéfact est composé de données avec limite de taille identifiant une assertion et un site source. Il est inclus dans la chaîne de requête URL et redirigé jusqu'au site de destination. Le nom par défaut est SAMLart. Avec la valeur par défaut SAMLart, la chaîne de requête de redirection peut être http://host:port /deploy-URI/ SamlAwareServlet?TARGET=target-URL /&SAMLart=artifact123.
