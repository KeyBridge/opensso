Modification d'un profil de cercle d'approbation

Suivez cette procédure pour modifier les attributs globaux d'un cercle d'approbation existant ou pour lui ajouter des fournisseurs. Le point de départ est Cercle d'approbation sous l'interface Fédération.

  1. Cliquez sur le nom d'un cercle d'approbation configuré afin de modifier son profil ou de lui ajouter des fournisseurs.

    La page Modification du cercle d'approbation s'affiche.

  2. Tapez les nouvelles valeurs ou modifiez celles présentes dans les attributs généraux du cercle d'approbation :
    Nom

    La valeur statique de cet attribut est le nom fourni lors de la création du cercle d'approbation.

    Description

    La valeur de cet attribut est une description du cercle d'approbation. Si nécessaire, vous pouvez modifier la description déjà saisie.

    URL du service écriture IDFF

    Cet attribut indique l'emplacement du service qui écrit le cookie du domaine commun. L'URL a le format http://common-domain-host:port/common/writer .

    URL du service lecture IDFF

    Cet attribut indique l'emplacement du service qui lit le cookie du domaine commun. L'URL a le format http://common-domain-host:port/common/transfer .

    URL du service écriture SAML2

    Cet attribut spécifie l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun.

    URL du service lecture SAML2

    Cet attribut spécifie l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun.

    Statut

    Le statut par défaut est Actif. Sélectionner Inactif désactive la communication avec le cercle d'approbation.

  3. Choisissez un ou plusieurs fournisseurs disponibles et cliquez sur la flèche Ajouter pour les sélectionner.

    La liste fournie contient les noms des entités créées et renseignées avec des fournisseurs. Pour plus d'informations, voir Ajouts de fournisseurs à un cercle d'approbation

  4. Cliquez sur Enregistrer pour terminer l'opération.
