Domaines

Un domaine de contrôle d'accès est un groupe de propriétés d'authentification et de stratégies d'autorisation que vous pouvez associer à un utilisateur ou un groupe d'utilisateurs. Les données de domaine sont stockées dans une arborescence d'informations qui est créée par Access Manager dans le magasin de données que vous indiquez. La structure Access Manager regroupe les stratégies et les propriétés contenues dans chaque domaine de l'arborescence d'informations Access Manager. Par défaut, Access Manager 7.1 insère automatiquement l'arborescence d'informations Access Manager en tant que branche spéciale dans Sun Java Enterprise System Directory Server, en plus des données utilisateur. Vous pouvez utiliser les domaines de contrôle d'accès avec toute base de données LDAPv3.
