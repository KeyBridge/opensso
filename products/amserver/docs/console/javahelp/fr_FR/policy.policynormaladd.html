Création d'une stratégie normale

  1. Choisissez le domaine pour lequel vous voulez créer une stratégie.

  2. Cliquez sur Nouvelle stratégie dans la liste Stratégies.

  3. Ajoutez un nom et une description pour la stratégie.

  4. Si vous souhaitez que la stratégie soit active, vérifiez que Oui est sélectionné pour l'attribut Active.

    Oui est sélectionné par défaut. Pour que la stratégie ne s'applique pas pendant l'évaluation de la stratégie, désélectionnez cet attribut pour le désactiver.

  5. Il n'est pas nécessaire de définir tous les champs des stratégies normales à ce stade. Vous pouvez créer la stratégie, puis ajouter les règles, objets, conditions et fournisseurs de réponses ultérieurement. Si vous souhaitez définir les champs maintenant, reportez-vous aux rubriques suivantes :

    • Ajout d'une règle à une stratégie

    • Ajout d'un objet à une stratégie

    • Ajout d'une condition à une stratégie

    • Ajout d'un fournisseur de réponses à une stratégie

  6. Cliquez sur OK.
