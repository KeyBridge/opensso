Ajout d'une configuration secondaire

  1. Cliquez sur Nouveau dans la liste Instance de configuration secondaire.

  2. Entrez le nom de la nouvelle sous-configuration.

  3. Renseignez les champs suivants :

    Utilisateur de l'emplacement de stockage de sessions

    Définit l'utilisateur de la base de données qui est utilisé pour récupérer et stocker les données de session.

    Mot de passe pour l'emplacement de stockage de sessions

    Définit le mot de passe de l'utilisateur de la base de données défini dans Stockage de sessions.

    Mot de passe pour l'emplacement de stockage de sessions (confirmer)

    : confirmez le mot de passe.

    Temps d'attente maximal

    Définit le délai d'attente total que peut accepter un thread lors de l'acquisition d'un objet de connexion de base de données. Cette valeur est exprimée en millisecondes.

    URL de la base de données

    Indique l'URL de la base de données.

  4. Cliquez sur Ajouter.
