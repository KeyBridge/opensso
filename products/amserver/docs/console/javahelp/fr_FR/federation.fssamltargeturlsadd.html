Configuration d'un URL cible

Les sous-attributs suivants peuvent être définis (ou modifiés) pour chaque URL cible recevant des assertions via POST.

  1. Cliquez sur Nouveau pour ajouter un nouvel URL cible.

    La page Ajout d'un nouvel URL d'envoi à la cible d'affiche. Vous pouvez atteindre également cette page en sélectionnant le bouton Modifier ou Dupliquer d'un URL cible défini.

  2. Tapez les valeurs des attributs.

    Protocole

    Choisissez http ou https.

    Nom de serveur

    Nom du serveur sur lequel l'URL cible se trouve, par exemple www.sun.com.

    Port

    Cet attribut contient le numéro de port, par exemple 58080.

    Chemin

    Cet attribut contient l'URI, par exemple /amserver/console.

  3. Cliquez sur OK pour terminer la configuration de l'URL cible.

  4. Cliquez sur Enregistrer dans la page Profil SAML pour terminer la configuration SAML.
