Offres de ressources pour l'amorçage

Cet attribut définit une offre de ressources pour l'amorçage d'un service. Après la connexion unique (SSO), cette offre de ressources et les données d'authentification associées sont envoyées au client dans l'assertion SSO. Une seule offre de ressources est autorisée pour l'amorçage ; par défaut, cette offre contient des informations concernant le Service de découverte. Les tâches associées à cet attribut comprennent :

  • Configuration d'une offre de ressource pour l'amorçage

  • Configuration d'une description de service
