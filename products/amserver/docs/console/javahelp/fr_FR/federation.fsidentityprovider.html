Ajout d'un fournisseur d'identités à une entité fournisseur

La première étape du processus d'ajout d'un fournisseur d'identités à une entité fournisseur consiste à définir le type de fournisseur d'identités ajouté.

  • Un fournisseur hébergé est hébergé sur le même serveur qu'Access Manager.

  • Un fournisseur distant n'est pas hébergé sur le même serveur qu'Access Manager.

La procédure suivante décrit l'étape initiale. Le point de départ est l'écran Entités sous Fédération.

  1. Sélectionnez le nom d'une entité fournisseur configurée pour modifier son profil.

    La page du profil de l'entité s'affiche.

  2. Dans le menu Afficher, sélectionnez Fournisseur d'identités.

  3. Sélectionnez le type du fournisseur que vous ajoutez.

    • Choisissez Nouveau fournisseur hébergé si le fournisseur est hébergé sur le même serveur qu'Access Manager.

    • Choisissez Nouveau fournisseur distant si le fournisseur n'est pas hébergé sur le même serveur qu'Access Manager.

    La page du profil du fournisseur d'identités s'affiche.

  4. Selon votre sélection à l'étape précédente, faites votre choix parmi les possibilités suivantes :

    • Ajout d'un nouveau fournisseur d'identités hébergé à une entité fournisseur

    • Ajout d'un nouveau fournisseur d'identités distant à une entité fournisseur
