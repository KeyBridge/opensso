Ajout de services à une identité

  1. Sélectionnez l'identité à laquelle vous souhaitez ajouter des services.

  2. Cliquez sur l'onglet Services.

  3. Cliquez sur Ajouter.

  4. Selon le type d'identité sélectionné, une liste des services suivants apparaît :

    • Configuration de l'authentification

    • Service de découverte

    • Service de profil personnel Liberty

    • Session

    • Utilisateur

  5. Sélectionnez le service à ajouter et cliquez sur Suivant.

  6. Modifiez les attributs pour le service. Pour obtenir une description des services, cliquez sur le nom du service à l'étape 4.

  7. Cliquez sur Terminer.
