Agents

Les agents de stratégie Access Manager protègent le contenu des serveurs Web et proxy Web contre les accès non autorisés. Ils contrôlent l'accès aux services et aux ressources Web d'après les stratégies configurées par un administrateur.

L'objet agent définit un profil d'agent de stratégie et permet à Access Manager de stocker les informations d'authentification et d'autres informations de profil concernant l'agent qui assure la protection d'une ressource Access Manager. À l'aide de la console d'Access Manager, les administrateurs peuvent afficher, créer, modifier et supprimer des profils d'agent.

Reportez-vous à Création ou modification d'un agent.
