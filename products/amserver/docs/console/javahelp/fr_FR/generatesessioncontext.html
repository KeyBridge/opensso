Instructions comportant la session pour l'amorçage

Lorsqu'elle est sélectionnée, cette option spécifie si une instruction SessionContextStatement doit être générée pour l'amorçage. L'argument SessionConxtext de l'instruction SessionContextStatement est requis par le Service de découverte pour la prise en charge de la directive AuthenicateSessionContext. Par défaut, cette option n'est pas sélectionnée.
