Création d'un jeu de caractères

  1. Cliquez sur Nouveau dans la liste Jeux de caractères client.

  2. Entrez une valeur pour le Type de client.

  3. Entrez une valeur pour le Jeu de caractères. Pour connaître les jeux de caractères disponibles, reportez-vous à Environnements linguistiques pris en charge.

  4. Cliquez sur OK.

  5. Cliquez sur Enregistrer dans la page principale du service Plate-forme.
