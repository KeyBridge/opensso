Attributs de conteneur d'extension

Service de profil personnel Liberty vous permet de spécifier les attributs d'extension qui ne sont pas définis dans les spécifications Liberty Alliance Project. Les valeurs de cet attribut spécifient une liste d'attributs de conteneur d'extension. Toutes les extensions doivent être définies sous la forme : /PP/Extension/PPISExtension [@name='extensionattribute']
