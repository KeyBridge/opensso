Mappeur d'ID de ressource

La valeur de cet attribut spécifie l'implémentation de la classe com.sun.identity.liberty.ws.interfaces.ResourceIDMapper. Bien qu'une nouvelle implémentation puisse être développée, Access Manager fournit la classe par défaut com.sun.identity.liberty.ws.idpp.plugin.IDPPResourceIDMapper qui mappe un identificateur de ressource de découverte à un ID utilisateur.
