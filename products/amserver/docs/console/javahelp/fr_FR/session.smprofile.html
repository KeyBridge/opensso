Fenêtre de session

Affiche les utilisateurs connectés à Federated Access Manager et la durée de la session pour chacun d'entre eux. Les champs affichés sont les suivants :

ID d'utilisateur ;

Affiche l'ID utilisateur de l'utilisateur actuellement connecté.

Temps restant (en minutes)

Affiche la durée restante (en minutes) dont dispose l'utilisateur pour cette session avant de devoir se réauthentifier.

Durée max. de la session (en minutes)

Affiche la durée maximale (en minutes) pendant laquelle l'utilisateur peut rester connecté avant que la session n'arrive à expiration et qu'il doive se réauthentifier pour accéder de nouveau à Access Manager.

Durée d'inactivité (en minutes)

Affiche la durée (en minutes) depuis laquelle l'utilisateur est inactif.

Durée max. d'inactivité (en minutes)

Affiche la durée maximale (en minutes) pendant laquelle un utilisateur peut rester inactif sans avoir à se réauthentifier. Les limites de durée sont définies par l'administrateur dans le service Session. Pour une définition des attributs de session, reportez-vous à Session.
