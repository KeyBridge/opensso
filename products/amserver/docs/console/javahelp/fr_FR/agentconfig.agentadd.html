Création d'un agent

Créer un agent est également appelé créer un profil d'agent. Vous pouvez effectuer cette tâche dans la console d'administration de Federated Access Manager . Cette tâche s'applique aux agents Web, J2EE et 2.2. Cependant, certaines étapes particulières ne s'appliquent pas aux agents 2.2.

  1. Cliquez sur Agents dans l'onglet Configuration.
  2. Si nécessaire, sélectionnez l'onglet correspondant au type d'agent approprié.
    • Agents Web :

      L'onglet Web est destiné aux agents Web et est sélectionné par défaut. Aucune autre sélection n'est nécessaire.

    • Agents J2EE :

      L'onglet J2EE est destiné aux agents J2EE. Le cas échéant, cliquez sur cet onglet.

    • Agents 2.2

      L'onglet Agents 2.2 est destiné aux agents de la suite logicielle Policy Agent 2.2. Le cas échéant, cliquez sur cet onglet.

  3. Dans la section Agent, cliquez sur Nouveau.
  4. Dans le champ Nom, entrez le nom du nouveau profil d'agent.
  5. Entrez et confirmez le mot de passe.

    Attention – Ce mot de passe doit être le mot de passe entré dans le fichier de mots de passe de profil d'agent que vous spécifiez lorsque vous exécutez le programme agentadmin pour installer l'agent.

  6. Sélectionnez la configuration applicable : locale ou centralisée (ne s'applique pas aux agents 2.2).

    De plus, si une configuration locale est sélectionnée, les attributs associés à cet agent ne peuvent être modifiés qu'à partir d'un fichier dans l'agent. Ils ne peuvent pas être modifiés à partir de la console.

  7. Dans le champ URL de serveur, entrez l'URL du serveur de Federated Access Manager (ne s'applique pas aux agents 2.2).

    Par exemple :

    http://fam.example.com:8080/fam
  8. Dans le champ URL d'agent, entrez l'URL de l'application d'agent, agentapp (ne s'applique pas aux agents 2.2).

    Par exemple :

    http://fam.example.com:8090/agentapp
  9. Cliquez sur Créer.

    La console crée le profil d'agent et affiche de nouveau la page d'agent (Web, J2EE ou 2.2) avec un lien vers le nouveau profil d'agent.

    Pour effectuer d'autres tâches de configuration du profil d'agent, cliquez sur ce lien afin d'afficher la page Modification de groupe d'agents.
