Création d'un rôle filtré

  1. Dans le volet de navigation, accédez à l'organisation dans laquelle le rôle sera créé.

  2. Cliquez sur Nouveau.

  3. Entrez le nom du rôle filtré.

  4. Entrez les informations relatives aux critères de recherche.

    Par exemple :

    (&(uid=user1)(|(inetuserstatus=active)(!(inetuserstatus=*))))

    Si le filtre est laissé vide, par défaut, le rôle suivant est créé :

    (objectclass = inetorgperson)

  5. Cliquez sur OK pour lancer la recherche sur la base des critères de filtrage. Les identités définies par les critères de filtrage sont affectées automatiquement au rôle.

  6. Une fois le rôle filtré créé, cliquez sur le nom du rôle pour afficher les utilisateurs appartenant à ce rôle. Vous pouvez également ajouter des services au rôle en cliquant sur l'onglet Services. Reportez-vous à Ajout de services à une identité.
