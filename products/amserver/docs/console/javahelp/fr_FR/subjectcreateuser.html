Création ou modification d'un utilisateur

  1. Cliquez sur l'onglet Utilisateur.

  2. Cliquez sur Nouveau.

  3. Renseignez les champs suivants :

    ID utilisateur. ce champ contient le nom de l'utilisateur avec lequel il ou elle se connecte à Access Manager. Cette propriété peut être une valeur autre qu'un DN.

    Prénom. ce champ contient le prénom de l'utilisateur.

    Nom ce champ contient le nom de l'uitlisateur.

    Nom complet ce champ contient le nom complet de l'utilisateur.

    Mot de passe. ce champ indique le mot de passe correspondant au nom inclus dans le champ ID utilisateur.

    Mot de passe (Confirmer) confirmez le mot de passe.

    Statut de l'utilisateur. cette option indique si l'utilisateur est autorisé à s'authentifier via Access Manager.

  4. Cliquez sur OK.

  5. Une fois l'utilisateur créé, vous pouvez modifier les informations utilisateur en cliquant sur le nom de l'utilisateur. Pour plus d'informations sur les attributs utilisateur, reportez-vous aux Attributs utilisateur. Les autres modifications que vous pouvez effectuer sont les suivantes :

    • Ajout de services à une identité

    • Ajout d'un utilisateur aux rôles et groupes
