Aide en ligne de Sun Java System Access Manager 7.1

Ce système d'aide en ligne fournit des informations centrées sur les différentes champs et tâches permettant de gérer les accès par l'intermédiaire de la console Access Manager.

Pour voir les informations relatives à un sujet donné, cliquez sur le bouton Aide de la page en cours de consultation ou déplacez-vous dans les différentes rubriques d'aide en sélectionnant l'un des onglets suivants :

  • Sommaire affiche les rubriques principales et les sous-rubriques.

  • Index répertorie les rubriques par ordre alphabétique.

  • Rechercher permet d'effectuer des recherches dans l'aide d'Access Manager.
