Configuration d'un conteneur pris en charge

Un conteneur est un attribut qui définit un emplacement réservé pour chaque donnée d'identité. La procédure permettant d'ajouter de nouveaux attributs à Service de profil personnel Liberty est la suivante. Le point de départ est l'écran de configuration de Service de profil personnel Liberty du module Services Web.

  1. Cliquez sur Nouveau pour afficher les nouveaux attributs du conteneur pris en charge ou cliquez sur le nom d'un conteneur configuré pour modifier son profil.

  2. Fournissez les valeurs pour les attributs de conteneur en vous basant sur les informations suivantes :

    Nom du conteneur

    Entrez le nom du conteneur, par exemple Carte bancaire.

    Plug-in

    Entrez le nom d'une classe pour la gestion globale du conteneur. Cette classe permet de modifier l'implémentation par défaut com.sun.identity.liberty.ws.idpp.plugin.IDPPContainer.

  3. Cliquez sur OK pour terminer la configuration du conteneur.

  4. Cliquez sur Enregistrer dans la page Service de profil personnel Liberty pour terminer la configuration du service.
