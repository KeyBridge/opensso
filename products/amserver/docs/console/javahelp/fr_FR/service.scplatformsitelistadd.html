Création d'un nom de site

  1. Cliquez sur Nouveau dans la liste Nom du site.

  2. Entrez le nom d'hôte et le port dans le champ Serveur.

  3. Entrez le Nom du site.

    Cette valeur identifie le serveur de manière unique. Chaque serveur qui participe à l'équilibrage de la charge ou au basculement doit disposer d'un ID unique se composant d'un nombre à deux chiffres. Par exemple, 01.

  4. Cliquez sur Enregistrer.

    Pour modifier le nom d'un site, cliquez sur une entrée dans la liste Nom de site et modifiez les valeurs souhaitées.
