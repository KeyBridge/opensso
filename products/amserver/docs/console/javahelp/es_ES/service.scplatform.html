Plataforma

El servicio de plataforma es aquél en el que se pueden agregar servidores adicionales a las configuraciones de Access Manager, así como otras opciones aplicadas en el nivel superior de la aplicación Access Manager. Los atributos del servicio de plataforma son globales. Los atributos son:

  • Nombre del sitio

  • Nombre de instancia

  • Idioma de plataforma

  • Dominios de cookies

  • URL de servicio de inicio de sesión

  • URL de servicio de cierre de sesión

  • Idiomas disponibles

  • Conjunto de caracteres del cliente

Nombre del sitio

El servicio de asignación de nombres lee este atributo en el momento de la inicialización. La lista identifica de forma exclusiva el nombre FQDN con el número de puerto del equilibrador de carga o el SRA de equilibrado de carga para los servidores secundarios de Access Manager. Si el host especificado en una solicitud para una URL de servicio no se encuentra en la lista, el servicio de asignación de nombres rechazará la solicitud. En este atributo sólo debe utilizarse el protocolo de servicio de asignación de nombres. Consulte Para crear un nombre de sitio nuevo.

Nombre de instancia

El servicio de asignación de nombres lee este atributo en el momento de la inicialización. Esta lista contiene los servidores de la sesión de Access Manager en una única configuración de Access Manager. Por ejemplo, si hay dos Access Manager instalados que deben funcionar como uno, deben incluirse ambos en esta lista. Si el host especificado en una solicitud para una URL de servicio no se encuentra en la lista, el servicio de asignación de nombres rechazará la solicitud. En este atributo sólo debe utilizarse el protocolo de servicio de asignación de nombres. Consulte Para crear un nombre de instancia nuevo.

Idioma de plataforma

El valor del idioma de la plataforma es el subtipo de idioma predeterminado con el que se instaló Access Manager. Los servicios administración, registro y autenticación se administran con el idioma de este valor. El valor predeterminado es en_US. Consulte Idiomas admitidos para obtener una lista de los subtipos de idiomas admitidos.

Dominios de cookies

La lista de dominios que se devolverán en el encabezado de cookie al configurar una cookie en el explorador del usuario durante la autenticación. Si se deja vacío, no se configurará ningún dominio de cookie. Es decir, la cookie de sesión de Access Manager sólo se reenviará a Access Manager y no al resto de servidores del dominio.

Si se requiere SSO con otros servidores del dominio, este atributo debe configurarse con el dominio de cookies. Si dispone de dos interfaces en dominios distintos de un Access Manager, deberá configurar ambos dominios de cookies en este atributo. Si se utiliza un equilibrador de carga, el dominio de cookies debe ser el del dominio del equilibrador de carga y no los servidores detrás del equilibrador. El valor predeterminado de este campo es el dominio del Access Manager instalado.

URL de servicio de inicio de sesión

Este campo especifica la URL de la página de inicio de sesión. El valor predeterminado de este atributo es /Service_DEPLOY_URI/UI/Login.

URL de servicio de cierre de sesión

Este campo especifica la URL de la página de cierre de sesión. El valor predeterminado de este atributo es /Service_DEPLOY_URI/UI/Logout.

Idiomas disponibles

Este atributo almacena todos las configuraciones regionales configuradas para la plataforma. Considere una aplicación que permite al usuario elegir la configuración regional del usuario. La aplicación obtendrá el atributo del perfil de la plataforma y mostrará al usuario la lista de configuraciones regionales. El usuario elegirá una configuración regional y la aplicación la configurará en la entrada de usuario preferredLocale.

Conjunto de caracteres del cliente

Este atributo especifica el conjunto de caracteres establecido para los distintos clientes en el nivel de la plataforma. Contiene una lista con los tipos de clientes y los correspondientes conjuntos de caracteres. Consulte Para crear un conjunto de caracteres nuevo para obtener más información.
