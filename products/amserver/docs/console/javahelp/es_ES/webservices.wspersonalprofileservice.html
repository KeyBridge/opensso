Servicio de perfil personal de Liberty

La especificación de servicio de perfil personal ID-SIS de Liberty describe un servicio de datos que proporciona una información de perfil básico de la identidad. Está diseñado para ser el menor denominador común para albergar información basada en el consumidor sobre una entidad principal. Access Manager ha implementado esta especificación y desarrollado el Servicio de perfil personal de Liberty. Se puede consultar el Servicio de perfil personal de Liberty para buscar datos de identidad o también pueden actualizarse estos datos.

Nota – Para acceder a este servicio, el proveedor que aloja el Servicio de perfil personal de Liberty debe registrarse en el Servicio de detección en nombre de cada principal de identidad. Consulte Servicio de detección para obtener más información.
