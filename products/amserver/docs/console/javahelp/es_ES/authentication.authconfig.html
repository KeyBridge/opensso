Configuración de autenticación

Una vez definida la instancia del módulo de autenticación, ésta puede configurarse para el encadenamiento de módulo de autenticación con el fin de proporcionar las URL de redirección y una especificación de clase de Java de procesamiento posterior basada en un proceso de autenticación satisfactorio o fallido. Antes de poder configurar una instancia del módulo, debe modificarse el atributo de autenticación principal Configuración de autenticación de organización para incluir el nombre de instancia del módulo de autenticación específico.
