Autenticación

El servicio de autenticación general debe registrarse como servicio en un dominio para que un usuario pueda iniciar la sesión utilizando los otros tipos de módulos de autenticación. El servicio de autenticación principal permite al administrador de Access Manager definir valores predeterminados para los parámetros de autenticación de un dominio. Estos valores pueden utilizarse si no se ha definido ningún otro valor que los anule en el módulo de autenticación especificado. Los valores predeterminados del servicio de autenticación principal se definen en el archivo amAuth.xml y se almacenan en Directory Server después de la instalación.

Entre las propiedades de autenticación, se incluyen:

Propiedades avanzadas

Si hace clic en el botón Advanced Properties (Propiedades avanzadas), se muestran los atributos de autenticación principal que pueden definirse para el dominio. No se mostrarán los atributos globales, ya que no se pueden aplicar al dominio.

Cadena de autenticación predeterminada

Define la cadena de autenticación que se va a utilizar para el dominio. Una vez creada la nueva cadena en el atributo de cadena de autenticación, aparecerá en la lista desplegable y permitirá su selección. Consulte Para crear un nuevo encadenamiento de autenticación para obtener más información.

Cadena de autenticación del administrador

Define la cadena de autenticación sólo para los administradores. Este atributo se puede utilizar si la cadena de autenticación de los administradores tiene que ser distinta de la cadena de los usuarios finales. Los módulos configurados en este atributo se heredan cuando se accede a la consola de Access Manager.

URL de éxito predeterminada

Especifica la dirección URL a la que se redirigirá el usuario si la autenticación en el dominio se realiza con éxito.

Instancias del módulo

Permite definir y configurar los módulos de autenticación predeterminados de Access Manager para el dominio. Se pueden agregar varias instancias del mismo módulo de autenticación. Consulte Para crear una nueva instancia de módulo de autenticación para obtener más información.

Cadena de autenticación

Se pueden configurar uno o varios módulos de autenticación a los que el usuario debe transferir las credenciales de autenticación. Este proceso se denomina encadenamiento de autenticación. Este atributo permite definir las opciones de encadenamiento de autenticación para el dominio. Las cadenas que cree se mostrarán en la lista de cadenas de autenticación predeterminadas. Consulte Para crear un nuevo encadenamiento de autenticación para obtener más información.
