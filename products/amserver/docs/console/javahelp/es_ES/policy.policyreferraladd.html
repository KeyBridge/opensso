Para crear una nueva política de referencia

  1. Seleccione el dominio para el que desea crear la política.

  2. Haga clic en New Referral (Nueva referencia) en la lista de políticas.

  3. Agregue un nombre y una descripción para la política.

  4. Si desea que la política esté activa, seleccione Yes (Sí) en el atributo Active (Activo).

  5. No es necesario definir todos los campos de las políticas normales en este momento. Puede crear las políticas, y agregar las reglas y las referencias más adelante. Si desea definir los campos ahora, consulte los siguientes temas:

    • Para agregar una regla a una política

    • Para agregar referencias a una política

  6. Haga clic en Create (Crear).
