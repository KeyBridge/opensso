Para crear un nombre de instancia nuevo

El servicio de asignación de nombres lee este atributo durante la inicialización. Esta lista contiene los servidores de la sesión de Access Manager en una única configuración de Access Manager.

  1. Haga clic en Nuevo en la lista Instance Name (Nombre de instancia).

  2. Introduzca el nombre de host y el puerto en el campo Server (Servidor).

  3. Escriba el Nombre del sitio.

    Este valor sólo identifica al servidor. Cada servidor que participa en el equilibrado de carga o en la conmutación debe tener un identificador exclusivo. También se utiliza para acortar la longitud de la cookie mediante la asignación de la URL del servidor al Id. del servidor. La sintaxis es:

    intance_ID(|site_ID)

  4. Haga clic en "Aceptar".

    Para editar un nombre de instancia, haga clic en una entrada de la lista Instance Name (Nombre de instancia) y cambie los valores como corresponda.

  5. Haga clic en Guardar en la página principal del servicio de plataforma.
