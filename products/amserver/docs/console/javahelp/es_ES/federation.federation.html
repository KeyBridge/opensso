Fédération

La interfaz de Federación es la implementación Access Manager del Liberty Alliance Project Marco de federación de identidad (ID-FF de Liberty), SAMLv2 y especificaciones WS-FED. Definen un conjunto de protocolos, enlaces y perfiles que proporcionan una solución para la federación de identidades, la autenticación entre dominios y la administración de sesiones. Dans la Console Federated Access Manager, l'interface Fédération permet de configurer et de maintenir les éléments suivants :

  • Cercle d'approbation

  • Proveedores de identidades

  • SAML1.x
