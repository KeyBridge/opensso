Servicios

En Access Manager, un servicio es un grupo de atributos administrados conjuntamente por la consola de Access Manager. Los atributos pueden ser bits de información relacionada como, por ejemplo, el nombre de un empleado, su puesto y su dirección de correo electrónico, aunque se utilizan normalmente como parámetros de configuración de un módulo de software como, por ejemplo, una aplicación de software o un servicio de pago de nóminas.

Mediante la ficha Services (Servicios), puede agregar y configurar una serie de servicios predeterminados de Access Manager para un dominio. Puede agregar los siguientes servicios:

  • Administración

  • Discovery Service

  • Globalization Settings

  • Restablecimiento de contraseña

  • Session

  • Usuario
