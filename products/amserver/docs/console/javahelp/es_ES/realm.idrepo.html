Almacenes de datos

Un almacén de datos es una base de datos en la que se pueden almacenar atributos de usuarios y datos de configuración de usuario.

Access Manager proporciona complementos de repositorios de identidades que se conectan a una estructura de repositorio de identidades LDAPv3. Esta nueva función permite ver y recuperar la información de usuario de Access Manager sin necesidad de realizar cambios en la base de datos de usuarios existente. La estructura de Access Manager integra los datos del complemento de repositorio de identidades con los datos de otros complementos de Access Manager para formar una identidad virtual para cada usuario. Access Manager puede utilizar la identidad universal en los procesos de autenticación y autorización en varios repositorios de identidades. La identidad de usuarios virtual se destruye al finalizar la sesión del usuario.

Access Manager admite los siguientes almacenes de datos:

  • Complemento de repositorio de Access Manager

  • Active Directory

  • Repositorios de archivos sin formato

  • LDAPv3 genérico

  • Sun Directory Server con esquema de Access Manager
