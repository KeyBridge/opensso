Asignador de ID de recursos

El valor de este atributo especifica la implementación de com.sun.identity.liberty.ws.interfaces.ResourceIDMapper. Aunque puede desarrollarse una nueva implementación, Access Manager proporciona la clase predeterminada com.sun.identity.liberty.ws.idpp.plugin.IDPPResourceIDMapper, que asigna un identificador de recursos de detección a un Id. de usuario.
