Para crear un nuevo agente

La creación de un nuevo agente también recibe el nombre de creación de un nuevo perfil de agente. Puede realizar esta tarea en la consola de administración de Federated Access Manager . Esta tarea está destinada a los agentes web, J2EE y 2.2. Sin embargo, algunos de los pasos individuales no se aplican a los agentes 2.2.

  1. Haga clic en Agentes en la ficha Configuración.
  2. Si es necesario, seleccione la ficha del tipo de agente adecuado.
    • Agentes web:

      La ficha Web se utiliza para los agentes web y se selecciona de forma predeterminada. No es necesario realizar ninguna selección adicional.

    • Agentes J2EE:

      La ficha J2EE se utiliza para los agentes J2EE. Si es pertinente, haga clic en esta ficha.

    • Agentes 2.2.

      La ficha Agentes 2.2 se utiliza para los agentes del conjunto de software Policy Agent 2.2. Si es pertinente, haga clic en esta ficha.

  3. En la sección Agente, haga clic en Nuevo.
  4. En el campo Nombre, introduzca el nombre del nuevo perfil de agente.
  5. Introduzca y confirme la contraseña.

    Precaución – Esta contraseña debe ser igual a la que se introduzca en el archivo de contraseñas del perfil de agente especificado al ejecutar el programa agentadmin para instalar el agente.

  6. Seleccione la configuración pertinente: local o centralizada (No se aplica a los agentes 2.2).

    Además, si se selecciona local, los atributos relacionados con este agente sólo pueden modificarse desde un archivo del agente y no desde la consola.

  7. En el campo URL del servidor, introduzca la URL del servidor de Federated Access Manager (No se aplica a los agentes 2.2).

    Por ejemplo:

    http://fam.example.com:8080/fam
  8. En el campo URL de agente introduzca la URL de la aplicación de agente, agentapp (No se aplica a los agentes 2.2).

    Por ejemplo:

    http://fam.example.com:8090/agentapp
  9. Haga clic en Create (Crear).

    La consola crea el perfil de agente y muestra de nuevo la página del agente (agente web, J2EE o 2.2) con un vínculo al nuevo agente.

    Si desea realizar la configuración adicional del perfil de agente, haga clic en este vínculo para que se muestre la página Editar agente.
