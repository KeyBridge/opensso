Administración de directorio

La interfaz de administración de identidades permite la creación y administración de objetos relacionados con entidades. Pueden modificarse o eliminarse usuarios, roles, grupos, directivas, organizaciones, suborganizaciones, objetos de contenedor y agentes mediante la consola de Federated Access Manager o la interfaz de línea de comandos. La consola incluye administradores predeterminados con diversos niveles de privilegios para crear y administrar las organizaciones, grupos, contenedores, usuarios y políticas. (Pueden crearse administradores adicionales en función de los roles). Los administradores se definen en Directory Server durante su instalación con Federated Access Manager.

Para obtener descripciones de los tipos de objetos de identidades de Federated Access Manager, e instrucciones básicas de creación y configuración, consulte la sección Directory Management en la Guía de administración de Federated Access Manager 8.0.
