Control de acceso

La interfaz de control de acceso permite crear y administrar los servicios de autenticación y autorización para proteger y regular los recursos basados en dominios. Cuando un usuario de empresa solicita información, Access Manager comprueba la identidad del usuario y le brinda acceso a los recursos específicos solicitados por éste. Mediante la consola de Access Manager, se pueden crear y administrar los siguientes elementos:

  • Dominios

  • Autenticación

  • Servicios

  • Almacenes de datos

  • Privilegios

  • Políticas

  • Asuntos
