Agente de detección

El perfil del agente de detección contiene la configuración de la entidad emisora de confianza utilizada por el cliente/perfil de servicios web con el fin de comunicarse con el servicio de detección de Liberty para el registro y las búsquedas de servicios web, y para obtener las credenciales de seguridad.

Grupo

El mecanismo de grupo permite definir un conjunto de tipos de agentes similares. Debe definirse el grupo antes de incluir un agente específico en un conjunto.

Password (Contraseña)

Define la contraseña del agente de detección.

Confirmar contraseña

Confirme la contraseña.

Ubicación del depósito de configuración de agentes

Este atributo define la ubicación del agente del depósito de configuración para el agente de detección.

Alias de clave privada

Este atributo define el alias de certificado utilizado por la entidad emisora de confianza para firmar las solicitudes de mensajes SOAP realizadas al servicio de detección.

Punto final del servicio de detección

Este atributo define el punto final del servicio de detección con el que el cliente de la entidad emisora de confianza establece comunicaciones para los registros y las búsquedas de servicios.

Punto final del servicio web de autenticación

Este atributo define el punto final del servicio de autenticación que utiliza el cliente de servicios web para la autenticación mediante el testigo de SSO del usuario final con el fin de recibir la oferta de recursos del servicio de detección (denominada también oferta de recursos de bootstrapping).
