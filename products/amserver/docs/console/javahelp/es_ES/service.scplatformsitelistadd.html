Para crear un nombre de sitio nuevo

  1. Haga clic en Nuevo en la lista Nombre de sitio.

  2. Introduzca el nombre de host y el puerto en el campo Server (Servidor).

  3. Escriba el Nombre del sitio.

    Este valor sólo identifica al servidor. Cada servidor que participa en el equilibrado de carga o en la conmutación por error debe tener un identificador exclusivo compuesto por dos dígitos. Por ejemplo, 01.

  4. Haga clic en Guardar.

    Para editar un nombre de sitio, haga clic en una entrada de la lista de nombres de sitios y cambie los valores como corresponda.
