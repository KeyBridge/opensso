Création d'un cercle d'approbation

Procédez comme suit pour créer un cercle d'approbation. Le point de départ est Nouveau cercle d'approbation sous l'interface Fédération.

  1. Cliquez sur Nouveau pour afficher le cercle d'attributs d'approbation.

    La page Nouveau cercle de profil d'approbation s'affiche.

  2. Entrez le nom du cercle d'approbation.
  3. Entrez la description du cercle d'approbation dans le champ Description.
  4. Tapez la valeur de l'URL du service écriture IDFF.

    L'URL du service écriture IDFF indique l'emplacement du servlet qui écrit le cookie du domaine commun. Utilice el formato http:// common-domain-host :port/common/idffwriter .

  5. Tapez la valeur de l'URL du service lecture IDFF.

    L'URL du service lecture IDFF indique l'emplacement du servlet qui lit le cookie du domaine commun. Utilice el formato http:// common-domain-host :port/common/idffreader .

  6. Tapez la valeur de l'URL du service écriture SAML2.

    Este atributo especifica la ubicación del servicio de escritor de SAML2 que escribe la cookie en el dominio común. Utilice el formato http:// common-domain-host :port/common/saml2writer .

  7. Tapez la valeur de l'URL du service lecture SAML2.

    Este atributo especifica la ubicación del servicio de lector de SAML2 que escribe la cookie en el dominio común. Utilice el formato http:// common-domain-host :port/common/saml2reader .

  8. Seleccione Activo o Inactivo.

    El estado predeterminado es “Activo” (Active). Si selecciona Inactivo, se deshabilita la comunicación dentro del dominio de autenticación.

  9. Sélectionnez le domaine dans lequel le cercle d'approbation sera créé.
  10. Elija uno o varios de los proveedores disponibles y haga clic en la flecha Agregar para seleccionarlos.

    La lista proporcionada contiene los nombres de las entidades creadas y rellenadas con proveedores. Pour plus d'informations, voir Ajouts de fournisseurs à un cercle d'approbation

  11. Cliquez sur OK pour terminer la configuration.

    Le nouveau cercle d'approbation s'affiche dans la liste Cercle d'approbation.
