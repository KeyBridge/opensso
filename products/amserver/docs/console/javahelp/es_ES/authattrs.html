Autenticación

Access Manager se instala con un conjunto de tipos de módulos de autenticación predeterminados. Las instancias de los módulos de autenticación son complementos que recopilan información del usuario, como el Id. y la contraseña, cotejan la información con las entradas de una base de datos, y permiten o deniegan el acceso al usuario. Se pueden crear y configurar de forma independiente varias instancias del mismo tipo.
