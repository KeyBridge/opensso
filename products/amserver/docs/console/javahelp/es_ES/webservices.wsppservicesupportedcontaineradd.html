Para configurar un contenedor admitido

Un container es un atributo que define una ubicación en la que se guarda una parte de los datos de identidad. A continuación, se muestra el procedimiento para agregar nuevos atributos al Servicio de perfil personal de Liberty. El punto de partida es la pantalla de configuración del Servicio de perfil personal de Liberty en el módulo Servicios Web.

  1. Haga clic en Nuevo para mostrar los nuevos atributos de contenedor admitido o seleccione el nombre de un contenedor configurado para modificarlo.

  2. Proporcione los valores de los atributos de contenedores según la siguiente información:

    Nombre de contenedor

    Introduzca un nombre para el contenedor como Tarjeta_crédito.

    Complemento

    Introduzca un nombre de clase para administrar todo el contenedor. Esta opción se puede utilizar para anular la clase com.sun.identity.liberty.ws.idpp.plugin.IDPPContainer de la implementación predeterminada.

  3. Haga clic en Aceptar para completar la configuración del contenedor.

  4. Haga clic en Guardar en la página del Servicio de perfil personal de Liberty para completar la configuración del servicio.
