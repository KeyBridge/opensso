Para crear un nuevo almacén de datos

  1. Seleccione el dominio para el que desea agregar un nuevo almacén de datos.

  2. Haga clic en la ficha Data Store (Almacén de datos).

  3. Haga clic en Nuevo en la lista de almacenes de datos.

  4. Introduzca un nombre para el almacén de datos.

  5. Seleccione el tipo de almacén de datos que desea crear.

  6. Haga clic en Next (Siguiente).

  7. Configure el almacén de datos introduciendo los valores de atributos adecuados. Consulte Tipos de almacenes de datos de Access Manager para obtener vínculos a las definiciones de atributos de cada tipo de almacén de datos.

  8. Haga clic en Finish (Finalizar).
