Dirección IP/nombre DNS

Defina la condición en función de un intervalo de direcciones IP. Se pueden definir los siguientes campos:

Desde/hasta dirección IP

Especifica el intervalo de direcciones IP.

Nombre DNS

Especifica el nombre DNS. En este campo se puede especificar un nombre de host completo o una cadena con los siguientes formatos:

nombredominio

*.nombredominio
