Dominios

Un dominio de control de acceso es un grupo de propiedades de autenticación y políticas de autorización que pueden asociarse a un usuario o grupo de usuarios. Los datos del dominio se almacenan en un árbol de información de propietario creado por Access Manager en el almacén de datos que especifique. La estructura de Access Manager agrega las políticas y propiedades incluidas en cada dominio en el árbol de información de Access Manager. Access Manager 7.1 inserta de forma automática el árbol de información de Access Manager en forma de rama especial en Sun Java Enterprise System Directory Server, separado de los datos de usuarios. Puede utilizar los dominios de control de acceso junto con una base de datos LDAPv3.

En la ficha Realms (Dominios), puede configurar las siguientes propiedades de control de acceso:

  • Propiedades generales

  • Autenticación

  • Servicios
