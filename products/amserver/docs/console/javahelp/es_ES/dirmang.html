Administración de directorio

La ficha Administración de directorios sólo se muestra al instalar Access Manager en el modo tradicional. La función de administración de directorio proporciona una solución de administración de identidades para las implementaciones de Access Manager habilitadas Sun Java System Directory Server.

Para obtener más información sobre las ocpiones de instalación en el modo tradicional, consulte la Sun Java Enterprise System Installation Guide for UNIX.
