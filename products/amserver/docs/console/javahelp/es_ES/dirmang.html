Administración de directorio

La ficha Administración de directorio sólo se muestra al instalar Federated Access Manager en el modo tradicional. La función de administración de directorio proporciona una solución de administración de identidades para las implementaciones de Federated Access Manager habilitadas con Sun Java System Directory Server.

Para obtener más información sobre las opciones de instalación en el modo tradicional, consulte la Sun Java Enterprise System Installation Guide for UNIX.
