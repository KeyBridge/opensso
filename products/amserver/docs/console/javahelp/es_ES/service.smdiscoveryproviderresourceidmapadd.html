Para configurar un asignador de Id. de recursos

com.sun.identity.liberty.ws.interfaces.ResourceIDMapper es una interfaz utilizada para asignar un identificador de usuario al identificador de recurso asociado a él. Las distintas implementaciones se pueden desarrollar con la clase implementada y agregarse al atributo. A continuación, se muestra el procedimiento para agregar un nuevo asignador de Id. de recursos al Servicio de detección. El punto de partida es la pantalla de configuración del Servicio de detección en el módulo Servicios Web.

  1. Haga clic en "Nuevo" (New) para mostrar los nuevos atributos del asignador o seleccione el nombre de un asignador configurado para modificarlo.

  2. Proporcione los valores de los atributos del asignador según la siguiente información:

    Id. de proveedor

    Este atributo toma como valor un URI que señala al Servicio de detección. El valor presenta el siguiente formato: http:// host:port/amserver/Liberty/disco

    Asignador de Id.

    Introduzca el nombre para la clase implementada.

  3. Haga clic en "Aceptar" para completar la configuración del asignador.

  4. Haga clic en Guardar en la página del Servicio de detección para completar la configuración del servicio.
