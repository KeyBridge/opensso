Para crear una política normal nueva

  1. Seleccione el dominio para el que desee crear la política.

  2. Haga clic en New Policy (Nueva política) en la lista de políticas.

  3. Agregue un nombre y una descripción para la política.

  4. Si desea que la directiva esté activa, seleccione Sí en el atributo Activo.

    La opción Sí está seleccionada de forma predeterminada. Si desea que no se aplique la directiva durante la evaluación de directivas, anule la selección de este atributo para que esté inactivo.

  5. No es necesario definir todos los campos de las políticas normales en este momento. Puede crear las políticas, y agregar las reglas, las condiciones y los proveedores de respuesta más adelante. Si desea definir los campos ahora, consulte los siguientes temas:

    • Para agregar una regla a una política

    • Para agregar un asunto a una política

    • Para agregar una condición a una política

    • Para agregar un proveedor de respuesta a una política

  6. Haga clic en "Aceptar".
