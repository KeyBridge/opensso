Círculo de confianza.

Un círculo de confianza, denominado anteriormente dominio de autenticación, es una federación de un número cualquiera de proveedores de servicios (y, al menos, un proveedor de identidades) con los que las entidades principales pueden tramitar negocios en un entorno seguro y aparentemente perfecto. Para crear y rellenar un círculo de confianza, debe crear primero una entidad que albergue los metadatos (la información de configuración que define una determinada arquitectura de servicios de identidades) para cada proveedor que será miembro del dominio de autenticación. A continuación, configure y guarde el dominio de autenticación. Por último, para agregar una entidad (un proveedor configurado) al dominio de autenticación, edite las propiedades de la entidad.

Nota – Para obtener más información sobre las entidades, consulte Entidades: proveedor o afiliado.

A continuación se muestran distintas tareas asociadas a los dominios de autenticación:

  • Para modificar un perfil de círculo de confianza

  • Para agregar proveedores a un círculo de confianza

  • Para eliminar un perfil de círculo de confianza
