Agentes

La configuración centralizada de agentes proporciona a los administradores de agentes un método para administrar varias configuraciones de agentes desde una ubicación central. Las configuraciones de agentes se almacenan en el depósito de datos de Federated Access Manager y se administran mediante la consola de Federated Access Manager. Cualquier cambio realizado en las configuraciones de agentes se comunicará a los agentes afectados y éstos reaccionarán consecuentemente en función de la naturaleza de las propiedades actualizadas. Los perfiles de agentes específicos que puede crear son:

  • Agente de directivas web

  • Agente de directivas J2EE

  • Proveedor de servicios web

  • Atributos de clientes de servicios web

  • Servicio de testigos de seguridad

  • Agente de detección

  • Agente de directivas 2.2
