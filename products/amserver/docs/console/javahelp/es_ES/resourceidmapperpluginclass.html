Clases de complemento del asignador de Id. de recursos

El valor de este atributo es una lista de las clases que generan identificadores para una oferta de recursos configurada para una organización o rol. com.sun.identity.liberty.ws.interfaces.ResourceIDMapper es una interfaz utilizada para asignar un identificador de usuario al identificador de recurso asociado a él. El Servicio de detección proporciona dos implementaciones para esta interfaz:

  • com.sun.identity.liberty.ws.disco.plugins.Default64ResourceIDMapper (formato: providerID + "/" + the Base64 encoded userIDs)

  • com.sun.identity.liberty.ws.disco.plugins.DefaultHexResourceIDMapper (formato: providerID + "/" + the hex string of userID)

Se pueden desarrollar distintas implementaciones con la clase de implementación y agregarlas como valor de este atributo. Para ello, haga clic en Nuevo y utilice el siguiente formato:

providerid=providerID| class_name_and_path
