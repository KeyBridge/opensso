Modification d'un profil de cercle d'approbation

Suivez cette procédure pour modifier les attributs globaux d'un cercle d'approbation existant ou pour lui ajouter des fournisseurs. Le point de départ est Cercle d'approbation sous l'interface Fédération.

  1. Cliquez sur le nom d'un cercle d'approbation configuré afin de modifier son profil ou de lui ajouter des fournisseurs.

    La page Modification du cercle d'approbation s'affiche.

  2. Tapez les nouvelles valeurs ou modifiez celles présentes dans les attributs généraux du cercle d'approbation :
    Nom

    La valeur statique de cet attribut est le nom fourni lors de la création du cercle d'approbation.

    Description

    La valeur de cet attribut est une description du cercle d'approbation. Si nécessaire, vous pouvez modifier la description déjà saisie.

    URL du service écriture IDFF

    Este atributo especifica la ubicación del servicio que escribe la cookie del dominio común. La URL presenta el formato http://common-domain-host:port/common/idffwriter .

    URL du service lecture IDFF

    Este atributo especifica la ubicación del servicio que lee la cookie del dominio común. La URL presenta el formato http://common-domain-host:port/common/idffreader .

    URL du service écriture SAML2

    Cet attribut spécifie l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun. La URL presenta el formato http://common-domain-host: port/common/saml2writer

    URL du service lecture SAML2

    Cet attribut spécifie l'emplacement du service écriture SAML2 qui écrit le cookie dans le domaine commun. La URL presenta el formato http://common-domain-host: port/common/saml2reader

    Estado

    El estado predeterminado es “Activo” (Active). Sélectionner Inactif désactive la communication avec le cercle d'approbation.

  3. Elija uno o varios de los proveedores disponibles y haga clic en la flecha Agregar para seleccionarlos.

    La lista proporcionada contiene los nombres de las entidades creadas y rellenadas con proveedores. Pour plus d'informations, voir Ajouts de fournisseurs à un cercle d'approbation

  4. Cliquez sur Enregistrer pour terminer l'opération.
