Para modificar un perfil de dominio de autenticación

Siga este procedimiento para editar los atributos generales de un dominio de autenticación existente o para agregar proveedores a dicho dominio. El punto de partida es Dominios de autenticación en la interfaz Federación.

  1. Haga clic en el nombre de un dominio de autenticación configurado para modificar su perfil o para agregarle proveedores.

    Aparece la página Editar dominio de autenticación.

  2. Escriba los nuevos valores o edite los valores existentes para los atributos generales del dominio de autenticación:
    Nombre

    El valor estático de este atributo es el nombre proporcionado al crear el dominio de autenticación.

    Descripción

    El valor de este atributo es una descripción del dominio de autenticación. Puede modificar la descripción introducida anteriormente si es necesario.

    URL de servicio de escritor

    Este atributo especifica la ubicación del servicio que escribe la cookie del dominio común. La URL presenta el formato http://common-domain-host:port/common/writer .

    URL de servicio de lector

    Este atributo especifica la ubicación del servicio que lee la cookie del dominio común. La URL presenta el formato http://common-domain-host:port/common/transfer .

    Estado

    El estado predeterminado es âActivoâ (Active). Si selecciona âInactivoâ (Inactive), se deshabilita la comunicación dentro del dominio de autenticación.

  3. Haga clic en Agregar para agregar proveedores al dominio de autenticación.

    Aparece la página Seleccionar socio de confianza.

  4. Elija uno o varios de los proveedores disponibles y haga clic en la flecha Agregar para seleccionarlos.

    La lista proporcionada contiene los nombres de las entidades creadas y rellenadas con proveedores. Para obtener más información, consulte Para agregar proveedores a un dominio de autenticación.

  5. Pulse Aceptar para guardar los proveedores para el dominio de autenticación.

    A continuación, volverá a la pantalla de perfil Dominio de autenticación.

  6. Haga clic en Guardar para completar la operación.
