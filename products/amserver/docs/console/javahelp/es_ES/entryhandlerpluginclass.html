Clase de complemento del administrador de entradas:

El valor de este atributo es el nombre y la ruta a la clase de complemento que implementa la interfaz DiscoEntryHandler utilizada para definir o recuperar entradas de detección de una entidad principal. Se proporciona una implementación predeterminada. Para administrar de forma diferente las entradas de detección, implemente la interfaz com.sun.identity.liberty.ws.disco.plugins.DiscoEntryHandler y defina la clase de implementación como valor para este atributo.
