Asuntos

La interfaz de asuntos permite la administración básica de identidades en un dominio. Cualquier identidad creada en la interfaz de asuntos puede utilizarse en la definición del asunto para una política creada con el tipo de asunto de identidad de Access Manager.

Las identidades que se pueden crear y modificar son:

  • Usuario

  • Agentes

  • Rol filtrado

  • Rol

  • Grupo
