服务器和站点

“服务器和站点”配置属性用于整个部署的站点和服务器的集中配置管理。

可在至少两台不同的主机服务器上部署多个（两个或更多）Access Manager 实例。例如，您可能将两个实例部署在一台服务器上，并将第三个实例部署在另一台服务器上。或者，您可能将所有实例部署在不同的服务器上。此外，您还可以根据部署要求在“会话故障转移”模式下配置 Federated Access Manager 实例。

一个或多个负载平衡器会将客户机请求路由到各个 Federated Access Manager 实例。根据部署要求（例如，采用循环或平均负载）配置每个负载平衡器，以在 Federated Access Manager 实例之间分布负载。负载平衡器简化了部署，并解决了诸如客户机和后端 Access Manager 服务器之间防火墙之类的问题。您可以在部署 Access Manager 时采用硬件或软件负载平衡器。所有的 Access Manager 实例都访问相同的 Directory Server。
