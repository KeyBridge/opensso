会话窗口

显示当前登录到 Access Manager 的用户和每次登录的会话时间。显示的字段包括：

用户 ID

显示当前登录用户的用户 ID。

剩余时间（以分钟为单位）

显示需要重新验证之前，用户的该会话所剩余的时间（以分钟为单位）。

最长会话时间（以分钟为单位）

显示会话过期，用户必须重新验证以重新获得访问权限之前可以登录的最长时间（以分钟为单位）。

空闲时间（以分钟为单位）

显示用户处于空闲状态的时间（以分钟为单位）。

最长空闲时间（以分钟为单位）

显示用户在需要重新验证之前，可以处于空闲状态的最长时间（以分钟为单位）。时间限制由管理员在“会话”服务中定义。有关会话属性的定义，参见会话。
