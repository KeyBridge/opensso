配置资源 ID 映射程序

com.sun.identity.liberty.ws.interfaces.ResourceIDMapper 是用于将用户标识符映射到与其相关联的资源标识符的接口。使用实现类可以开发不同的实现方法，并且可以将这些方法添加到属性。以下是向搜索服务添加新资源 ID 映射程序的过程。起点是 Web 服务模块中的搜索服务配置屏幕。

  1. 单击“新建”以显示新的映射程序属性，或单击已配置的映射程序名称以对其进行修改。

  2. 基于以下信息，为“映射程序属性”提供值：

    提供者 ID

    此属性采用的值是指向搜索服务的 URI。此值的书写格式如下：http:// host:port/amserver/Liberty/disco

    ID 映射程序

    输入实现类的名称。

  3. 单击“确定”完成映射程序的配置操作。

  4. 单击“搜索服务”页面上的“保存”完成服务的配置操作。
