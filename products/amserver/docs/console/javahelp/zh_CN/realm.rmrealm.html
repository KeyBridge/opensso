领域

访问控制领域是可以与用户或用户组关联的一组验证属性和授权策略。领域数据存储于专有信息树中，该信息树由 Access Manager 在您指定的数据存储库中创建。Access Manager 框架聚集了 Access Manager 信息树中每个领域所包含的策略和属性。默认情况下，Access Manager 7 会将 Access Manager 信息树作为 Sun Java Enterprise System Directory Server 的特殊分支自动插入，但用户数据除外。您可以在使用任何 LDAPv3 数据库的同时使用访问控制领域。

在“领域”选项卡中，可以为访问控制配置以下属性：

  • 常规属性

  • 验证

  • 服务
