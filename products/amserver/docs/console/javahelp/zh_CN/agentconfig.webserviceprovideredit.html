Web 服务提供者

Web 服务提供者代理配置文件描述用于保护 Web 服务提供者的 Web 服务请求的配置。对每个 Web 服务提供者而言，Web 服务的名称必须唯一。

常规

以下“常规”属性定义 Web 服务提供者的基本属性：

组

可通过“组”机制来定义相似代理类型的集合。组必须在将特定代理纳入集合之前进行定义。

密码

定义 Web 服务提供者属性的密码。

密码确认

确认密码。

状态

定义 Web 服务提供者是否参与保护外部服务请求。默认情况下，该属性设置为激活，即参与保护。

通用标识符

列出用于唯一定义 Web 服务提供者代理的基本 LDAP 属性。

安全

以下属性定义 Web 服务提供者的安全属性：

安全机制

定义用于保护 Web 服务提供者请求的安全证书类型。可选择以下安全类型：

  • 匿名 — 匿名安全机制不包含安全证书。

  • LibertyBearerToken

  • LibertySAMLToken

  • LibertyX509Token — 使用 Libery 定义的 X509 证书来保护 Web 提供者请求。

  • SAML-HolderOfKey — 使用 SAML 1.1 声明类型 Holder-Of-Key 来保护请求。

  • SAML-SenderVouches — 使用 SAML 1.1 声明类型 Sender Vouches 来保护请求。

  • SAML2–HolderOfKey — 使用 SAML 2.0 声明令牌类型 Holder-Of-Key 来保护请求。

  • SAML2–SenderVouches — 使用 SAML 2.0 声明令牌类型 Sender Vouches 来保护请求。

  • UserNameToken — 使用用户名令牌来保护 Web 提供者客户机请求。

  • UserNameToken-Plain — 使用带有明文密码的用户名令牌来保护 Web 提供者请求。

  • X509Token — 使用 X509 证书来保护 Web 提供者请求。

验证链

定义 –

保留消息中的安全头

启用后，该属性定义为 Web 服务提供者保留 SOAP 安全头，以进行进一步处理。

Liberty 服务类型 URN

URN（Universal Resource Name，通用资源名称）描述 Web 服务提供者用于服务查找的 Liberty 服务类型。

用户令牌的证书

该属性表示 Web 服务提供者用来生成用户名安全令牌的用户名/密码共享密钥。

私钥类型

定义 –

SAML 配置

以下属性配置 Web 服务提供者的安全声明标记语言 (Security Assertion Markup Language, SAML)。

SAML 属性映射

该配置表示在声明创建期间需由安全令牌服务生成的 SAML 属性。相同的映射用于 Web 服务提供者以将其填充到 SSOToken。

SAML NameID 映射器插件

定义用于 SAML 帐户映射的 NameID 映射器插件类。

SAML 属性名称空间

定义用于生成 SAML 属性的名称空间。

包含成员资格

如果启用，该属性定义主要成员资格必须包含作为 SAML 属性。

签署和加密

以下属性定义 Web 提供者安全的签署和加密配置：

响应是否已签名

启用后，Web 服务提供者使用给定的令牌类型签署响应。

响应是否已加密

启用后，将加密 Web 服务响应。

请求签名是否已验证

启用后，会验证 Web 服务请求签名。

请求标头是否已加密

启用后，将加密 Web 服务客户机的安全标头。

请求是否已解密

启用后，将解密 Web 服务提供者请求。

密钥库

以下属性配置用于证书存储和检索的密钥库：

Web 服务客户机的公钥别名

该属性定义用于加密 Web 服务响应的公共证书密钥别名。

私钥别名

该属性定义用于签署/解密 Web 服务请求的私有证书密钥别名。

密钥存储库用法

该配置定义是使用默认的密钥库，还是使用自定义的密钥库。必须为自定义密钥库定义以下值：

  • 密钥库位置

  • 密钥库密码

  • 密钥密码

端点

以下属性定义 Web 服务端点：

Web 服务端点

该属性定义 Web 服务客户机发送请求的 Web 服务端点。
