用于资源 ID 映射程序插件的类

此属性的值是类的列表，这些类用于为针对组织或角色配置的资源提供生成标识符。com.sun.identity.liberty.ws.interfaces.ResourceIDMapper 是用于将用户标识符映射到与其相关联的资源标识符的接口。搜索服务为此接口提供了两种实现：

  • com.sun.identity.liberty.ws.disco.plugins.Default64ResourceIDMapper（格式：提供者 ID + "/" + Base64 编码的用户 ID）

  • com.sun.identity.liberty.ws.disco.plugins.DefaultHexResourceIDMapper（格式：提供者 ID + "/" + 用户 ID 的十六进制字符串）

可以用实现类开发不同的实现方法，通过单击“新建”可以将其添加为该属性的值，使用的格式如下：

providerid=providerID| class_name_and_path

参见配置资源 ID 映射程序。
