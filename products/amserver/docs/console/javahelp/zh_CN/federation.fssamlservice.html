SAML

安全声明标记语言 (Security Assertion Markup Language, SAML) 是一个开放标准的协议，它使用 XML 框架在授权机构与可信赖的伙伴站点之间交换安全信息。这些安全信息与主题的验证状态、访问授权和属性信息有关。（由电子邮件地址标识的人是一个主题，正如是一台打印机那样。）SAML 授权（也称为声明方）是已与 SAML API 集成在一起的平台或应用程序，因而它可以中继安全信息。例如，通过传递所需的属性来声明某个主题已通过其系统的验证。可信赖伙伴站点接收安全信息并信任其真实性。

可以为您的实现配置以下 SAML 属性：

  • 目标指示符

  • 站点标识符

  • 可信赖的伙伴

  • 目标 URL

  • 声明超时

  • 声明不早于偏差因数

  • 辅件超时

  • SAML 辅件名称

  • 签署 SAML 声明

  • 签署 SAML 请求

  • 签署 SAML 响应

以下任务与配置 SAML 关联：

  • 配置站点标识符

  • 可信赖的伙伴：选择伙伴类型和配置文件

  • 可信赖的伙伴：配置可信赖的伙伴属性

  • 配置目标 URL
