搜索代理

“搜索代理”配置文件保留信任授权机构配置，Web 服务客户机/配置文件使用该配置与 Liberty 搜索服务进行通信，以进行 Web 服务查找、注册和获取安全证书。

组

可通过“组”机制来定义相似代理类型的集合。组必须在将特定代理纳入集合之前进行定义。

密码

定义搜索代理的密码。

密码确认

确认密码。

代理配置系统信息库的位置

此属性定义搜索代理配置系统信息库的代理位置。

私钥别名

此属性定义信任授权机构客户机用来签署搜索服务的 SOAP 消息请求的证书别名。

搜索服务端点

此属性定义信任授权机构客户机为注册和查找服务而在其中建立通信的搜索服务端点。

验证 Web 服务端点

此属性定义验证服务端点， Web 服务客户机使用该端点进行验证，方法是使用最终用户的 SSOToken 来接收搜索服务资源提供（也称为引导资源提供）。
