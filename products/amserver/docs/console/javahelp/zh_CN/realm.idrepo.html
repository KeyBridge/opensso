数据存储库

数据存储库是一个可以存储用户属性和用户配置数据的数据库。

Federated Access Manager 提供身份系统信息库插件，这些插件连接到 LDAPv3 身份系统信息库框架。从而不必在现有用户数据库中做任何更改，便可查看和检索 Federated Access Manager 用户信息。Federated Access Manager 框架将来自身份系统信息库插件的数据和其他 Federated Access Manager 插件里的数据整合在一起，为每个用户形成了一个虚拟身份。而后，Federated Access Manager 能使用通用身份在多个身份系统信息库之间进行验证和授权过程。在用户会话结束后将会销毁虚拟用户身份。

Federated Access Manager 支持以下数据存储库：

  • Federated Access Manager 系统信息库插件

  • Active Directory

  • 平面文件系统信息库

  • 普通 LDAPv3

  • 使用 Federated Access Manager 模式的 Sun Directory Server

有关创建新数据存储库以及配置系统信息库插件的信息，参见创建新的数据存储库。
