修改验证域配置文件

按照以下过程编辑现有验证域已配置的常规属性，或向其中添加提供者。起点是联合界面下的“验证域”。

  1. 单击已配置验证域的名称，修改其配置文件，或向其中添加提供者。

    随即显示“编辑验证域”页面。

  2. 键入验证域“常规”属性的新值或编辑其现有值：
    名称

    此属性的静态值是您在创建验证域时提供的名称。

    说明

    此属性的值是对验证域所做的说明。可以修改已输入的说明（如果适用）。

    记录器服务 URL

    此属性指定了记录通用域 cookie 的服务的位置。该 URL 的格式为 http://common-domain-host:port/common/writer。

    读取器服务 URL

    此属性指定了读取通用域 cookie 的服务的位置。该 URL 的格式为 http://common-domain-host:port/common/transfer。

    状态

    默认值为活动。选择不活动将禁用验证域之间的通信。

  3. 单击“添加”为验证域添加提供者。

    随即显示“选择可信赖的伙伴”页面。

  4. 选择一个或多个可用的提供者并单击“添加”箭头来选定它们。

    所提供的列表包含已创建并填充了提供者的实体的名称 。有关更多信息，参见向验证域添加提供者。

  5. 单击“确定”将提供者保存至验证域。

    这将使您返回到“验证域配置文件”屏幕。

  6. 单击“保存”完成该操作。
