使代理能够继承组的属性
开始之前

必须首先创建要使代理继承其属性的组。

  1. 单击“配置”选项卡下的“代理”。
  2. 如有必要，选择适当代理类型的选项卡。
    • Web 代理：

      "Web" 选项卡适用于 Web 代理且为默认选项。无需再次选择。

    • J2EE 代理：

      "J2EE" 选项卡适用于 J2EE 代理。如果适用，单击该选项卡。

    • 2.2 代理

      “2.2 代理”选项卡适用于“策略代理 2.2”软件集中的代理。如果适用，单击该选项卡。

  3. 在“代理”部分中单击要配置的代理的名称。
  4. 选择“全局”选项卡后，对于标为“组”的属性，选择要使代理继承其属性的组的名称。

    在页面顶部，“继承设置”按钮变为活动状态。

  5. 单击“继承设置”。
  6. 选择要使代理从组中继承的属性（特性）。
  7. 单击“保存”。
