机制处理程序列表

“机制处理程序列表”属性存储验证 Web 服务所支持的 SASL 机制相关信息。它显示的条目包含用竖线分隔的关键字/值对，格式如下：key=PLAIN|class=com.sun.identity.liberty.ws.authnsvc.mechanism.PlainMechanismHandler。这些参数将在以下表格中加以说明。

参数

值

关键字

关键字参数是必填参数，它用于定义验证 Web 服务所支持的 SASL 机制。

类

类参数是必填参数，它用于为 SASL 机制指定实现类的名称。验证 Web 服务层提供需要由每个 SASL 机制实现的处理程序接口，以处理请求的消息和返回响应。

要添加新的请求处理程序，单击“添加”，并参见配置新的机制处理程序。
