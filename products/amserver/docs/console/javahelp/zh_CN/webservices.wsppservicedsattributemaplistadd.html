配置属性映射

映射是用于定义身份数据持有者的属性。以下是向 Liberty 个人配置文件服务添加新属性的过程。起点是 Web 服务模块中的 Liberty 个人配置文件服务配置屏幕。

  1. 单击“新建”以显示新的“属性映射”属性，或单击已配置的映射名称以对其进行修改。

  2. 基于以下信息，为容器属性提供值：

    名称前缀

    输入要映射的 Liberty 个人配置文件服务身份属性的名称。

    LDAP 属性

    输入“名称前缀”所映射到的 Access Manager LDAP 属性的名称。

  3. 单击“确定”完成“映射”的配置操作。

  4. 单击“Liberty 个人配置文件服务”页面上的“保存”完成服务的配置操作。
