This document describes various customizable files in the OpenID Provider.

web/page.xhtml
  The XHTML template for pages rendered by the OpenID provider.

web/unknown.xhtml
  The default page to render when an unknown (or no) mode is requested of the
  OpenID provider.

web/style.css
  The style sheet used for all pages rendered by the OpenID provider.

web/field.xhtml
  The XHTML template for a single field in a form, including selection
  checkbox, required flag, field value and error message.

web/date.xhtml
  The XHTML template for a date input field, which breaks it down into
  separate year, month and day select lists.

web/button.xhtml
  The XHTML template for a button in a form.

web/setup.xhtml
  The XHTML page that provides the user interface for the checkid_setup
  trust and simple registration extension user interface.
