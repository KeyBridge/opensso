This directory contains the external library dependencies for the OpenID
provider. The following files should be here:

 * commons-beanutils.jar
 * commons-collections.jar
 * commons-digester.jar
 * commons-logging.jar
 * el-api.jar
 * el-ri.jar
 * jsf-api.jar
 * jsf-facelets.jar
 * jsf-impl.jar
 * jsp-api.jar
 * commons-codec-1.3.jar
 * amclientsdk.jar
 * j2ee.jar

For more information, see the README.txt file in the project root directory.
